Firefox for iOS 的進度更新

5月252015

Mozilla 想讓 Firefox 儘可能支援所有的語言、平台，以及裝置。雖然我們因 iOS 的諸多限制，而無法在 iOS 上呈現完整的 Firefox 經驗與繪圖引擎，但我們仍注意到 iOS 8 的最新工具與其提升之處，並把握機會為 iOS 開發 Firefox 瀏覽經驗。


圖片出處：http://techcrunch.com/

過去數個月來，Firefox 團隊持續努力開發此新版瀏覽器。我們現正在 Github 上分享原始碼給熱血的早期測試者，亦將於短時間內主動邀請 Firefox for iOS 測試版 (Beta) 給特定人員。當然，我們期待更大規模的開放測試活動。但在此之前，Mozilla 須完成必要的開發與釋出程序，於 iOS 中實際測試 Firefox App 完畢，才能正式分享給全世界。

敬請期待 Firefox for iOS 的後續開發訊息。
