Mozilla 宣佈投資 Everything.me，深耕 Web 打造強大豐富 APP 行動平台

11月292012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－ Mozilla 基金會，2012 年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。於美西時間 (28日) 宣佈與 Everything.me 合作，此舉顯示 Mozilla 在行動裝置系統上推動 Open Web 與 HTML5 跨平台應用程式的決心。Everything.me 將會在 2013 年推出的 Firefox OS 上扮演重要角色，提供上千款最豐富、使用 HTML5 技術打造的 Apps、遊戲、服務，直接貼近使用者的需求。

Mozilla 目前正與 Everything.me 共同進行平台開發的工作，讓智慧型手機使用者可透過 Everything.me 在線上找到上千款用 HTML5 技術打造的 App、遊戲、服務，直接貼近使用者的使用需求。Everything.me 與 Mozilla 的遠見及目標十分吻合，一起發掘 Web 發展的更多可能性。

參與本次 Everything.me 共 2500 萬美金投資計劃業者包括 Telefónica Digital (西班牙電信公司)、SingTel Innov8 (新加坡電信集團附屬公司) 等，Mozilla 期待藉由本次合作，進一步推動 Open Web 與 HTML5 應用程式，實現 Mozilla 在行動通訊市場的願景。Everything.me 將會在 2013 年推出的 Firefox OS 上扮演重要角色，提供更豐富的 Web Apps 項目，並將打造優質行動內容體驗，更多訊息即將陸續釋出。

關於 Everything.me：

Everything.me 正在翻轉傳統行動通訊市場的常規，並打造 HTML5 的 App 行動搜尋平台。使用者開啓 Everything.me 後，Everything.me 會針對手機進行 HTML5 App 的建議，以符合您使用上的需求。任何由 Everything.me 提供的 APP 都可以從雲端（使用HTML5）來直接使用，不需要下載和安裝。Everything.me 適用於HTML5，已推出 iPhone 應用並很快推出 Android 應用。您可造訪 http://everything.me 或 Twitter (@EverythingMe) 了解更多。
