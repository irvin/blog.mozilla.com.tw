介紹「DevTools Reload」附加元件

3月292016

為了鼓勵更多人加入 Firefox 開發者工具 (DevTools) 的貢獻行列，開發者工具團隊建構了「DevTools Reload」附加元件，期能再降低某些常見的入門門檻。

一般來說，成熟的開放源碼專案應該縮小其程式碼庫的規模，避免可能妨礙新貢獻者加入，且龐大的程式碼也需要更長的學習曲線。此外，雖然找個簡單臭蟲修正是極佳的入門手段，不過太過龐大的程式碼另會提高臭蟲修正的難度。

大型程式碼庫的另一個難題，就是須耗時下載並編譯；這也當然會嚇走想加入的新貢獻者。又，就算貢獻者義無反顧把頭洗下去，但即使簡單的修正作業也要耗上極長的重新編譯時間；同樣可能逼退許多有耐心的開發者。

「DevTools Reload」則讓開發者能變更 JavaScript 與 CSS 原始碼檔案，且只要透過快捷鍵組合，就能自動重新載入新變更過的工具。Firefox 認為這些工具理應像 Web 一樣可供他人開發。

此一方式可讓開發者輕鬆修正 Firefox 開發者工具的錯誤，並讓進行變更的開發者獲得最即時的反饋，有利於快速更替並實驗可能的解決方案。

可欣賞將此附加元件安裝於 Firefox Nightly 上並展示功能的影片。如果你比較習慣白紙黑字，我們也提供入門指南。

此屬於相對極新的功能且仍在開發階段，所以你可能會發現許多運作不甚順暢的地方。只要你發現任何錯誤，歡迎提交問題或聯絡我們。你也可追蹤開發者工具團隊 (@firefoxdevtools) 以隨時獲得新訊息。另請透過 bugzilla 提交自己的貢獻成果。
