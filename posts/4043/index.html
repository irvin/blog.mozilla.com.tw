Lightbeam for Firefox 附加元件介紹

10月262013

網路隱私教育與開放資料認知

若能將線上資料的追蹤過程攤在陽光下，就能協助大家更了解自身周遭的 Web 環境。也因為如此，Mozilla 在 1 年多前就針對 Firefox 開始建構廣受關注的實驗性附加元件 ─ Collusion。

隨著不斷有人爆出政府監控的情事，相關議題也更受到大眾所關注。而有關線上資料蒐集作業，與其背後的作業機制透明度不僅再度受到質疑，相關重要性更顯得不言可喻。各式各樣的第三方為我們提供目前絕大部分的線上經驗。我們現在仍不甚清楚這些第三方公司所提供的價值，也極少在事先被告知的情況下提供個人資料。這些情形都一再傷害著使用者對網際網路的信任感。

Mozilla 今天很榮幸能夠發表新一代的 Collusion 附加元件，我們將之稱為 Lightbeam for Firefox。並將於今年的 MozFest 時發表此版本。


Lightbeam 亦屬於 Firefox 的附加元件之一，將針對所有人每天在 Web 上接觸的第一方與第三方公司，嘗試提供更透明的互動資訊。Lightbeam for Firefox 的開發過程，是由福特基金會 (Ford Foundation) 與自然科學與工程研究委員會 (Natural Sciences and Engineering Research Council，NSERC) 所贊助，並由艾蜜莉卡藝術及設計大學 (Emily Carr University of Art and Design) 的學生與教職員提供相關協助所完成。

Lightbeam for Firefox 的運作方式：

  • Web 行為視覺化 ： 下載 Lightbeam 附加元件並安裝至 Firefox 之後，就會針對你所瀏覽的網頁，以及正於該網頁上活動的第三方，即時呈現網站的相關狀態。
  • 分析個人連線： 以視覺呈現網站的狀態之後，只要是儲存於自己資料中的多個第一方與第三方網站，你均能進一步了解它們之間的關係。使用者亦可隨時重設或儲存自己的資料。
  • 貢獻開放資料： Mozilla 為 Lightbeam 所規劃的願景之一，就是要了解網路追蹤作業，以及第一方與第三方網站的連接方式。如果你願意協助我們，則可將自己的資料貢獻到 Mozilla 的 Lightbeam 資料庫。我們將此資料庫稱為 Web 的奧茲魔法，將透過 Lightbeamfor Firefox 使用者所提供的寶貴資料，掀開網站內部運作的神祕面紗。
  • 支援網頁發佈者： 在開發 Lightbeam for Firefox 期間，我們另與多位領先的網頁發佈者合作，以研究群眾外包 (Crowdsourced) 資料的價值。當然現在仍有許多工作等著我們去完成。一旦開放資料集成熟之後，Mozilla 將進一步透過第三方在發佈者網站上的互動情形，了解網頁發佈者是如何從中創造利潤。

Mozilla 將使用者放在第一位，並且深信每個人都應該可以自行控制 Web 的瀏覽體驗。透過 Lightbeam for Firefox 附加元件與開放資料，我們將建構可貴的社群研究平台，藉以提高公眾認知、提升分析效能，以期最後能打破目前線上追蹤與隱私性的黑盒子，並造成實質性的影響。

Lightbeam for Firefox 現仍處於早期開發階段，目前仍只是第二版本。Mozilla 現已將完整程式碼發佈至 Github 上，以利社群成員開發、擴充、強化此附加元件。Lightbeam 附加元件現已開放下載。

Lightbeam 已於今年的 MozFest 上正式發佈。MozFest 為 Mozilla 最大的群眾會面活動。與會者整個週末都將腦力激盪並建構技術與教材的原型，以期喚起 Web 使用者對自身網路隱私權的重視。
