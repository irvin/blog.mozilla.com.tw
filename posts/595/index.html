Mozilla Firefox Beta 號召重視安全機制的您搶先下載！

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，旗下著名之瀏覽器Firefox Beta 版本於美西時間7日釋出，提供Windows、Mac 和Linux 測試版本供使用者下載測試。此次的 Beta 版將加入安全機制和全新開發者工具，期待愛嘗鮮的你搶先下載測試。

最新版本的 Firefox Beta 目前已開放測試。此次的 Beta 版將加入安全機制和全新開發者工具。

針對一般使用者的新功能：

  • 使用SSL 為Google搜尋加密(Google Search SSL by Default)：授權HTTPS 做為Google搜尋的加密預設，可從基層開始保護您的網路資料和搜尋結果，同時也可防止第三方網站藉由點擊一個頁面項目，而蒐集您的網路行為。Firefox 期待在未來有更多的搜尋引擎，加入SSL加密功能的行列。
  • 全新的網站加密狀態指標：Firefox Beta 在智慧位址列中採用新方法顯示網站的加密狀態。在網站未使用 SSL 加密功能時，網域名稱旁將出現一個地球圖示；若網站使用 SSL 加密，則該位置顯示鎖頭圖示及「https」字樣；若網站另外還擁有延伸驗證（Extended Validation Certificate），則會出現綠色鎖頭圖示； 網頁中混合 http 及 https 內容時，則顯示為灰色的警告圖示。
  • 點選後播放插件：Firefox 讓你可以控制 Flash 及 Quicktime 等插件的播放狀態。啟用此功能後，Firefox Beta 會將所有插件內容加上「播放」按鈕，在點選後即可立即查看其內容。現 在您可以在 about:config 中把 plugins.click_to_play 設定為 true 來測試這個功能，未 來將提供其他選項及設定介面。
  • 全螢幕支援OS X Lion 10.7：Mac的用戶現在可以使用原生 Lion 全螢幕選項，擁有更豐富和身臨其境的的瀏覽的體驗。
  • 智慧位址列(Awesome Bar) 網址自動完成：當使用者一鍵入網址，智慧位址列會幫您主動完成剩下的URL網域名。

針對網站設計師的新功能：

  • 游標鎖定 API：現在，網路程式（例如第一人稱遊戲）能隱藏滑鼠游標，讓程式自己捕捉滑鼠移動狀態、控制其對應行為。這讓開發者在強調視覺的程式中，能再進一步控制滑鼠、提供更好的使用體驗。
  • 檢視虛擬類別的狀態：此功能讓開發者更容易查閱 CSS 虛擬類別及其對應的樣式。在頁面檢測工具中選定一個元素時，其選單將列出 :hover、:active 及 :focus 等虛擬類別，點選其中一個後，就可以將該元素鎖定於此虛擬類別的狀態，方便你檢閱相關樣式。
