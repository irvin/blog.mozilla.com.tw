Firefox 在 Web 上提供 3D 遊戲、視訊電話、檔案共享功能！

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2013年以 HTML5 為基礎的行動作業系統 Firefox OS 獲得市場廣大期待。Mozilla於美西時間 (25日) 釋出 Windows、Mac 和 Linux 版本適用的新版 Firefox 瀏覽器，以及行動版專用的 Firefox for Android。新版本正式加入 WebRTC (Web Real-time Communications) 網路即時通訊，並針對開發者釋出優化過的 JavaScript  (asm.js)，執行3D 遊戲和圖像處理等高效能的應用程式。

在 Web 上執行遊戲與視訊電話是 Web 的二大挑戰，一旦完成就足以證明 Web 絕對是可執行複雜作業的強大平台。Mozilla 的使命之一，就是要解決Web 的這個挑戰，讓 Web 成為開放、創新、充滿機會的平台，使所有人都能暢遊其中。

Firefox 讓開發者得以創造高效能的 Web Apps，並使用瀏覽器直接撥打視訊電話和資料共享，無需外掛程式或第三方軟體。之前在 Web 上做不到的，現在都能以簡單、快速，又有趣的方式實現。

同時，Mozilla 在新版本中釋出優化過的 JavaScript  (asm.js)，開發者不需額外安裝軟體或外掛程式，即可在 Web 上建構高效能的應用程式，例如在 Web 上執行 3D 遊戲和圖像處理。以 Emscripten 跨平台編譯器轉譯 asm.js 之後，諸如遊戲的高效能原生 App 均可直接帶入 Web 之中。

Mozilla 與戰爭機器 (Gears of War) 遊戲公司 Epic 攜手合作，透過 Unreal Engine 3 展現了上述特色。在 JavaScript 平台上可達到近乎原生程式的速度，著實令人驚艷不已。現在開發者僅需低成本的解決方案，搭配如 JavaScript、Emscripten、WebGL 的技術，即可為 Web 帶來高效能的 Apps 與遊戲。

在 Firefox 上執行 Web Apps 如今也同樣支援視訊電話、音訊電話、檔案分享等功能，同樣無需安裝額外軟體或使用第三方外掛程式。利用 Web Real-Time Communications (WebRTC) 再搭配幾行程式碼，即能輕鬆將這些功能整合至 Web Apps 之內。開發者能建構更豐富、更高互動性的網站，讓大家均能跨瀏覽器而相互溝通。

WebRTC  這項技術使得無論行動瀏覽器或桌面瀏覽器，均可直接撥打或接通視訊電話，亦能與親朋好友分享檔案、即時視訊、圖片影像。今年2月，Mozilla 在 MWC (Mobile World Congress) 上與 Ericsson 攜手展示 Firefox 如何能無縫地透過手機裝置進行技術整合，並使用語音、視訊及檔案分享。對 Web 未來的通訊作業，WebRTC 著實開啟了無限的潛能。

Mozilla 不斷挑戰 Web 的極限，透過此進階功能的發展，證明了 Web 才是最好的開發平台，得以改善平台間的互通性，並終止目前各平台互不相容的狀況。Mozilla 打造 BananaBread 測試遊戲來呈現由 Mozilla 所領導的 Web 技術革新，它是採用 WebGL、Emscripten、asm.js、WebRTC 等的 Web 技術所建構而成，展現在執行高階的 3D 多人遊戲時，亦能提供高速且令人咋舌的遊戲效能。

誠如上述，新技術為 Web 帶來了無限可能，Mozilla 除了興奮期盼之外，也期待看到開發者將新技術、新體驗發揮地淋漓盡致。
