Mozilla 的 2013 年度回顧

2013 年是 Web 充滿驚奇的一年。我們在今年歡慶 Mozilla 的十五歲生日，以及網路變得更好的 15 年，期間我們有源源不絕的創新動能，並持續打造開放並易於取用的 Web 資源。接著要和你分享 Mozilla 在 2013 年完成的幾件大事。

Mozilla 在 2013 年發表首款完全以 Web 技術打造的 Open Web 裝置 ─ Firefox OS。可在此了解 Firefox OS 給全世界的消息。我們與四家電信營運商合作，在十四個國家共發售三款裝置。我們很榮幸能與樂金 (LG)、華為 (ZTE)、阿爾卡特 (Alcatel) 等硬體夥伴合作，配合 Telenor、Telefonica、Deutsche telekom、Telecom Italia Mobile 等營運商夥伴共同推出 Firefox OS 裝置。

Mozilla 在不斷推出新技術，推動 Web 成為真正平台的同時，也努力讓 Firefox 更快、更安全，提供更有趣的自訂功能。在以 JavaScript 效能基準 Octane 完成測試之後，我們驕傲宣布 Firefox 再度提升 88% 的速度。為了解放 Web 成為高階遊戲的開發平台，我們開發最佳化的 JavaScript 子集 ─ asm.js，讓 Web App 與遊戲可加快最高六倍速度；另外更與《戰爭機器 (Gears of War)》系列製作公司 Epic Games 合作，將 Unreal Engine 3 帶入 Web 之中。我們最近才剛發表的 Monster Madness，成為首款由 asm.js 打造的 3D 商業遊戲。當然，Mozilla 也很開心將 WebRTC 加入至 Firefox，現能直接在瀏覽器之間互撥視訊電話並分享檔案。

Mozilla 在去年重新開發 Firefox for Android 而達到絕佳的速度效能，讓使用者評價一飛沖天並穩定保持在平均 4.5 顆星，且 Firefox for Android 下載次數已累積突破五千萬次。在 2013 年，我們進一步讓 Firefox for Android 預先安裝於 Kobo 的 Arc 平板電腦，還有集嘉通訊 (Gigabyte) 的 GSmart 智慧型手機之內。

針對瀏覽經驗，Mozilla 特別關心使用者的線上隱私，希望大家都能掌握自己線上資訊的流向。我們今年除了成為 Ponemon Institute 調查報告中 2012 年最受信賴的網際網路公司之外，更於十月份發表 Lightbeam 附加元件，讓使用者能看到自己在 Web 上受追蹤的情形。Lightbeam 截至目前為止已接近一百萬次下載。為了回應美國國家安全局 (NSA) 的監聽問題，我們更提出 StopWatching.Us，以期全球公民都能了解本身早已受監控的線上資料與通訊行為。最後有超過五十八萬多人簽署並響應 StopWatchingUs.org 的訴求。

我們在今年舉辦了第三屆 Mozilla Summit，來自於九十個國家共二千八百名 Mozillians 齊聚於三大都市。所有人歷時三天討論其共通信念、審視反思，及後續所應採取的行動。目前已有超過三百七十名 Mozilla Reps 官方代表，積極於自己所屬的社群中宣揚 Mozilla 與 Firefox 的理念。Mozilla 現更常態推動校園大使方案，已擁有超過八千名校園大使積極響應。目前全球已有數千名 Mozillians 志工廣佈於超過一百零九個國家之中，構成我們的核心社群。

Webmaker.org 更是齊聚活潑創造力的專案，已吸引數千人每天傳授、學習 Web 技術。另針對熱心傳授 Web 知識的社群，Mozilla 另免費提供開放源碼的 Thimble、Popcorn Maker、X-Ray Goggles 等工具，以滿足相關的教學需求。而全球社群亦持續協助 Mozilla 改善工具並設計完整課程。此外，我們才剛結束一系列的大型 Maker Party，共於五十五個國家的三百三十個城市，成功舉辦超過一千六百場的 Web饗宴。

展望 2014 年，Mozilla 更迫不亟待要為全世界數百萬人帶來絕妙的 Web 經驗。
