Firefox 新整合的社群功能

4月82013

Mozilla 所開發的 Social API，可將社群服務直接整合至 Firefox 中，讓使用者體驗更客製化、個人化的社群經驗。Social API 可讓你輕鬆看到好友與家人的動態，而不需切換分頁或開啟新的分頁。

現在 Mozilla 更透過 Firefox Nightly，讓更多社群服務商參與全球性的測試。新加入測試行列的服務商包含 CliqZ、Mixi、MSN Now、微博 (Weibo) 等。

我們很高興能透過 Social API 創造更多可能。未來的 Firefox 瀏覽體驗，將濃縮了電子郵件、金融動態、即時新聞、社群服務，更多其他應用與功能。Mozilla 將立刻開始多家服務商的合作測試，請千萬繼續留意相關主題。

英文原文：https://blog.mozilla.org/futurereleases/2013/04/05/new-social-providers-in-firefox/
