Firefox for Android 測試版更能輕鬆自訂 Web 經驗

Firefox for Android 測試版 (Beta) 現針對開發者與使用者，分別提供更多 API 與自訂功能。且目前由 Firefox for Android 測試版獨家提供這些新功能，讓使用者擁有更個人化的選擇並掌握自己的 Web 經驗。

更多自訂選項

使用者現可自訂首頁，隨時看到自己常用的多樣網站、服務、訊息摘要等網頁內容。其中已由 Mozilla 嚐試串接的如 Instagram、Pocket、Vimeo、Wikipedia等相關服務，現在 Firefox for Android 測試版同樣也讓使用者能隨手取得。


Firefox for Android 測試版所呈現的 Instagram 首頁

使用者可自行選擇並設定瀏覽器的預設首頁內容、重新排序頁面、隱藏不常用的頁面，甚至隱藏所有頁面而獲得「乾淨」的分頁。

使用者可進入系統的設定選單，找到「工具」→「附加元件」，安裝新的首頁頁面；就與一般附加元件的安裝方式完全相同。亦能透過 Mozilla 附加元件網站上的 Home Panels 收藏集而安裝這些頁面。

新一代附加元件

這些新的 API將轉換並延伸首頁的潛力，讓附加元件開發者能將新一代附加元件轉化為 Firefox for Android 測試版使用者經驗的一部分。開發者另可透過這些 API 打造自己的首頁畫面，讓使用者能在這個專屬空間內和附加元件直接互動。

可輕鬆切換語言

我們另外為 Firefox for Android 測試版添增新功能，讓使用者不需重新啟動瀏覽器即可輕鬆切換語言。Firefox for Android 測試版現具備五十四種語言。不論使用者 Android 裝置所支援的語言為何，都能切換自己所需的語言。Mozilla 是根據使用者的需求，直接與熱情的本地化志工合作出此一功能。

請協助我們一同試用新的首頁經驗，並測試新的語言切換功能。當然歡迎分享你的意見並提交任何問題，讓 Mozilla 能持續提供絕佳的效能與功能。
