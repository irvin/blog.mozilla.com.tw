Firefox 桌機、行動雙強出擊，全新版本強勢釋出！

10月92012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012年推出以HTML5 為基礎的行動作業系統Firefox OS並獲得市場廣大期待。今於美西時間9日釋出全新Windows、Mac和Linux 版本適用的Firefox 瀏覽器，以及行動版的 Firefox for Android。桌機版 Firefox 針對開發者工具提供獨家『 開發者命令列 』，命令列可輔助完成程式命令與參數，開發者可使用鍵盤操作讓開發網頁更快速容易。行動版本 Firefox for Android 針對網頁『 閱讀模式』進行優化，雙雙打造絕佳瀏覽體驗！

全新桌機版 Firefox 瀏覽器，針對網站設計師提供便利、快速又具互動性的全新開發者工具列。使用者可藉由『開發者命令列 』(Developer Command Line) 進行操作讓手不需離開鍵盤，以最直覺性、完整指令 (completing commands) 和參數 (parameters) 方式完成指令。同時，無前綴式 (Unprefixing) 的穩定 CSS4 特性，協助開發者無縫支援CSS3 Animation, Transitions, Transforms, Image Values, Values & Units 和 IndexedDB 等功能或效果。同時，由 Mozilla協助開發的無前綴式 (Unprefixing) 的電力狀態 API（Battery Status API）和震動 API（Vibration API）也可在此版本率先體驗。

全新行動版 Firefox for Android 針對使用者的瀏覽行為，提供使用者最便利的『 閱讀模式』，使用者不論是從手機或平板進行瀏覽、閱讀、分享文章都十分方便，只需點擊「閱讀」按鈕即進入閱讀模式。閱讀模式會將網頁上的廣告移除，重新排列圖片，並將字體放大，提供使用者更無擾的閱讀環境。
