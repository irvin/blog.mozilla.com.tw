Firefox 的 5 項設定讓你安心悠遊網路

「凡瀏覽過必留下痕跡」。你在瀏覽網路時，多多少少會留下個人資料。Mozilla 重視使用者在 Web 上的隱私與安全性。也因為這樣，Firefox 內建許多功能，讓使用者能安心悠遊網路。下面列出其中幾項功能與其運作方式：

  • 不要追蹤我 (Do Not Track)：許多網站會追蹤你的線上行為，再將資料販售給廣告商。如果這樣的行為讓你不舒服，就可以讓 Firefox 告知網站你不想被追蹤，讓自己的瀏覽行為保持私密性。只要啟動「不要追蹤我」功能，則只要是你瀏覽的所有網站 (包含這些網站的廣告商或其他內容供應商)，Firefox 都會通知網站「你不想被追蹤自己的瀏覽行為」。接下來就看網站是不是尊重你的要求了。但只要有越多人提出要求，網站一般也會從善如流停止追蹤。了解應如何拒絕被追蹤。
  • 清除自己的歷史記錄 (或只清除單一網站的歷史記錄)：在你瀏覽網路時，Firefox 會協助記錄使用者之前的許多瀏覽資訊 (當然 Mozilla 不會販售這些資料)，而這些資訊統稱為使用者的歷史記錄。當然任何人都可能「哎呀！」一下，點了某個鏈結卻意外錯進了其他網站。這時就可以從歷史記錄中刪除這個網站，也可以篩除整個歷史記錄。這篇文章提供逐步說明，讓使用者能管理自己的瀏覽歷史記錄。
  • 隱私瀏覽 (Private Browsing)：有時候能秘密瀏覽自己喜歡的網站也不錯。隱私瀏覽功能則能讓你悠遊網路，而不會讓網站或網頁儲存任何資訊。保護自己的瀏覽記錄。
  • 自定安全選項：我們提供多個自定選項，讓你能自己控制 Firefox 所能給予各個網站的安全層級。控制自己的安全設定。
  • 網站身分識別 (Site Identity) 鈕：這個是 Firefox 的安全功能，針對使用者目前瀏覽中的網站，可提供更多相關資訊。而透過色彩區別的系統，可讓使用者輕鬆辨識可疑網站並避免前往偽造的網頁。使用網站身分識別鈕。

這些不過是 Firefox 諸多安全功能的其中幾項而已。參閱我們的使用者隱私與安全協定。
