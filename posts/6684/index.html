慶祝 Web 的多元選擇、控制、獨立性

11月112014

Mozilla 在十年前打造 Firefox，要將網際網路留在所有人的手上，也就是提供更多選擇，要讓大家都能掌握自己的線上生活。

今天，我們邀請你站出來一同支持應該為大家所共有，而不專屬於任何人的網路。

生日就是回顧過去成就的時候。意義非凡的 Mozilla 生日，同時也是你的生日：我們一同拯救了網際網路。所謂「拯救」不是維持現狀，亦不是因任何公司的利益而控制我們的線上生活。Microsoft 在當時可說是支配了整個 Web，讓網路無法依他人的創意自由發展，卻停滯於單一公司的眼界之下。所幸有 Firefox 改變了此一現象。

Mozilla 不遺餘力的推廣線上選擇與機會，並專注建構開放標準而非專利＼私有的技術。我們讓使用者得以控制自己的線上經驗，也努力保有 Web 的開放性。我們並未建構更好的瀏覽器；卻是打造與眾不同的瀏覽器。事實證明，我們確實展現了更靈活的可能性、更好的方法、更好的未來。我們觸發了能量、競爭力、創造性，也藉此塑造出大家今天所熱愛的 Web：開放、獨立、無所不在。

身為 Mozillians，我們均努力榮耀此一傳奇，同時也是我們一直念茲在茲的責任。為了彰顯今天的 Firefox 十周年慶，我們要慶祝並再次宣揚理念。

先與大家一同慶祝。Mozilla 與眾不同，並非傳統的軟體公司。我們是全球性的非營利組織，持續推廣線上的開放、創新、機會等核心理念。不論你是才剛開始接觸 Mozilla，最近才加入專案貢獻的行列，甚或是上禮拜才開始使用 Firefox，我們都希望你能了解並一起宣揚此社群所成就的事、對世界造成的影響，以及這一路上結交的許多朋友。

到今天為止，Firefox 已橫跨桌機與行動平台提供超過九十種語言的版本，讓全球數億人享有獨立的機會與選擇。

為了慶祝此一里程碑，我們也在今天發表新產品與新規劃，要繼續打造更好的網路：

  • Firefox 週年紀念版 不僅是目前最快的版本，也將透過新功能讓大家再進一步掌握 Web 的力量。 我們將「DuckDuckGo」搜尋引擎內建於桌面版與行動版瀏覽器之中，也導入新的隱私權控制功能。可參閱 Johnathan Nightingale 的文章以了解 Firefox 週年紀念版的細節。
  • 新戰略規劃「 Polaris 」 即整合最好、最明確的方針，藉以強化線上的隱私權控制功能。 Polaris 期在匯集我們的全球社群，並公開招募民主與科技中心 (Center for Democracy and Technology，CDT)、Tor Project，以及其他組織的專家。我們剛開始會建構實驗性的反追蹤工具並與 Tor 密切合作，藉以測試且強化其網路結構。可參閱 Denelle Dixon-Thayer 的文章以了解 Polaris 的細節。
  • 首次展示的首頁磚 (Tile)，以及「興趣計分板 (Interest Dashboard)」附加元件，將取得廣告與建議內容之間的平衡；當然尊重使用者仍為首要考量。 參閱 Darren Herman 的文章可進一步了解新分頁的首頁磚規劃，還有更以使用者為中心的廣告趨勢。
  • Firefox 開發者 (Developer) 版本 是首款專為開發者打造的瀏覽器，內建的獨立工具可提升產能並建構更好的 Web 。我們要讓開發者撰寫整體的 Web，而並非單純提供單一公司所需的內容。參閱 Dave Camp 的文章以進一步了解 Firefox 開發者版本。
  • 強化的技術與搜尋功能， 確保 Mozilla 產品仍具備領先業界的優勢 ，也是賴以推動Web 平台持續向前的動力。 Javascript 效能現居領頭羊的地位，Firefox 每夜更新版(Nightly) 持續更新 E10S，並另已發表 Firefox 的 64 位元 Windows 版本，且在 3D 遊戲、通訊、安全性等方面亦各有斬獲。參閱 Andreas Gal 的文章以進一步了解細節與酷炫展示。

接下來數週到數個月之內，我們將陸續分享更多資訊。

再次宣揚的理念：我們不但尚未結束，而且才剛要開始。

與之前相較，Mozilla 認為自己目前在全世界所扮演的角色從未如此重要過。數位權、隱私權、網路中立性、線上安全性，均在在對我們的日常生活產生實質影響。線上生活的步伐與複雜度只會越來越快、越高。我們現在做的決定，會對未來線上生活產生本質上的影響。

Mozilla 正積極爭取的未來，就是要讓 Web 成為所有人創新、創造、機會的動力來源，且能留給後代子孫受用無窮。我們所打造的工具、所接觸的人們、所突破的規則，都屬於Web 保衛戰的一部分。我們也因此不斷強化使用者的線上選擇，並持續爭取眾人的網路獨立性。只要有人重視自己數位生活的獨立價值，Mozilla 就會努力提供最好的選擇。

我們的理念將專注於現今網路所面對最重要的問題，並隨著未來趨勢而不斷演變。Mozilla將繼續贏得大家的信賴，亦提供具備獨立精神且顛覆常規的方式，為所有人允諾網際網路的未來。

祝 Firefox 十週年快樂！

─ Mozilla 執行長 Chris Beard
