Firefox Beta 新增開發者工具及附加元件同步功能

2月42012

最新版本的 Firefox Beta 目前已開放測試。此次的 Beta 版將更多網頁開發工具內建於 Firefox，開發者可以直接從 Firefox 開啟查看網頁相關資訊，同時 Firefox Sync 功能讓使用者可以在多台電腦間同步附加元件。

針對網站設計師的新功能：

  • 網頁結構 3D 看：Tilt 是新的 WebGL 網頁視覺工具，能以 3D 圖形方式呈現網頁各元件之間的關係。Tilt 3D跟 Page Inspector 工具整合，對於在 HTML 結構中搜尋特別有用。若想進一步瞭解 Tilt，請參閱此篇文章。
  • 樣式編輯器：Firefox Beta 內建最新樣式編輯器，讓開發者可以自由編輯某一網頁的 CSS 檔案，而不需要關閉瀏覽器。編輯後亦可以將 CSS 檔案儲存於桌面。
  • 支 援 SPDY：Firefox Beta 如今支援 SDPY。SPDY是一個傳輸協議，作為 HTTP 的承接者，透過壓縮網頁標頭來減少網頁下載時間，同時減輕伺服器的工作量。SPDY 透過 SSL 載入網頁，讓瀏覽更加安全。若要啓動SPDY，請至 about:config 裡，將 network.http.spdy.enabled 設定為「true」。

針對一般使用者的新功能：

  • 附加套件同步：使用者可以選擇同步各個電腦/平台間的附加功能，因此可以任意在不同電腦完成之前未完成的工作。要啓用此同步功能，請見「選項」（偏好設定）中的「同步」頁。
  • 新增 Google Chrome 轉移工具：先前 Firefox 可將您儲存在 IE、Opera、Safari 等瀏覽器的資料轉移至Firefox，而此版 Firefox Beta 則又增加轉移 Google Chrome 瀏覽器資料的能力，使用者可以輕易的在Firefox 裡找到過去所有儲存的書籤、瀏覽歷史、cookies 和個人設定等。未來版本還新增會有更多功能。
