讓大家重拾對 Web 的信任

1月282014

1 月 28 日是國際資料隱私權日 (Data Privacy Day)，旨在強化大家對個人線上資訊的意識並推動相關的管理教育。Mozilla 每年都會慶祝這一天。而從去年同一天起算的一年之中，有太多新聞都涉及政府部門與商業組織濫用隱私資料，進而監視並追蹤個人在 Web 上的活動。因此對所有的網路使用者來說，今年的資料隱私權日似乎特別重要。

許多資料濫用的情事仍不斷遭人揭露，對公眾的影響也不言可喻：大家對 Web 的信任幾乎已蕩然無存。我們知道 Web 能跨越階層、文化、語言而為所有人提供強大力量，也能針對經濟發展造就無限商機。但如果公眾在自己的手機、電腦或其他任何裝置上，都無法信任 Web 對個人資訊的處理方式，人們就會大幅縮減與 Web 的互動時間，也會有越來越多人長期封鎖自己的資料。最後導致的資料保護壁壘，將對網際網路造成不可挽回的傷害並嚴重危及經濟。

上述的後果聽起來都很可怕。正因如此，Mozilla 現正傾力打造更值得信賴的線上環境，以利 Web 的永續運作。我們持續開發如 Lightbeam 與不要追蹤我 (Do Not Track) 等的服務，並提供 Firefox 此類開放源碼的透明化產品，讓全球社群能隨時檢查資料安全與否。我們更透過 Webmaker 強化對線上隱私權的教育。Mozilla 除了與立法官員密切合作之外，亦不斷遊說多方政治領袖，希望能將法律與文化環境帶向更開放的境地。

請透過這個網站進一步了解 Mozilla 是如何將隱私權擺在第一位。加入 Mozilla 一起讓大家重拾信心，並打造能滿足全世界需要的網際網路。
