支援加州的公立教育以及 Web 素養活動

7月122016

Web 素養 (Literacy；即線上的讀、寫、參與能力)，已成為 21 世紀最重要的技能之一。除了「讀、寫、算數」的傳統基礎教育 3 要素 (the Three Rs) 之外，我們認為 Web 素養理應成為第四個「R」。從開放源碼學習工具，到免費的教育課程，Mozilla 均致力傳授 Web 素養以強化個人能力。

在 2015 年，加州仍有 65% 的公立學校完全沒有資訊科學的課程。但公立學校理應設法培養學生的 Web 素養。貧乏的數位技術課程，將連帶影響甚至傷害學生與國家的未來。

今年 6 月 30 日，我們對加州最高法院遞交了一份陳情書，希望法院方面能審視〈Campaign for Quality Education v. State of California〉一案。本案重點在於加州憲法要求加州當局必須為公立學校學生提供高品質教育。Mozilla 提出此陳情書的理由在於：我們認為現今的高品質教育並未納入 Web 素養相關技術，將讓加州學生可能面臨「遭現正急遽數位化的社會淘汰」的風險。

網際網路已成為許多人日常生活的一部分，其對全球的影響力也持續增長。協助 Web 世代的學生培養其網路素養，除了是他們未來能否成功的關鍵之外，也會是拓展線上生態系統與地方經濟的基礎。「使用技術」雖為 Web 素養的重要環節，但並非全部。Web 素養更包含是否深入了解技術本質及其影響力，進而讓學生能有意義的參與網路生活。

我們希望加州最高法院能重視此一案例。包含 Web 素養在內的高品質教育，最後將為學生提供更多機會、成為更善於交流的公民，更使他們能在今日技術導向的市場中搶占先機。
