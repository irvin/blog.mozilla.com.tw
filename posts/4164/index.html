Firefox for Android 內建於全新行動裝置

11月82013

成為 Kobo 平板電腦與 Gigabyte 智慧型手機的預裝瀏覽器

【美商謀智新聞稿】Mozilla 是全球性的非營利組織，以推動網路的平等、自由、開放為宗旨，於今日宣佈 Firefox for Android 行動瀏覽器將正式內建於 Kobo 新款 Arc 系列平板電腦，以及 Gigabyte 集嘉通訊 (技嘉科技子公司) 的 GSmart Simba SX1 智慧型手機，成為此二款行動裝置的預裝行動瀏覽器。

快速、聰明、安全的 Firefox 瀏覽器，始終把使用者放在第一位，讓使用者完全掌握自己的 Web 體驗。Firefox for Android 行動瀏覽器同樣具備適應性、直覺性的設計，絕佳的自訂功能更超越其他行動瀏覽器，且其隱私與安全功能亦同樣領先業界 ─ 如隱私瀏覽模式、訪客瀏覽 (Guest Browsing)、不要追蹤我 (Do Not Track) 等功能。Firefox 亦可讓使用者跨多款裝置而同步密碼、瀏覽記錄、已開啟的分頁，進而確實掌握自己 Web 經驗。

Kobo Arc 平板電腦

Firefox for Android 行動瀏覽器已預先安裝於 Kobo 的新款平板電腦上，機種包括：Kobo Arc 7、Kobo Arc 7HD、Arc Kobo 10HD。該款平板電腦由全球電子書佼佼者 Kobo 專為重度閱讀愛好者所設計，現已於北美地區搶先上市。

不論是閱讀電子書或暢遊網海，Kobo 平板電腦與 Firefox for Android 將攜手提供嶄新的閱讀體驗。如同 Kobo 專為電子書與平板電腦所設計的「Readability」功能，Firefox 的閱讀器 (Reader) 另可移除廣告、調整圖片、放大字體而清楚呈現網頁，讓 Kobo 使用者能享受清晰又簡潔的版面。即使在離線狀態亦可開啟 Firefox 閱讀清單 (Reading List) 內的所有文章，讓你在通勤或其他無法上網 (如搭乘飛機或地鐵) 的時候，同樣能以離線模式繼續閱讀網路文章。

使用者現已可於美國與加拿大的 Kobo.com (kobo.com) 訂購 Kobo Arc 平板電腦。未來也將陸續於其它市場上市。

GIGABYTE GSmart Simba SX1 智慧型手機

Firefox for Android 行動瀏覽器也已預先安裝於 GSmart Simba SX1 智慧型手機。由 Gigabyte 所研發的 GSmart Simba SX1 為雙 SIM 卡智慧型手機，具備雙卡雙待雙通特性，並支援三方通話功能。

GSmart Simba SX1 內建 Firefox for Android 行動瀏覽器，使該款智慧型手機得以提供絕佳的 Web 瀏覽體驗，結合 21Mbps 的高速連線，使用者將能真正體驗一手掌握安全、隱私、個人化的特色。

GSmart Simba SX1 智慧型手機將於今年 11 月在台灣、俄羅斯、白俄羅斯、EEU、烏克蘭等地上市，並於 12 月起於全球發售。

Firefox for Android 資深產品經理 Karen Rudnitski 表示：「Mozilla 很高興能和 Kobo 與 Gigabyte 成為合作夥伴，讓 Firefox 成為 Kobo Arc 平板電腦與 GSmart Simba SX1 智慧型手機中的預裝行動瀏覽器。Firefox 致力讓全球使用者享有絕妙的個人化行動瀏覽經驗，也很榮幸能透過這次合作將這些體驗直接將帶給使用者。」
