Firefox Beta 測試版整合 Social API 功能！

10月122012

Mozilla 在美西時間11日釋出全新 Firefox Beta 版，適用於 Windows、Mac 和 Linux 作業系統。Firefox Beta 首度針對網站設計師，支援社交網站 API (Social API) 並置入由 Mozilla 開發的社交功能。Social API 是 Firefox 研發和實驗的結果，目的在將社交功能整合進 Firefox 網頁瀏覽器中。

Social API 如同 Open Search 搜尋模組標準，可幫助網站設計師將社交服務整合進瀏覽器，提供使用者方便又智慧化的使用體驗。一但將社群邊框欄位整合進瀏覽器後，使用者便可以隨時隨地和朋 友無遠弗屆的串連在一起，無需再另開一個新的網頁或切換分頁登入社群網站，在你瀏覽網頁間、看影片或打電玩時，都不會錯失社群網站上的任一個訊息。

我們在未來的幾週就會在 Firefox Beta 版開始測試 Social API 的功能，Firefox Beta 的測試者將可第一時間體驗此功能。同時我們也正與 Facebook 合作，測試 Facebook 的即時通 (Facebook Messenger) 與 Social API 的整合。

在可預期的未來，Social API 的功能終將支援多個社群網站，擁有把社群平台、郵件、個人財務、新聞和其他個人化應用整合進瀏覽器裡，對於想多了解社群網路 API 的使用者，可以點選這裡。

在 Firefox 內測試 Social API 的功能即將開始，敬請密切關注!
