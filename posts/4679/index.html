Mozilla 為推廣 Firefox OS 平板電腦 啟動 Contribution Program

1月72014

Mozilla 於數個月之前發佈了首款 Firefox OS 智慧型手機，而現正努力讓 Firefox OS 登上更多平台與裝置。

在諸多專案中 Firefox OS 平板電腦進展較為快速，而我們希望招募更多貢獻者協助完成程式端的開發。為了讓大家能順利參與，Mozilla 啟動了一個全新的 Contribution Program，重點在加速建構平板電腦版本的 Firefox OS 並支援其生態系統。為達成此一目標，我們將提供相關資源與平板電腦硬體測試機，以利熱心的貢獻者能專注開發。在軟體正式定案之前，我們會提供硬體測試機給全球的志工貢獻者，以儘早完成 Firefox OS 平板電腦版本。

Mozilla 與鴻海科技集團 (Foxconn) 以及其他開發者夥伴合作打造 Firefox OS 平板電腦。由於 Firefox OS 本來就是以隨時可擴充的 Web 平台所打造，因此我們可持續針對智慧型手機進行 Firefox OS 的優化，同時透過社群的協助而建構平板電腦與其他裝置所適用的 Firefox OS。

透過此 Contribution Program 完成的 Firefox OS 平板電腦版本，我們將與全世界分享。

初期合作夥伴與硬體

規格如下：

InFocus ─ 10 吋平板電腦

1280×800 解析度，24 位元高彩

16GB 儲存容量

2GB RAM

802.11 b/g/n

機身尺寸 267x171x9.7 mm

機身重量 580 g

我們將與鴻海以及未來的合作夥伴繼續為更多開發者共同擴展這個計畫。

專案運作方式：

此專案將於數個星期內正式啟動。屆時 Mozilla 將釋出更多細節，讓貢獻者能申請平板電腦測試機。

  • Mozilla 將協力完成平板電腦版本的 Firefox OS。
  • Mozilla 將提供相關開發指南，說明 Firefox OS 平板電腦版本需貢獻者參與之處。
  • Mozilla 將提供每夜更新版 (Nightly)，以利開發者掌握最新版本。
  • 完整的專案細節將於數週後發佈於 Mozilla 網站。

專案參與方式：

  • 只要你具備滿腔熱血並躍躍欲試，或是已具備 Firefox OS 核心 (特別是Gecko/GFX、Gaia、UX、系統/生產力類的 App) 開發知識的開發者，均可參與此專案。
  • Firefox OS 翻譯貢獻者 ─ 可協助翻譯 Firefox OS 平板電腦版本，以利全球市場推廣。
  • 測試者與除錯者 ─ Mozilla 需要你們測試最新版本並找出錯誤。開發者亦可加入測試與除錯的行列。

我們將於數週後提供進一步專案細節。請隨時留意更新訊息。
