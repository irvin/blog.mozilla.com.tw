Mozilla Firefox Beta 版本開放下載測試！

4月272012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，旗下著名之瀏覽器Firefox Beta 版本於美西時間26日釋出，提供Windows、Mac 和Linux 測試版本供使用者下載測試。此次的 Beta 版將著重在提升便利性的網路搜尋體驗。在首頁和分頁都做了更一目了然的便利設計，針對網站設計師也更新了開發者工具和支援 SPDY 等功能，期待愛嘗鮮的你搶先下載測試。

以下話不多說，直接進到Firefox Beta 版內容介紹。

最新版本的 Firefox Beta 目前已開放測試。此次的 Beta 版將更著重在提升便利性的網路搜尋體驗。

針對一般使用者的新功能：

  • 新版首頁：為使用者貼心在首頁上置入書籤、瀏覽歷史、設定、附加元件、下載和同步等喜好選項，讓您透過捷徑都可馬上使用。

  • 首頁的新分頁功能：當使用者打開新分頁時，Firefox 將為您把最常用及最近使用的分頁列在前面。同時，您也可以依據個人喜好、增加或移除擴充套件，並透過捷徑的方式，一點即可開始使用。

  • 新分頁介面：Firefox 為分頁歷史做紀錄，當使用者需要恢復先前使用的頁面時，您將會發現Firefox在Windows系統中，為您快速回復您先前瀏覽的頁面。

針對網站設計師的新功能：

  • 支援 SPDY：Firefox Beta 如今支援 SDPY。SPDY是一個傳輸協議，作為 HTTP 的承接者，透過壓縮網頁標頭來減少網頁下載時間，同時減輕伺服器的工作量。SPDY 透過 SSL 載入網頁，讓瀏覽更加安全。對使用者而言，支援SPDY傳輸協議的網站會有更快速的頁面下載速度。
  • 開發者工具更新：Firefox Beta 版中包含了多樣的開發者工具更新。頁面檢測器現在允許你鎖定所檢測頁面元件的CSS擬類別（pseudo- classes），樣式編輯器現在遇上以 file:// URL 讀入的檔案時，更將自動儲存修改內容，加速網頁開發者測試 CSS 的工作流程。
