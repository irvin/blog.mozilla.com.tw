Firefox Beta 測試版加入 IonMonkey 增進 JavaScript 效能

11月272012

Mozilla 在美西時間 26 日釋出全新 Firefox Beta 版供下載與測試，適用於 Windows、Mac 和 Linux 作業系統。Firefox Beta 主要包含 JavaScript 的優化以增進 Web apps 與遊戲進行中的表現效能，並開始支援 W3C Touch Events。同時，Mac 用戶還可以看到 Firefox Beta 新增對 Retina 顯示器的支援。

以下列舉重點更新：

  • IonMonkey：IonMonkey 是全新的 JavaScript 引擎 (JavaScript JIT compiler)，令執行 JavaScript 的速度更快捷，有了 IonMonkey，Firefox 跑 Web apps 的速度也會變快。
  • 支援高解析度的 Retina 顯示器：Firefox Beta 針對 Mac 用戶支援 Retina 顯示器，使用者在觀賞電影、玩遊戲、逛網頁時都可以有更好的瀏覽體驗。
  • 關閉不安全的內容：Firefox Beta 現在在 HTTPS 安全網站裡，可選擇關閉當中不安全的內容，以便保護您的隱私。這個功能目前必須先於 about:config 中啟用。
  • 支援 W3C Touch 事件：除了原先實驗性的 MozTouch 外，現在 Firefox Beta 支援標準的 W3C Touch 事件。
