Firefox Hello 讓視訊電話就如同當面交談一樣簡單

1月142015

現在撥打視訊電話，應該要像是面對面對話那樣簡單，但實際上不然。你必須先填寫自己的電子郵件、設定帳戶、送出個人資訊、下載軟體之後才能開始交談。

Mozilla 不久前介紹的「Firefox Hello」，是第一款直接內建於瀏覽器之中的全球通訊系統，目的就是要讓大家能更輕鬆的維繫情感。

我們縮減了數個步驟，簡化了整個撥打程序。請先在功能表或「 自訂」面板上找到「Hello」圖示，點擊「開始對話 (Start a conversation)」鈕即可通話。在啟動對話之後，就會開啟視窗並顯示自己鏡頭所拍攝的畫面，直到你邀請的通話者點擊了連結並加入對話為止。在你等待對方加入對話之時，仍可繼續上網不受限制。等到受邀人加入對話，Hello 就會發出聲音通知你，且本身的圖示也跟著轉為藍色。

每個對話視窗都提供專屬且可分享的網址，讓兩個人能輕鬆以聲音或視訊方式溝通。你也能另外建立多個對話視窗並分別命名，不須每次都還要建立新連結，也能找到常聊天的對象並開始對話。上述這些方便功能都完全不需要再設定帳戶，也不用填寫自己的個人資訊。舉例來說，你可針對每週例行的視訊會議命名為「努力打拼」；或可把聯繫家人用的對話命名為「溫馨家庭時間」。系統將保留你儲存的對話以便你隨時使用。


Firefox Hello 中的對話視窗

####

另外，只要雙方都擁有「Firefox Accounts」就能直接聯繫他人。如果你想通話的對象就有 Firefox Account 而且正在線上，就可直接透過 Firefox 撥打給對方。當然，你可以在任何電腦登入自己的 Firefox Account，不論你在家或辦公室，別人都能隨時聯繫上你。若你已有 Google 帳戶，也能輕鬆將聯絡人資訊匯入自己的 Hello 通訊錄。只要在通訊錄上點擊「匯入聯絡人 (Import Contacts)」，接著登入 Google 帳戶就等於允許資料匯出作業。

一同體驗 Web

Mozilla 與長期合作夥伴 Telefónica 一同開發 Firefox Hello，並搭配 TokBox 技術不斷強化此服務的功能與品質，而這一切才剛起步而已。我們期待 Firefox Hello 未來拓展為完整的 Web 通訊工具，且現正測試如畫面分享與線上協作的新功能，要讓使用者能在視訊通話期間擁有更多附加服務，包含能一同規劃假期、線上購物，甚至合作打怪。有太多可能的有趣應用。

你還在等什麼？今天就找親朋好友來玩玩「Hello」吧！先從這 裡著手入門。我們希望你能享受 Hello，也將陸續分享更多新功能。
