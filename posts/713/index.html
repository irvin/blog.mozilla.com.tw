Firefox 安全機制亮眼升級，再推遊戲全新開發功能！

7月182012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012年推出以HTML5 為基礎的行動作業系統Firefox OS。今於美西時間17日釋出全新Windows、Mac和Linux 版本適用的Firefox 瀏覽器。此版本旨在強調獨一無二的安全機制，同時提供程式開發者（如遊戲開發等）最強大的網路應用工具。

首次預設加入 Google 安全搜尋，可從基層開始保護您的網路資料和搜尋結果，防止資料透過未加密 WiFi 等方式外流，避免他人趁機蒐集您的網路行為。目前 Firefox 中暫時僅有Google對用戶提供這樣的安全搜尋服務，期待在未來有更多搜尋引擎加入此行列。此外，針對網站識別與加密狀態等相關資訊，Firefox 也提供更簡單的顯示方式，請見優化網站資訊顯示方式。

對遊戲開發者而言，新版 Firefox 支援 Pointer Lock API (滑鼠游標API)，讓開發者在設計如第一人稱射擊遊戲時，對滑鼠可做更好的控制。Firefox 如今也支援 Mac OS X Lion 10.7平台上的全螢幕模式，在進行視訊和Web遊戲 (如 Mozilla 的BrowserQuest) 時，擁有更豐富和身歷其境的體驗。
