Firefox OS 2.6 版本與智慧型手機

2月52016

Mozilla 日前宣佈將停止透過電信營運商的管道發售智慧型手機。此一決策的主要理由，即因為我們無法提供高差異性的消費者價值以做出市場區隔，也無力形成完整的生態系統。我們的團隊與社群均盡心盡力演出，打造出眾人驚艷的平台，但整個態勢不足以讓 Mozilla 在智慧型手機的商業遊戲中脫穎而出。

我們仍正努力鋪陳著下一步，期能打造「連網裝置 (Connected Devices)」解決方案的雛形，並藉以延續 Firefox OS 的未來。為了支援此一策略，我們正透過 Firefox OS 2.6 版本為智慧型手機專案劃下完美句點。此一計畫另將針對智慧型手機、Firefox 桌面版、Firefox 行動版等多個平台，將同步關閉 Firefox Marketplace。

雖然 Mozilla 的智慧型手機 OS 開發步伐將停止於 2.6 版本，但 OS 將持續用於智慧電視與深具未來潛力的其他裝置。一如往常，我們的平台仍開放給貢獻者與熱血人士，隨時歡迎大家為 OS 的未來盡一份心力。

我們與日本松下電器 (Panasonic) 的智慧電視產品線計畫，不會有任何改變。Mozilla 很高興能與 Panasonic 在 SmartTV 端繼續合作 Firefox OS 2.6 及未來版本，持續於智慧電視上建構豐富的 Web 內容。

相關心血與其建構出的程式碼，都是連網裝置與 Web 平台的未來基石。我們另新增了超過 30 組 WebAPI，並證明 Web 靈活性絕對足以支援從手機到電視的多樣產品；這當然也代表下一個「連網裝置」階段的絕佳起始點。
