Firefox Marketplace 展現 Web 的力量

11月202013

跨平台提供全球性與地方特色 App

全球性非營利組織 Mozilla 致力打造 Web 的美好未來，並以推動網路的平等、自由、開放為宗旨。我們所建構的產品 (如 Firefox) 除了能秉持核心價值之外，也深獲全球使用者的喜愛。

Mozilla 與全球的開發者、硬體製造商、電信營運商合作，透過 Firefox OS 於行動產業中提供更多選擇，並讓全世界更多人能擁有智慧型手機。Firefox OS 智慧型手機是首款全以 Web 技術打造的裝置，以漂亮、直覺、簡潔的使用經驗，達到使用者對智慧型手機所期待的效能、個人化、平實價格等特色；均是目前其他手機所無法比擬之處。

Firefox OS 提供二大管道可發掘並使用 App。首先就是智慧型手機的全新概念 ─ 自訂性 App 搜尋功能，可隨使用者情境而調適 App 搜尋作業。其次就是透過 Firefox Marketplace 輕鬆找出最高人氣的全球性 App，以及相關的當地特色 App。

Firefox Marketplace 跟著 Firefox OS 首次發表而啟用以來才不過幾個月，開發者就已經提交數千個 App 並正式上架，其中還有多個 App 是由 Firefox Marketplace 所獨佔，如《Manana》閱讀 App 與《Senzari Music Graph》。

而另外幾個高人氣的全球性 App 則包含 Badoo、Box、Chatty (片語翻譯機)、Cut the Rope、Discovery、Easy Taxi、Evernav、Facebook、KAYAK, Manana、multiConvert、Poppit (由 Pogo 公司出品的遊戲)、Projector (待辦事項管理)、Rhapsody、Sticky (相片貼紙產生器)、SoundCloud、The New York Times、The Weather Channel、TimeOut、TLC、TMZ、Tuenti、Twitter、wger (運動與減重)、World Radio Player、YouTube。

Mozilla 和合作夥伴不斷與全球開發者相互交流，確保使用者能擁有其他平台尚未提供的絕妙本地特色 App。

目前受歡迎的 Firefox OS 本地特色 App 包含：

  • 西班牙：Despegar、Tuenti、El Pais、Cinco Dias、La Caixa、Tiempo
  • 波蘭：Tabilca、Sport.pl、Pogoda、WP.pl
  • 哥倫比亞：Despegar、Guru、Gol Caracol、El Tiempo
  • 委內瑞拉：Globovision、Despegar、Cinex、Ultimas Noticias、Tu Descuenteon、Lider en Deportes
  • 祕魯：Guia Desastres、Despegar、Peru Cookbook、OLX
  • 烏拉圭：Despegar、ibus、OLX
  • 德國：Kicker、Leo、Immobilienscout24、General-Anzeiger Bonn
  • 墨西哥：CNN Mexico、Despegar、Media Tiempo、Quien、El Universal、Chilango、M3
  • 匈牙利：Mozi24、Aprod
  • 巴西：UOL、OLX、Decolar、Guias Mais、BOL、SporTV、Dieta e Saúde、Editora Globo、Galinha Pintadinha、Kekanto、Vivo Chat

「Mozilla 致力將 Web 的力量交到所有人手中。不需再透過專利 App 與封閉式平台，Firefox OS 也能確實體現 Web 的承諾與潛力。」Mozilla Firefox Marketplace 副總裁 Rick Fant 表示：「在啟用 Firefox Marketplace 之後，我們很高興能看到開發者於短時間內展現高度興趣，並讓我們感受到 Mozilla 率先發表 WebAPI 後所激發的創造力與創新概念。」

Mozilla 透過新 WebAPI 解放 Web 的力量，並使 Web 成為全球最大型的Marketplace。這些 WebAPI 可存取手機的所有硬體，包含相機、簡訊、藍牙、加速規、撥號鍵盤等等。現在全世界開發者均可設計豐富且有趣的 Web 經驗，跳脫先前專利原生程式碼必用於特定平台的限制。

Firefox OS 更為開發者提供靈活的付款選擇，如電信帳單代繳，或以開放式 API 執行 in-app payments (應用程式內部付費機制)；且 Mozilla 正努力促使後者成為所有 Web 瀏覽器與作業系統的標準，以期能於所有平台之上運作。Mozilla 現亦與 Inneractive 網路廣告商一同建立簡易的函式庫，以期將廣告整合至 Firefox OS 的 App 或遊戲之中。

Firefox OS 將於行動產業中設定平等的起跑點，並解決如 App 撰寫方式過多 (不同平台均有其專屬方式)，以及提升 App 可見度等問題。Firefox OS App 的移植程序甚為簡單且迅速，因此開發者不需擔心多個作業系統的相容性問題。Firefox OS App 可跨平台、跨裝置使用，其內容完全受開發者所掌握，而不會受限於特定裝置或作業系統。

也就是說，開發者不需取得特定權限，即可建立並發佈 App，且內容亦不會受制於獨特裝置或平台之中。

國際數據資訊 (IDC) 研究機構副總裁 John Jackson 表示：「Firefox OS 正在許多新興市場中蓄積自己的潛能，並將為 Web 生態系統帶來絕佳優勢。此成長態勢更確立了電信營運商對 Firefox OS 的策略，並對消費者未來的採用情況充滿信心。在此高競爭度的市場中，其他欲成為第三大智慧型手機的平台正努力確立其定位、規模、永續性的同時，更促使 Firefox OS 進入擴張的第二階段。Firefox OS 將持續為其目標市場添增合適的新功能，找到消費者與當地物價水準之間的價格平衡點。而更重要的是，Firefox OS 將吸引新的 App 與內容合作關係繼續投入，最後形成平台的命脈，並將進一步提升消費者採用率，再次為 Web 的開發者社群激盪出更多創新火花。」

Firefox Marketplace 已隨附於 Firefox OS 智慧型手機中，並可於 Android、Windows、Mac 版 Firefox 中預覽其功能。

Firefox OS 智慧型手機已於今年七月時首次發佈，且年底之前將於全球超過十個市場中現身。
