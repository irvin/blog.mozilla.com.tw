全新企劃：Phones for Apps for Firefox OS

5月102013

各位 HTML5 App 的開發者，Open Web 正呼喊著你們！

我們都知道你已經就定位、摩拳擦掌、撰寫程式碼、測試、閱讀記錄文件、下載並執行 Firefox OS 模擬器 (Firefox OS Simulator)。你甚至已經準備好「Send to Device」，現在就等著能親手把玩裝置。

我們今天發表你心中渴望的新企劃，就叫做：Phones for Apps for Firefox OS。

Firefox Marketplace 現身於 Geeksphone Keon

你可能已經針對 Chrome、webOS、Blackberry WebWorks、PhoneGap 等平台建立了 App。或許你已於桌面環境上撰寫了漂亮的 Web App，現在想把自己的作品移植到行動裝置。又或許你是等著放暑假的學生。你也正巧不住在哥倫比亞的首都波哥大 (Bogota)，也不住在波蘭的首都華沙 (Warsaw)，或任何後續 App 研討會舉辦地點附近。

任何地點都可參加

不論你身在何方，只要你心裡冒出酷炫的 App 概念，而且自己也能寫出雛形來，那請把握今年夏天即將問市的 Firefox OS，我們期待在 Marketplace 中看到你撰寫的 App。光是跟你講這些當然不夠。我們還要加碼送你 Firefox OS 開發者預覽手機，讓你能順利開發自己的構想。

今年夏天，在特定國家開賣消費者版本的 Firefox OS 手機之前，你絕對該把握這個全世界僅此一次的機會，搶佔 Firefox Marketplace 的開發優勢。不論是東歐、南美洲，或任何手機開賣國家的末端消費者/使用者，都一定會安裝好玩或實用的 App：遊戲、工具、公用程式，還有當地的相關新聞、體育、旅遊、娛樂、評測應用，當然也少不了社交經驗分享。你現在就應該著手撰寫 App 並立刻提交給我們！

馬上申請

告訴我們你想撰寫或移植的 Firefox OS App。一但我們採納了你的建議或想法，就會立刻寄一支 Geeksphone Keon 給你。手機數量不多而且時間不等人，想要就快點行動。此企劃活動只到 5 月底止；如果手機在 5 月底前發送完畢，同樣提早結束。一個 App 企劃僅限索取一支手機。我們很期待大家的作品與想法。現在馬上動手開始吧！

點我到這裡申請。
