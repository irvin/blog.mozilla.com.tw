Mozilla 的 A-Frame 為虛擬實境新網站 #360Syria 提供震撼效果

國際特赦組織 (Amnesty International) 的新「#360Syria」虛擬導覽網站今天上線，呈現敘利亞政府對阿勒坡 (Aleppo) 城失陷之後進行精確轟炸所造成的斷垣殘壁。此網站 (www.360Syria.com) 即透過 Mozilla 的「A-Frame」技術，向外界展示了「從天而降的恐懼 (Fear of the Sky)」。

像 #360Syria 如此讓大家能以虛擬實境 (Virtual Reality；VR) 方式瀏覽阿勒坡城現況的網站，就是 WebVR 的全新應用案例。科技，可為平常鮮受關注的人們發聲；除了藉此引起更多關注與同情之外，亦能將現實情況揭露給更多世人知道。

「#360Syria」網站融合了特別設計的 360 度環景照片與敘事方式，再由國際特赦組織所訓練的敘利亞媒體行動者彙整 3D 資料圖像與影片。另與舊金山的 Junior (www.junior.io) 設計與技術公司合作架設此網站。

開放源碼的「A-Frame」框架簡化了 WebVR 開發過程，並透過 HTML 輕鬆打造 WebVR 體驗。可輕鬆擴充的 A-Frame 基礎架構幾乎能產生無限組合，因此提供極高自由度的創意。此一框架是由 MozVR (即 Mozilla 的虛擬實境研究團隊) 設計並維護。不論是剛接觸 VR 技術的開發者，或是需要更高彈性的資深工程師，均不致過長學習時間就能順利入門。

Mozilla 的目標之一，就是為 Open Web 提供高效能且反應迅速的 VR 技術。我們所發表的開源 A-Frame 可於 Web 上建構 VR 經驗。Web 開發者可先繞過如 WebGL 的複雜 3D API，僅需一行 HTML 程式碼就能架設 VR 網站。

我們希望 A-Frame 在成為 WebVR 開發工具的基石之餘，能吸引更多、更具建設性的貢獻成果，協助培養 VR 的 Web 開發者並帶來更豐富的使用體驗。
