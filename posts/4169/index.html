Firefox 滿九歲了！生日快樂！

11月82013

全球性非營利組織 Mozilla 致力打造 Firefox 成為有趣、安全、快速的最佳 Web 瀏覽器，持續堅持網路創新、開放、機會的理念與價值。從第一次發表 Firefox 當天起至今已過了九個年頭，期間除了協助 Web 不斷進步之外，也讓 Web 成為這個時代最有價值的資源之一。

我們很高興在 11 月 9 日這一天，有 Mozillians 全球社群加入一同慶祝 Firefox 的九歲生日。在此列出過去一年來 Mozilla 獻給 Web 的九個最佳禮物。

  • Firefox 加入行動領域： Firefox 持續為 Web 帶來重大的影響，不僅讓 Firefox 支援 Windows、Mac、Linux 等系統，更進一步延伸至 Android 的平板電腦與智慧型手機。隨著越來越多人透過行動裝置瀏覽 Web，我們的 Firefox for Android 行動瀏覽器也持續提供絕妙的使用經驗，並於 Google Play Store 獲得 4.5 顆星的高評價。
  • 持續茁壯的 Firefox 家族： Mozilla 今年讓 Firefox 家族喜添新生力軍 ─ 完全以 Open Web 技術打造的 Firefox OS 智慧型手機。
  • 速度： Firefox 比之前更快。和去年的 Kraken 與 Octane 測量基準相較，Firefox 分別提升了74% 與 88% 的速度。而隨著 JavaScript 進化和 asm.js 新技術的誕生，Web App 與遊戲的速度也提升達 25%。最近更由 Tom’s Hardware 網站稱呼 Firefox 為「新的速度之皇」。另由於 Firefox 本身的穩定性、啟動時間、頁面載入速度，更榮登 Web Browser Grand Prix 的冠軍。
  • 社交功能： Firefox 努力整合社交網站與服務，以期使用者能輕鬆且迅速聯絡自己的親友。Mozilla 在去年宣佈 Facebook 成為 Firefox 第一個社交功能整合夥伴之後，不久也加入了 Cliqz 與 Mixi。我們另正積極與更多社交網站與服務洽談，以提供更為個人化、更高自訂性、更多元的方式。
  • 隱私性： 根據 Ponemon Institute 的研究報告，Mozilla 榮幸獲選為 2012 年最受信賴的網際網路公司。我們剛發表了 Lightbeam 作為 Firefox 附加元件，可讓使用者了解每天在 Web 上發生互動的第一方與第三方公司。
  • 安全性： Firefox 現提供封鎖混合內容 (Mixed Content Blocker) 功能，確保使用者在 Web 上的安全性。Firefox 另具備外掛執行許可 (Click to Play) 功能，讓使用者決定是否要讓網站使用外掛程式，藉以保護使用者並讓瀏覽器執行更順暢。
  • 最佳開發者工具： Firefox 現提供強大的開發工具，不論是在桌面或行動裝置上，Web 開發者均能針對 HTML、CSS、JavaScript 進行編輯與除錯作業。最近發表的工具則包含：應用程式管理員 (App Manager)、適應性設計檢視 (Responsive Design View) 模式、網路監測器 (Network Monitor)、頁面檢測器 (Page Inspector)、JavaScript 除錯器 (JavaScript Debugger)。
  • WebRTC ： Firefox 現可直接進行視訊通話與檔案分享作業，而且完全不需再借助於其他具備 WebRTC 功能的外掛程式或第三方軟體。
  • Web 遊戲新發展： 過去一年來，我們的開發者與工程師不斷努力讓 Web 成為高階遊戲的開發平台。而最艱難的幾項挑戰之一，就是要證明 Web 確實能作為真正的開發平台。Mozilla 開發全新的超快速 JavaScript 子集 ─ asm.js，讓開發者能直接在 Web 上建構高系統需求的應用程式。WebAudioAPI 現亦加入 Firefox 而讓 Web 成為更完善的絕佳遊戲平台。

Firefox 不久將會再新增其他絕妙特色，例如新設計的瀏覽器介面，並針對 Windows 8 平板電腦進行優化。而明年又將為 Web 帶來哪些嶄新創意呢？且讓我們一起拭目以待。快到 Twitter 透過 #HappyBirthdayFirefox 一同為 Firefox 慶生，並大聲說出你對 Firefox 最深刻的感想當做祝福吧！
