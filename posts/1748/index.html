新版 Firefox 桌機、行動盛大推出！

1月92013

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012 年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。今於美西時間 8 日釋出全新 Windows、Mac 和 Linux 版本適用的新版 Firefox 瀏覽器，以及行動版專用的 Firefox for Android。此版本針對桌機版 Firefox，主要加入可讓執行速度上升 25 個百分比的全新 JavaScript 引擎「IonMonkey」，以及支援 Mac 用戶的高解析度 Retina 顯示器。同時間釋出的行動版 Firefox for Android 加入最人性化的強大搜尋建議 (Search suggestions) 功能，決心提供使用者無比絕佳桌機與行動瀏覽體驗。

本次 Firefox 桌機版主要加入名為「IonMonkey」的全新 JavaScript 引擎 (JavaScript compiler)，可使 Web Apps 以及遊戲進行中的表現效能提升 25 個百分比，執行速度更快捷。您可點選由 Mozilla Developer Network (開發者社群網站) 開發，由 HTML5、WebGL 和 JavaScript 為技術的網頁遊戲 BananaBread 實際體驗。另外，新版本同時針對 Mac 用戶支援高解析度的 Retina 顯示器，使用者在觀賞電影、玩遊戲、逛網頁時都可以有更好的瀏覽體驗。

新版 Firefox for Android 全新內建的「搜尋建議」(Search suggestions)讓你不用完整輸入想搜尋的關鍵字也能找到相對應的內容。當你開啓 Firefox 後 你將看到請你開啓「搜尋建議」的選項，點選 YES 完成後即可於網址列（又稱 Awesome Bar，智慧位址列）輸入詞彙開始使用。為了您的使用與隱私安全，「搜尋建議」是以 HTTPS 加密連線產生。同時，Firefox for Android 新增警告頁面，當使用者前往惡意網站進行可能的下載動作前，Firefox 將提醒您這可能是網路釣魚或包含惡意程式的網站，保護使用者遠離惡意網站。
