Mozilla 任命 JavaScript 發明人 Brendan Eich 為新任執行長

3月252014

現任亞洲區營運總裁兼行動裝置資深副總裁 宮力博士升任為全球營運長

Mozilla 董事會今日正式宣佈由 JavaScript 發明人、 Mozilla 共同創辦人兼現任技術長 Brendan Eich 升任為執行長。另外現任亞洲區營運總裁兼行動裝置資深副總裁的宮力博士，則升任為營運長，將同時掌管包含行動與研究 (Mobile & Research)、雲端服務 (Cloud Services) 在內等多項業務。與 Brendan 同樣身為共同創辦人的 Mitchell Baker 則仍續任 Mozilla 基金會主席，持續領導 Mozilla 專案以繼續推動 Web 的平等、自由、開放之使命。

歷經近 1 年的尋覓，Mozilla 正式宣佈由 Brendan Eich 接任執行長職位。在尋找適合人選期間，代理執行長 Jay Sullivan 持續以熱情領導 Mozilla，並對各項專案產生重大影響。Mozilla 在 Jay 的領導之下催生了多項重要產品，包含 Firefox 瀏覽器升級了多個版本，也正式發表了  Firefox OS。Jay 在 Mozilla 服務 6 年後，將離開 Mozilla 發掘更多新的機會。

Brendan Eich 從 1998 年開始，就一直與 Mozilla 各項開發專案息息相關。Brendan 在技術與產品領域均具備專業水準，而 Web 部分當然也不落人後。他的聰明才智再加上對技術方面的視野，讓他過去 20 年不僅在 Mozilla 發光發熱，同時也在 Web 領域有極大成就。Brendan 所創造的 JavaScript 現已成為 Web 的關鍵技術。此外，他也將創辦 Mozilla 時的視野和魄力帶入目前諸多專案中。這些特質都是 Mozilla 執行現有計畫 (如 Firefox OS 與雲端服務) 所不可或缺的獨特資產。Brendan 與 Mitchell Baker 共事長達 15 年的絕佳默契，將是 Mozilla 不斷茁壯的必要條件。

現任亞洲區營運總裁兼行動裝置資深副總裁的宮力博士在升任營運長之後，將一併掌管包含雲端服務 (Cloud Services)、資訊科技 (IT)、Firefox Marketplace、行動與研究 (Mobile & Research)、平台工程 (Platform Engineering) 等在內的多項業務。

Mozilla 董事會成員 Reid Hoffman 指出：「Mozilla 持續為 Web 的開放而發聲。不論是消費者或開發者，都能免受商業化、獲利導向技術的限制，確實且真誠的表達自己的心聲。Mozilla 組織與其 Firefox 產品，將繼續讓 Web 能兼顧所有人的需求，而不只一味迎合最高利益的目標。Brendan Eich 身為 Mozilla 共同創辦人與 Web 技術的創始者，將透過自己最了解的核心價值與技術視野，繼續專注領導 Mozilla 向前邁開大步。」

全球每一位 Mozilla 貢獻者與使用者都驅使著 Mozilla 不斷向前並致力完成使命，讓 Mozilla 不僅擁有獨特的運作方式，最終也將主導 Web 未來的走向。Mozilla 所完成的多項專案均完成重要成就，包含後續版本的 Firefox 將提供嶄新的使用者經驗，Firefox Accounts 也將帶來更輕鬆的同步功能與其他優點。此外，與 Unity Technologies 以及 Epic Games 之間的合作，相關工具讓開發者不需任何外掛程式，即可提供更高品質的遊戲。而 Firefox OS 現已和 4 家電信營運商與 4 家裝置製造商合作，現身於全球共 15 個國家之中。在全球最盛大的行動產業展覽 ─ 2014 全球通訊大展 (MWC 2014) 之後，Mozilla 與 Firefox OS 已進一步提高在行動產業能見度並深獲肯定。

Mozilla 執行長 Brendan Eich 表示：「我首先要感謝 Jay Sullivan 這幾年對 Mozilla 與 Web 所作出的貢獻。他一直熱心且積極領導 Mozilla，尤其是去年發表的 Firefox OS 更成為 Mozilla 的重要里程碑。而我很榮幸能接著領導 Mozilla，期許大家能朝著產品與專案的整體目標勇往直前！」
