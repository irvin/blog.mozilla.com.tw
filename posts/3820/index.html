Firefox for Android Beta 擁有更多自訂選項

9月252013

Firefox for Android Beta 現已供下載與測試之用。此次更新除了改善 UI 之外，另加入了「訪客瀏覽 (Guest browsing，暫譯)」功能，讓使用者能和親友共享行動裝置上的 Firefox。另具備如遠端除錯 (Remote debugging) 的新開發工具，並提升了 SkiaGL 的效能，將提供更快、更好的 Web 經驗。

  • 訪客瀏覽： 訪客瀏覽模式可讓 Android 裝置擁有人先行登出目前的 Firefox 瀏覽狀態，以區隔出自己的書籤、瀏覽記錄、已開啟的分頁，再讓訪客能同樣透過 Firefox 瀏覽 Web。當訪客使用完畢，裝置擁有人只要關閉訪客瀏覽視窗，就會一併刪除訪客的瀏覽資料，並隨即恢復裝置擁有人之前的瀏覽狀態。
  • 設定桌布與聯絡人相片： 從現正瀏覽的網頁中，使用者可直接將任何圖片設為桌布，或指定為任一聯絡人的相片。只要長按喜歡的圖片就會出現選單，此時再點選「Set image as」即可將網頁圖片設定進自己的裝置中。
  • 更新「設定」選單： 「設定」選單現具備全新外觀，讓你能更輕鬆自訂個人風格的 Firefox。新選單將類似的設定 (如「同步」、「自訂 (Customize，暫譯)」、「隱私 (Privacy，暫譯)」) 歸納為同一分類，讓使用者更直覺的自訂 Firefox 風格。
  • 新增語系： Firefox for Android Beta 新加入了烏克蘭文、愛爾蘭文、羅馬尼亞文，現已提供 27 種語言。
  • 阻擋混合內容 ： 將停止載入非加密 (HTTP) 的混合內容，進而保護安全加密 (HTTPS) 網站。此功能可阻止惡意人士透過混合內容而讀取或修改安全加密網頁。
  • 支援網址列快捷鍵： 附加元件開發者可為附加元件設定快捷鍵，並將之置於 Firefox 網址列上。不論使用者正瀏覽哪個網頁，均可點選快捷鍵圖示而立刻啟用該附加元件。
  • 遠端除錯： Firefox for Android Beta 的遠端除錯器 (Remote Debugger)，與 Firefox 桌面版所使用的開發工具完全相同，開發者將可順利進行網頁除錯，輕鬆開發更多 Firefox for Android 的相關功能。
