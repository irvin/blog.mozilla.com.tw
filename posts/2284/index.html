Social API 日益強大，Firefox 測試版加入更多社群廠商！

4月102013

今天開始，我們在 Firefox 測試版中納入更多 Social API 廠商，包含 CliqZ、MSN Now、Mixi。而且微博 (Weibo) 現在也加入中國的 Firefox Nightly 測試行列。

Social API 可提供如社群邊欄、工具列提示按鈕等功能，讓你可輕鬆聯繫自己的親朋好友。

隨著 Firefox 透過 Social API 邊欄整合越來越多的服務，使用者將能隨時知道朋友與家人的動態，亦可隨地追蹤自己感興趣的話題與新聞。舉例來說，你可隨時連上自己最愛的社群網站或服務，卻又同時打電動、瀏覽網頁、觀看影片等。

若要測試 CliqZ，請先升級至最新版的 Firefox，再前往 CliqZ 啟動頁面並點選「Install」。

若要測試 MSN Now，請先升級至最新版的 Firefox，再前往 MSN Now 啟動頁面並點選「Turn On」。

若你目前擁有日本 Mixi 的帳號，請至 Mixi 頁面並點選「Turn On」。

在發佈正式版本的 Firefox 之前，我們需要測試人員協助了解這些全新社群功能的穩定度。若要反應意見，可點選「Help」再點選「送出反饋 (submit feedback)」。
