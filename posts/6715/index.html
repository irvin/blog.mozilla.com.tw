祝賀 Firefox 的十年

Mozilla 在十年前打造 Firefox 以提供更多上網選擇。當時的 Web 不僅是單一文化的環境，且唯一的上網方式更由獨佔了作業系統的公司所提供。不僅是我們，相信許多人也認為 Web 應該有其他的獨立選擇。今天有數億人因為相信 Mozilla 與眾不同，所以也相信獨立的 Firefox 時時將使用者擺在第一位。

我們很高興能隨著十週年紀念日發佈 Firefox 特殊版本，且新功能將讓使用者更進一步掌握自己的線上生活。

Mozilla 不斷聽到使用者詢問應如何妥善控制自己的線上資訊。哈里斯民意調查 (Harris Poll) 代表Mozilla，於 2014 年 10 月對七千多名成人進行了全球性的線上問卷。結果顯示有 3/4 受測者 (74%)認為網路公司取得過多的使用者資料；另有超過一半受測者 (54%) 想要「忘記」自己曾在網路做過的事情 *。

Mozilla 為了能交還更多控制權給使用者，首先就是將 DuckDuckGo 搜尋功能內建於Windows、Mac、Linux、Android 版本的 Firefox 中。「搜尋」其實屬於高度隱私的線上生活。你所搜尋的事物就能反映出自己的大部分特質。雖然 DuckDuckGo 同樣會給你搜尋結果，卻不會追蹤你的人格特性或你所搜尋的標的。但其他搜尋引擎卻可能追蹤之後再強化搜尋結果。我們認為這應該是使用者可自行選擇的地方。

我們這次也為 Firefox 新增了「忘記 (Forget)」功能，可讓你輕鬆要求 Firefox 清除某些你最近的網上活動。「忘記」不會詢問一堆複雜的技術問題，而只會問你：你要忘記多少事情？一旦你要求Firefox 忘記最近 5 分鐘、2 小時、24 小時內的事情，系統就會自動幫你完成。許多人在和親友共用電腦之餘，很容易忘記先開啟「隱私瀏覽」視窗。現在簡單易用的「忘記」就能迅速清除相關資訊。

在你更新 Firefox 時，就會看到我們新增的操作指南，帶領你逐步看過 Firefox 的特定隱私選項，其中就包含 DuckDuckGo 與「忘記」。而 Firefox for Android 同樣具備新的「Privacy Coach」，內含設定與附加元件的資訊，可讓你控制自己的行動瀏覽器。

Mozilla 的週年紀念內容當然沒有忽略了偏技術性的使用者。針對其他關注線上隱私與控制權的團體，我們剛發表新的「Polaris」隱私方案，藉以整合雙方對隱私權的努力成果。另更為 Web 開發者提供新的「Firefox 開發者版本 (Developer Edition)」，內含絕佳的新開發工具。Mozilla 內容服務團隊將持續多項實驗，要讓瀏覽器能推薦相關內容，也要讓使用者能獲得更高的控制權。我們的工程與研究團隊也將繼續帶來絕妙的新功能與效能。

十年了，你應該也已經清楚 Mozilla 所堅持的使命感。我們是全球性的非營利社群，恪守清楚的核心價值：使用者的選擇與控制權，以及成為眾人生活中心的 Web 之創新與開放性。各款瀏覽器都依不同的價值理念而誕生，因此你的選擇格外重要。在 Firefox 的十歲生日，Mozilla 要感謝所有人的支持與信任。只要你選用 Firefox，就讓我們說的話更有份量；同時你也成為此數百萬人社群的一份子，將我們共有的價值理念編織成為廣大網路。沒有你就沒有 Mozilla。希望大家都能享受新版本 Firefox。

─ Firefox 副總裁 Johnathan Nightingale

  • 本次線上問卷期間為 2014 年 10 月 22 ~ 29 日，遍及英國、法國、德國、巴西、印度、西班牙等國。由 Harris Poll 代表 Mozilla 對年齡 18 ~ 64 歲的 7,077 名成人，使用該公司的「Global Omnibus」進行問卷。而年齡、性別、種族、教育、地區、家庭收入等指數，另依照受測人員的人口比例而加權。此份資料亦進行適當的加權，藉以反應成年網路人口的組成情形。若需要包含加權變數在內的完整問卷資料，請聯絡 press@mozilla.com。
