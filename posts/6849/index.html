Firefox 搜尋功能之改變

12月122014

先回想你上週曾經搜尋過的東西：旅行規劃、要送家人的禮物、身體的健康警訊、久未聯絡的朋友。搜尋服務已經成為我們線上生活的重點，也屬於更私人、更隱私的活動範疇。我因此對 Firefox 即將發生的變化覺得特別開心。Mozilla 為你打造了 Firefox，亦不斷要讓 Firefox 更貼近你的線上生活。

一如執行長 Chris 不久前所宣佈的事情，Firefox 不再提供單一的預設搜尋服務。這一點特別重要：Firefox 現已提供超過 80 種語系，也能讓各語系各自選擇最適合的預設服務。當然，Mozilla 仍同樣內建了多個搜尋服務供你選用，並能安裝新的服務。

針對美國地區的使用者，新預設的 Yahoo 亦提供前衛的 Yahoo 搜尋體驗。我們與 Yahoo 合作，為 Firefox 使用者開發特殊功能，將從最佳網站拉出更豐富的搜尋結果，如整合 Yelp 上的餐廳評價、Tripadvisor 上的旅行感想，還有更多功能。Yahoo 也承諾其所有設定均將尊重 Firefox 使用者的「不要追蹤我 (Do Not Track，DNT)」要求。只要 Firefox 使用者選擇不要追蹤己身的 Web 活動，Mozilla 就提供能夠尊重相關選擇的預設搜尋服務。

你會發現 Firefox 最大的變化就是新搜尋框。我們聽到使用者的意見：除了常見的預設服務之外，也希望某些特殊項目能直接送入其他的搜尋服務 (如 Wikipedia)。現在只要輕鬆點擊就能選用其他搜尋引擎，且不用更改自己原來的預設值。

如果你還沒能看到 Firefox 的變化，也不用著急。像是搜尋這類的重點服務，Mozilla 均謹慎跨出每一步，也就是說這項功能會較晚推出 (如果你等不及了，也可參考其他預覽版本)。

對你、對我們，甚至對整個 Web 來說，搜尋服務的差異性很重要。因此相較於其他瀏覽器，Firefox 樂於提供更多搜尋選擇。Mozilla 希望你能享受 Firefox 所強化的搜尋功能，也希望能在 2015 年繼續為你帶來更多、更好的功能。

─ Firefox 副總裁 Johnathan Nightingale
