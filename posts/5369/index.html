Firefox OS 與 Medic Mobile 透過 Web 串連全世界的醫療照護

4月112014

全球性非營利組織 Mozilla 致力推動網路的平等、自由、開放，要將 Web 的力量交到所有人手上。我們今天很高興宣佈與 Medic Mobile 展開合作。Medic Mobile 目前領導著開發中國家的行動醫療 (mHealth) 解決方案，志在改善偏遠地區的醫療品質。

Medic Mobile 率先透過功能型手機與文字簡訊，聯繫偏遠社區的醫療活動。除了用以追蹤疫情爆發的情形之外，也排定偏遠地區的醫療訪查時程，並建立起各村莊醫院之間的聯絡網。目前已於非洲、亞洲、拉丁美洲共二十個國家中運作，協助社區衛生工作者能夠顧及五百萬人的醫療品質。Medic Mobile 另啟動了《Hope Phones》手機回收計畫，希望能募得更多資金而導入更多必須的行動醫療技術。

Medic Mobile 也是 Firefox OS 的第一款社區醫療 App，現已於 Firefox Marketplace 上架。目前版本可讓衛生工作者在手機上記錄麻疹疫情，且未來版本將加入更多功能。

Medic Mobile 首席執行長 Josh Nesbit 表示：「蓬勃發展中的 Web 將協助我們邁出重要的下一步。我們會透過 Web 提供本地化的資料、圖片、影片、分析結果，讓社區衛生工作者能迅速掌握所需的資訊。Firefox OS 對設備的需求不高且相容性極佳，足以作為開發中國家的低價位 Web 平台。」

Mozilla 品牌策略與服務副總裁 Mary Ellen Muckerman 也說道：「我們很期待能讓 Medic Mobile 透過 Firefox OS 而觸及更多地區。Mozilla 也將持續推動這類地區性與本地化社區的 App，讓 Web 為更多人們提供更強的助力。」

Mozilla 很榮幸能與 Medic Mobile 合作，透過可改善人類生活的技術，進一步實現非營利的理念與使命。
