全球首支 Firefox OS 智慧型手機，2013 即將顛覆想像！

9月72012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012年推出以 HTML5 為基礎的行動作業系統 Firefox OS 引起廣大期待。預計全球第一款 Firefox OS 將由 Telefonica (西班牙電信公司) 於巴西推出。以下是來自舊金山的記者會實況報導：

Firefox OS 重點摘要

  • 預計發行時間：2013 上半年
  • 發行市場：巴西將是第一個推出的市場，接著其他拉丁美洲市場
  • 市場價格：仍未定，但估計在 $100 USD左右
  • OEMs 廠商：TCL (Alcatel) + ZTE 中興通訊
  • 其他支持電信商：Deutsche Telekom 德國電信公司, Etisalat 阿聯酋電信公司, Smart 菲律賓 Smart 電信公司, Sprint, Telecom Italia 義大利電信, Telenor 挪威電信公司
  • 手機芯片商：Qualcomm 高通

主要訊息

  • 為何要打造 Firefox OS? 智慧型手機市場的重要性無庸置疑，而在像拉丁美洲… 等開發中市場，智慧型手機的市佔率約佔 16%，而中低階價位的手機 ($100 USD) 效能並不足以滿足消費者對智慧型手機的需求。
  • 手機產業的創新被封閉系統所阻滯，而目前每個封閉系統內各自有以不同技術語言寫成的 Apps，也阻礙了內容共享與發展。
  • HTML5 有極大的潛力可將網路的開放與能量帶進行動平台，但卻因缺乏在封閉作業系統撰寫程式的能力，及不夠開放的 APIs，而阻礙其大幅進步發展。
  • Firefox OS 是一個全新的行動生態系統，完全以 Open Web (開放網路技術) HTML5 打造。Firefox OS讓每一支智慧型手機的每一個功能，都可以是一個獨立的 HTML 5 應用程式。而Firefox OS也讓 HTML5 得以跟硬體對話。
  • 結果將是一個全新的行動 Web 平台，有更輕盈的 OS 環境、更快速的瀏覽器，以及絕佳的網站內容與App體驗。

Firefox 優勢

  • Firefox OS 將提供同等價位的智慧型手機更優質的體驗，我們將能為廣大消費者提供更容易負擔的智慧型手機，更好地使用者體驗，以及更好的行動網路支援。
  • Firefox OS 將創造更平衡且開放的行動生態體系
  • Firefox 為瀏覽器市場帶來的創新，將延續至行動市場。
  • 為內容製造商和應用程式開發商帶來真正的跨平台機會，也降低開發門檻，小型的地區內容提供者、小型企業、教育者和政府機構，都有能力開發自己的 App 提供服務。

Firefox OS 正在形成一個日益壯大的生態系統

  • 合作 OEMs 廠商：TCL (Alcatel) & ZTE
  • 手機芯片製造商：Qualcomm
  • 網路公司：Facebook、Twitter
  • 電信公司：Deutsche Telekom 德國電信公司, Etisalat 阿聯酋電信公司, Smart 菲律賓 Smart 電信公司, Sprint, Telecom Italia 義大利電信, Telenor 挪威電信公司
