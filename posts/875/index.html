全新 Firefox for Android 邀您體驗高速、強大平板體驗！

8月292012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－ Mozilla 基金會，旗下瀏覽器 Firefox for Android 於美西時間28日釋出新版本。Mozilla 將桌機市場成功推動的開放精神延伸至行動市場，針對平板電腦使用體驗進行優化的全新Firefox for Android 現在可以在 Google Play store 進行下載。新版本特別針對平板使用者進行速度上大幅升級，從開機、網頁開啓時間、網頁平移縮放，到 Web App 的使用性能，為使用者提供更優質的使用體驗！

全新 Firefox for Android 提供使用者極便利的瀏覽體驗，一指輕觸立即到達最愛的網頁，並提供充滿個人獨特性的起始畫面。此版本更提供透過智慧位址列 (Awesome Bar) 功能直接連結到使用者的個人書籤、瀏覽歷史、密碼，以及表單資料。同時，使用者也可藉由同步功能，將儲存於其他電腦平台的 Firefox 設定同步至行動版 Firefox！此外，新版本支援 Android 3.0 (Honeycomb) 版本在平板上的 Flash 顯示，提供行動影音和遊戲互動更豐富的體驗！

Firefox for Android 新版本，讓使用者可隨意切換桌機版或手機版網頁的顯示，欲瀏覽桌機版網頁，請點選右上方的「開啟桌面版網站」功能即可輕易轉換。最尊重您個人隱私的 Firefox於此版本中提供「不要追蹤我 (Do Not Track)」、主控密碼、HTTP 強制安全傳輸技術（HSTS, HTTP Strict Transport Security）等各項安全功能來保護您的隱私。

全新 Firefox for Android支援更多 HTML5 功能，當開發者在使用HTML5、JavaScript、CSS，及其他開放網路標準為基礎製作網頁、開發網頁應用程式時更加順手。身為網路先鋒，Mozilla 已將數個新的 Web API 提交標準化，協助制定攝影（Camera API）、震動（Vibration API）、行動連線（Mobile Connection API）、電力狀態（Battery Status API）、螢幕方位（Screen Orientation API）及地理資訊（Geolocation API）等標準 API。

邀請使用者在電腦及行動裝置上試玩 Mozilla 的 HTML5 BrowserQuest 多人線上角色扮演遊戲，體驗最快、有趣、威力強大的網頁速度。
