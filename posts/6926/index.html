「Hacks」部落格 2014 年回顧

12月312014

不管你住哪裡，一定會遇到上班休假、學校放長假、年終部落格文章發佈、詳列過去一整年事件回顧的時候。Mozilla 以開發導向的「Hacks」部落格也將在 2015 年繼續與大家分享精采文章，介紹由 Mozilla 以及全世界 Open Web 開發者所打造的技術。

就讓我們透過下列重大事項，一同回顧 Mozilla、Firefox、Web 平台在 2014 年所樹立的里程碑，以及許許多多值得紀念的時刻：

  • 2014 年初我們強化了「Firefox Sync」同步功能，並新增「Firefox Accounts」為其架構。可進一步了解新的 Sync 協定。
  • Mozilla 與 Apache Cordova 團隊合作，整合 Firefox OS 與 Cordova 的開放源碼裝置 API，讓行動 App 開發者也能透過 JavaScript 取用原生裝置的函式，並釋出 Firefox OS 平台所適用的 Cordova App。Cordova 為 Adobe PhoneGap 的底層核心。此一整合讓 PhoneGap 開發者能夠輕鬆將自己的 App 移植到 Firefox Marketplace 上架。
  • Mozilla Research 對 Servo 是一大躍進。瀏覽器引擎原型「Servo」本身即以 Rust ─ 系統程式設計語言，為支援新硬體的並行 (Concurrency) 與平行 (Parallelism) 特性 ─ 所撰寫而得。另於春季發表了 Servo 的另一里程碑「Acid2」。
  • 我們 於 4 月底發表 的「 Australis 」，為 Firefox 桌機版瀏覽器的嶄新設計。主要負責此一專案的設計師 Stephen Horlander 另撰寫了一篇《 (Re)Designing Firefox 》文章說明整個過程。
  • 夏天發表的 Firefox 31 為遊戲開發者帶來多項新功能，包含 Emscripten 可協助將遊戲編譯為 JavaScript 格式。而透過《HTML5 遊戲開發資源》，我們針對複雜且高效能的 HTML5 遊戲，介紹某些可協助撰寫並除錯的工具＼技術。另外亦透過部落格披露了 Mozilla 率先開發之技術首次應用於 Web 商業遊戲，說明 HTML5 已經是完整的遊戲平台。
  • 在 10 月份，我們分享了 Firefox 的新功能「Firefox Hello」，此亦為 WebRTC 架構的語音＼視訊聊天服務。
  • 為了慶祝 Firefox 的十歲生日，Mozilla Research 另發表了「MozVR」，以期將虛擬實境帶入 Web。快戴上「oculi」來體驗吧。
  • 在此同時，Firefox 開發者工具團隊釋出 Firefox 開發者 (Developer) 版本，為第一款專為開發者所打造的瀏覽器。雖然此一版本仍處於早期階段，但已內建了「Valence」附加元件，可橫跨多款裝置與瀏覽器，以開發＼除錯自己的 App。另亦提供「WebIDE」編輯環境，可開發 App 並進行遠端除錯。
  • 「asm.js」則透過「Massive」持續演進。Massive 為開放源碼的效能基準量測工具，專為評估 asm.js 的效能所開發。可到這裡執行 Massive。
  • Mozilla 與合作夥伴一同設立「Internet Security Research Group (ISRG)」，並正式啟動「Let’s Encrypt」計畫。新設立的 Let’s Encrypt 為免費、開放、自動化的憑證機構 (Certificate Authority，CA)，預計於 2015 年夏季上線。
  • Mozilla 合作夥伴挪威電信 (Telenor) 發表「Gonzo」專案，將發掘 Firefox OS 作為嵌入式平台的潛力。Telenor 工程師與 Firefox OS 技術傳教士 Jan Jongboom 將一同開發無線相機，並分享此一過程中所學習到的經驗。
  • Firefox OS 已拓展至近 30 國市場，為收獲頗豐的一年。截至年底為止，Firefox OS 已現身於 29 國的 16 款智慧型手機之上。而在 12 月發佈的最新消息，Mozilla 亦與 KDDI 發表了日本國內首款 Firefox OS 智慧型手機，並已於 12 月 25 日起開始銷售。「Fx0」為第一支高規格的 Firefox OS 智慧型手機，而且真的很漂亮！

如果你心裡期待著 2015 年「Hacks」所應分享訊息的主題，歡迎隨時提供意見給我們。就寫電子郵件來吧！同時祝大家平安度過歡樂的節日，也可以找空檔繼續開發自己感興趣的東西！
