Firefox 桌面版、Firefox for Android 隱私功能聯手大躍進！

4月32013

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。今於美西時間(2日)釋出 Windows、Mac 和 Linux 版本適用的桌面版 Firefox 瀏覽器，以及行動版專用的 Firefox for Android。Firefox 桌面版與 Firefox for Android 同時強化隱私瀏覽模式的功能，而 Firefox 桌面版更新增了下載管理功能，使用 Firefox 能更輕鬆的下載檔案。此外，Firefox for Android 再擴大支援 ARMv6 處理器手機，預計到 9 月可以讓超過 5000 萬支的手機使用者擁有更理想的 Web 使用經驗。

Firefox 桌面版強化原本的隱私瀏覽模式，讓你可開啟新的隱私瀏覽視窗，而不需關閉或變更目前的瀏覽網頁，以進一步控制自己的私密瀏覽。你可透過隱私瀏覽視窗購買生日禮物，而不需中斷目前瀏覽中的網頁；亦可透過隱私瀏覽視窗而同步檢查多個電子郵件信箱；或對鏈結按下滑鼠右鍵，點選「用新隱私視窗開啟鏈結 (Open Link in a New Private Window)」，即可輕鬆以隱私瀏覽視窗開啟鏈結。我們也很驕傲的宣布，Firefox for Android 同樣支援單一分頁的隱私瀏覽功能。除了目前的瀏覽作業之外，Firefox for Android 亦可讓使用者開啟新的隱私瀏覽分頁；在單一瀏覽器中切換隱私與標準分頁。

Firefox 桌面版在工具列中新增了下載管理功能，使用者不需切換到其他視窗，即可監控、檢視、尋找已下載的檔案。新的下載管理功能讓 Firefox 能更輕鬆的下載檔案。Firefox for Android 則可讓使用者自訂首頁畫面上的捷徑，往後只要輕點幾下即可前往自己最愛或最常瀏覽的網站。

針對較低效能的 ARMv6 處理器，Firefox for Android 亦新增了支援功能。目前搭載該款處理器的常見手機包含 Samsung Galaxy Next、HTC Aria、HTC Legend、Samsung Dart、Samsung Galaxy Pop、Samsung Galaxy Q。而 Mozilla 預計將於九月份擴大支援這些 ARMv6 裝置，讓更多使用者能夠擁有絕佳的 Web 經驗。經過估算，我們將能讓超過 5 千萬支行動電話提供更理想的 Web 使用經驗。

針對開發人員，Firefox則提供 getUserMedia，屬於 WebRTC 規格的重要部分。getUserMedia 可讓開發人員輕鬆且迅速的撰寫程式碼，以存取使用者的相機或麥克風。Firefox 亦具備開發者專屬工具列，透過單一視窗即可迅速取得相關的開發工具，讓開發人員可輕鬆切換多種工具。圖層重疊模式 (Canvas Blend Modes)，則可讓開發人員將不同圖素加在既有圖素上，而達到圖層重疊的多樣視覺效果。
