Firefox ESR 企業版計劃啟動

1月122012

嘉惠企業/學校用戶

(2012/01/12 台北訊) 致力於網路開放自由的 Mozilla，其 Firefox 瀏覽器在全球擁有數億名愛好者，今日宣佈正式啓動 Firefox ESR (Extended Support Release) 企業專用版計畫。Firefox ESR 是針對企業用戶，公家機關，社群組織，或各級院校所開發，方便資訊管理人員方便於企業內部署 Firefox 並管理其資訊環境。Firefox ESR 預計一年更新一次版本，各企業或組織的 Firefox 將持續接收安全性更新，但無需就整個網路架構或附加元件平台做任何改動。Mozilla 目前正和許多公司企業合作，確保 ESR 的釋出符合企業對安全更新的需求，同時減輕對應用程式進行認證的負擔。

今日的網路正面臨前所未有的快速進化，提供更好的網路與 Firefox 使用經驗是 Mozilla 的使命。提供 Firefox 快速版本更新，可以保障網路使用者擁有一個安全不受威脅的環境，並提供一個創新的環境平台。然而快速更新版本的結果使得也使得 Firefox 的企業用戶面臨挑戰，IT人員必需不斷反覆測試透過 Firefox 提供的各項應用程式。為解決此問題，Mozilla 與 Mozilla Enterprise User Working Group 密切合作深入了解企業用戶需求，並啟動 Mozilla Firefox ESR 計畫。

Mozilla 承諾提供更好的網路經驗，Firefox ESR 的目標則是提供一般公司，公家機關，社群組織以及各級學校更容易管理的網路環境。過去幾個月以來 Mozilla 接到來自全球各地社群的回應與肯定，Mozilla 有信心 ESR 計畫將符合多數人的期待。

ESR 的實際執行細節將於一周內公佈於 mozilla.dev.planning 新聞群組，若您對 ESR 有興趣，想將Firefox部署於您管理的網路環境中，歡迎加入 Mozilla Enterprise Working Group並參與討論，或加入 Mozilla Taiwan Facebook 專頁瞭解更多。
