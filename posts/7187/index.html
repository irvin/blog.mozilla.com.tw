2015 Mozilla 校園徵才博覽會 等你來相會！

3月82015

地圖不可用

活動時間

日期 2015/03/08

10:00:00 - 16:00:00

地點

以捍衛 Web 自由與開放為使命的 Mozilla，即將於三月份展開一系列的徵才招募活動！

2014 年 Mozilla 發表了眾多 Firefox OS 生態系統的合作夥伴與新款裝置，今年延續著此番銳不可擋的趨勢，我們極力邀請更多台灣的年輕菁英加入 Firefox OS 的研發陣容，與我們攜手合作開發搭載以 HTML5 與開放網路技術為核心的 Firefox OS 裝置！

活動現場將由 Firefox OS 研發團隊工程師與你介紹徵才職務內容、分享身於 Mozilla 與國際交流的機會、Mozilla 為甚麼與眾不同，以及最重要的開源軟體專案的實務經驗等，讓你一探 Mozilla 豐富多元的工作環境與各種機會。

Mozilla Taiwan 美商謀智 校園徵才攤位暨說明會活動時間：

  • 3/8 (日) 國立台灣大學椰林大道，攤位編號 42、44
  • 3/14 (六) 國立交通大學綜合一館地下一樓，攤位編號 e1、e2
  • 3/16 (一) 19:00~19:50 國立清華大學校園徵才說明會，活動中心 102 演藝廳

除了招募徵才外，最受大家喜愛的 Firefox 火狐本尊也將親臨 Mozilla 攤位為大家加油打氣。別忘了現場給火狐一個熱烈的愛的抱抱，還有機會獲得 Mozilla 限量精美贈品喔！

若你對網路抱有熱忱、支持開源專案、對新型態的智慧型手機、裝置有高度興趣，歡迎加入我們的行列，一起打造行動網路、釋放未來！
