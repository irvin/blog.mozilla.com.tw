5 個方法大聲散播 Firefox OS 上市的消息

Mozilla 很興奮的宣布：從七月起我們會在幾個特定市場中推出 Firefox OS 智慧型手機。Mozilla 讓 Open Web 不再侷限於瀏覽器之內，而是進一步將之帶入行動作業系統，打破專利平台築起的那道高牆。

如果在你的市場所在地還買不到 Firefox OS 智慧型手機，也同樣可以加入慶祝的行列！以下是幾個方法：

  1. 向大家宣傳這個消息！在自己的社交網路上貼文、告知你的家人、朋友、隔壁鄰居這個好消息，甚至是跟陌生人大聲講！別忘了用 #FirefoxOS 的標記功能，讓其他有興趣的人能輕鬆加入對話內容。你當然也能直接導引其他人前往 facebook.com/MozillaTaiwan，就可以隨時獲得最新、最有趣的資訊！
  1. 在社交網路上換成 Firefox OS 的圖片吧！更換自己的大頭貼，表示你支持 Firefox OS ─ 同時也請大家共襄盛舉！到這裡下載圖片，或直接對下圖按下滑鼠右鍵，直接儲存到自己的桌面上。

  1. 和大家一起在社交網站上貼文！Mozilla Taiwan 特別推出慶祝 Firefox OS 正式發表的活動，邀請大家一起來為首支 Firefox OS 手機加油！今天起就來更新自己的 Facebook 狀態，並成為  Firefox OS 的粉絲。快來這裡簽到。
  • 活動名稱： 塗鴉牆分享 Go，支持 Firefox OS 超級有看頭
  • 活動時間：7/3 – 7/17
  • 活動方式：

– 於自己的塗鴉牆分享Firefox OS 小狐狸照片

– 寫下“慶祝全球首支 Firefox OS 手機上市，快到 @Mozilla Taiwan 粉絲團按讚留言抽大獎” 並確保有  Tag 出 Mozilla Taiwan 粉絲團

– 將分享的截圖上傳至活動頁面

– 您將具備參加 Firefox 驚喜抽獎資格

註：Firefox 禮包週週抽，每分享一次就增加一次獲獎機會（以活動期間一天分享一次為限）。

  1. 為自己的網站或部落格添增 Firefox OS 橫幅！在自己的網站或部落格上添增 Firefox OS橫幅，讓大家隨處都能看到Firefox OS 的蹤跡。立刻登入至 http://affiliates.mozilla.org 並取得橫幅嵌入碼 (而且有你熟悉的語言喔！)

  1. 加入成為火狐行動大使 (Mobilizer)！只要你是 Firefox 的粉絲，而且隨時都想散播 Firefox OS 的點點滴滴，你就是火狐行動大使！行動大使會定期接到有關任務，以簡單的行動讓別人了解 Firefox OS。行動大使也可於自己所在地區參加、協辦、主辦相關活動。

我們現正於波蘭、西班牙、巴西、墨西哥、哥倫比亞、委內瑞拉等地招募火狐行動大使團隊。如果 Mozilla 尚未開始在你的國家招募行動大使，也請你持續關注 https://wiki.mozilla.org/Mobilizers 上的消息。我們很快就會在世界各地啟動招募計畫。

快來向全世界發佈 Firefox OS 的消息吧！
