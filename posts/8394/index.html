Mozilla 開放源碼支援專案首波得獎名單

1月252016

很高興宣佈 Mozilla 開放源碼支援專案 (Mozilla Open Source Support，MOSS) 的首波「基礎技術 (Foundational Technology)」獎項。

感謝 MOSS 的「基礎技術」委員會耗時費心評估相關 App 並決定得獎者。

首波的 7 名得獎者為：

  • Buildbot：美金 $15,000。Buildbot 是連續性的建構與整合系統，過去幾年為 Mozilla 提供極高的價值。這筆獎金將用於移除其所有 API、測試、記錄文件之中的「slave」字彙，也將協助 Buildbot 在 Amazon EC2 雲端上更順利的運作。
  • CodeMirror：美金 $20,000。CodeMirror 是用 Web 技術所建構的強大原始碼編輯器，並已用於開發者工具與 Mozilla Thimble 之中。這筆獎金將用以提升由右到左語言，以及複雜指令碼輸入的支援功能。
  • Discourse：美金 $25,000。多個 Mozilla 社群均使用 Discourse 線上討論區軟體。這筆獎金將讓 Discourse 採用電子郵件為其a first-class 互動機制，並取代 Discourse 實作而提升現有的郵件群組。
  • Read The Docs：美金 $48,000。Read The Docs 是建構、架設說明文件的網站，已有 Mozilla 的許多 Web 專案所採用。這筆獎金將用以從程式碼產生說明文件，而不需安裝 it,。往後即使是複雜專案亦可輕鬆產生說明文件。
  • Mercurial：美金 $75,000。Mercurial 為分散式的原始碼管理系統，受 Mozilla 的核心 Repo (如 mozilla-central) 極度依賴。這筆獎金將用以更妥善支援「blame (顯示最後更改某段程式碼的人)」並建構更好的 Web UI。
  • Django：美金 $150,000。Django 是目前高人氣的伺服端 Web 開發框架，已有 Mozilla 的多個網站所採用。針對使用 WebSockets 的 Web App，這筆獎金將用以讓 Django 更適用於這類 App 的後端。
  • Bro：美金 $200,000。Bro 為網路監控軟體，也是 Mozilla 網路入侵偵測系統的核心。這筆獎金將針對 Bro 建構「Comprehensive Bro Archive Network」，即模組與外掛程式的公用 Repo。

MOSS 是持續進行中的專案，第一波獎金總額為美金一百萬，而上述頒發總額也不過用了一半 (美金 $503,000)。目前持續開放申請 「基礎技術」。我們希望能援助更多 Mozilla 使用或依賴的專案。Mozilla 沒有你就辦不到今天正在做的事情。感謝大家！
