打造健康的網路並交給下一代

12月262014

十年前，由 10 名 Mozilla 員工與數千名 Mozillian 組成的團體，為打破微軟 (Microsoft) 所壟斷的上網方式而釋出 Firefox 1.0 供大家使用。我們所凝聚的全球性社群共同著眼於單一目標：要讓 Web 成為全球共用的資源，開放給所有人隨時享受。

與十年前相比，我們現在所面臨的挑戰有過之而無不及。Web 提升了我們的生活品質，但也促成新帝國崛起。現有許多組織控制了我們的搜尋、傳訊、資料儲存＼讀取等方式，再次威脅到「Web 成為全人類開放且受信賴的資源」之承諾。

Mozilla 如何打造我們所需要的網路

「Mozilla」專案為了解決這些難題而誕生。我們的社群讓 Firefox 產品得以風行於全世界，並實踐保護 Web 的使命。我們也努力教導更多人協助打造 Web、保護個人隱私、建立技術標準，並開發新的開放源碼技術以利他人使用。

Mozilla 去年結算約三億一千四百萬美金的收入。這筆金額橫豎都是一大筆金額，但若加總我們所有競爭對手的資源，只不過是九牛一毛。而 Google 在去年僅廣告部分的費用，就超越我們所有花費的總和。

Mozilla 專案的成功與否，均端賴「貢獻」的結果。全球有一群人貢獻自己的時間與資源打造我們所須的 Web，並傳承「開放、共有、相互操作」的理念。但我們仍須面對極嚴苛的挑戰。其實，讓系統一開始就朝著封閉方向發展並集中於少數人之手，又何嘗不是更簡單的方法？

為了讓 Web 切合自己的需求，我們必須一手包辦從程式碼釋出、發表產品、設法獲得微薄收入或募資、建立標準、尋找夥伴、與志工交流溝通，直到培養出下一代的 Open Web 戰士。這是嚴峻的任務，而且同樣需要時間與資金，再加上大家無比的熱情以及盡心盡力的社群，這一切才得以實現。

錯誤觀念

媒體與許多人往往著眼於我們的財務情況，卻未通盤了解完整的架構。在此特別說明，某些文章只點出我們從搜尋服務所獲得的收益正在衰退，還暗示我們所簽訂的搜尋服務合約，與我們的募款活動有所掛勾。這些說法都不符合事實。

事實上，整個專案的財務狀況健康無虞。2013 年的收益情形優於 2011 與 2012 年的結果。而 2012 年到 2013 年的成長幅度，也符合我們簽約時的條件，這一切都在我們預期之中。Mozilla 預估 2014 年 (與 Google 所簽訂三年約的最後一年) 也會是類似的狀況。此外，我們也再度強調，搜尋服務合約與募款活動之間全無關係。對 Mozilla 來說，此兩項是截然不同的獨立活動。

永續經營

與其他組織相較，我們對「永續性」有截然不同的想法。Mozilla 是以本身使命的永續性為首要考量。財務永續固然重要，但並非我們心中的首要之務。對 Mozilla 來說，永續的核心就是維持專案運作無虞：建立 Web 使用者心中的價值、教育網民如何豐富自己的生活、為志工與貢獻者設立專屬的社群。當然，最後仍需要資金來維持相關活動。

Mozilla 的收入均來自於募款與營收所得。為了澄清任何可能的誤會，我們理應解釋不同類型收入的運用方式，又是如何使用我們所募得之款項。

Mozilla 的慈善收益包含贊助金與個人捐款，合作夥伴則有全球性的基金會 (如 Knight、MacArthur、Ford)，並從 2010 年起獲得超過兩千五百萬的善款。除了這些金額較大的贊助之外，也有多位熱情的使用者不斷詢問自己該如何貢獻。另外，由社群主導的募款活動 (也包含由 Firefox 瀏覽器目前重點推動的募款訊息)，已成為我們成長最迅速的款項來源；從 2011 年起的四項活動，我們共募得超過三百萬美金。所有相關款項均直接進入 Mozilla 基金會的帳戶，以支援我們的教育、研究、提升大眾認知等活動。

我們的收入包含了最近與 Yahoo、Yandex、百度所簽訂的搜尋服務合約。這些合約也是 Mozilla 最大宗的收入來源，讓我們能夠穩定、獨立、靈活的持續在全球闡揚己身理念。我們另依照既有的使命開發出新產品，並擴展目前的產品組合，包含 Windows、Mac、Linux 所適用的桌面版 Firefox、行動版 Firefox (Firefox for Android)、Firefox OS、網頁內容服務、雲端服務。這些收入均歸到 Mozilla 基金會之下，而非基金會所轄的 Mozilla Corporation，藉以繼續支持我們的產品開發與行銷等活動。

為了保護 Web 這項公共資源，簽約收入確實是 Mozilla 得以繼續運作的動力之一。如同上述，我們所獲得的款項均用以打造產品並執行計畫，進而持續推廣理念。但影響Mozilla 永續經營的最重要因素，是全球社群所貢獻的時間與勞力。2013 年全球共超過十二萬人捐款，今年更已突破二十萬人。這些人當然不只出錢，也傾全力熱血支援 Web。過去幾年已累積數百萬人次簽署 Mozilla 所舉辦的活動，聲援網路中立性以及其他問題。另外還有超過兩萬名志工協助打造 Mozilla 產品並推動 Mozilla 教育計畫，這些人都代表了 Mozilla 的精神。

Mozilla 之所以能夠挑戰在Web 世界中的帝國，並塑造 Web 將來的樣貌，千千萬萬的人都扮演了關鍵角色。這些人就是我們永續經營的核心，也代表了 Mozilla。

─ Mozilla 基金會執行總監 Mark Surman

Business and Legal Affairs 資深副總裁 Denelle Dixon-Thayer
