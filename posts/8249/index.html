Focus by Firefox ─ iOS 專用內容攔截功能

12月82015

專為 iOS 9 的 Safari 所設計「Focus by Firefox」免費內容攔截 App，很高興在今天和大家正式見面。Focus by Firefox 將依照追蹤器類型 (如用於廣告、分析、社交媒體的追蹤器) 進行攔截，可讓使用者控制自己所需的隱私程度，並進一步控制自己的行動 Web 經驗。Focus by Firefox 另可攔截 Web 字型，藉以提升效能並減少行動資料流量。我們歡迎任何人提出有建設性的反饋意見，協助我們未來開發出 Focus by Firefox 的更多功能。

使用者現可於蘋果的 App Store 下載並安裝 Focus by Firefox。在安裝完畢之後，只要啟動該 App 就會出現設定指南。相較於 Windows、Mac、Linux、Android 版本 Firefox 隱私瀏覽模式的「追蹤保護 (Tracking Protection)」功能 (即以「Disconnect」公司所提供的「開放源碼攔截名單」為基礎)，Focus by Firefox 亦攔截完全相同的追蹤器。

Focus by Firefox 目前僅限用於 Safari 之上，而尚未支援 Firefox for iOS。這當然本非 Mozilla 所願，而是因為蘋果公司限制 iOS 上的第三方瀏覽器無法使用內容攔截功能。我們正努力研究，希望能儘早讓 Focus by Firefox 也能用於 Firefox for iOS 之上。

你可從這裡進一步了解我們打造 Focus by Firefox 的理由。Mozilla 很高興能和大家分享此一功能，未來也將持續改良 Focus by Firefox！
