Mozilla 正解放 Web 的遊戲平台潛力

3月282013

身為非營利組織的 Mozilla，除了致力提升 Web 的創新、可能性、開放性之外，現更進一步要讓 Web 成為高端遊戲的開發平台。透過 Mozilla 最新開發的 JavaScript，遊戲廠商與開發人員現在可利用高速的效能特性，加上 Web 無遠弗屆的規模，且不需第三方附加元件的額外成本，進而將競爭對手遠遠拋在背後。遊戲廠商在開發畫質細膩的高效能遊戲之後，將以更輕鬆、更高成本效益的方式推廣予數十億玩家。

為了提供相關優勢，Mozilla 開發了高度最佳化的 JavaScript 版本，可將瀏覽器內的遊戲程式碼效能發揮到極致，讓 Web 提供快速且令人驚嘆的 3D 遊戲經驗。而隨著 JavaScript 的效能將持續突破，Mozilla 亦將為行動裝置啟動 Web 架構的 3D 遊戲。

我們很高興能與 Epic Games 合作，並在今天針對高效能遊戲而提供這項技術。在利用了新的 JavaScript 最佳化技術之後，Mozilla 可將 Epic 公司的 Unreal 3 Engine 帶入至 Web 中。而基於此架構，投身 Web 遊戲的開發人員將近一步拓展無限可能。

Mozilla 最近所發表的《BananaBread》測試遊戲，更印證了 Web 確實可以成為吸睛的遊戲平台。此測試遊戲是以 Mozilla 的 Web 技術為架構，包含 WebGL 與 Emscripten，現更加上 asm.j。而《BananaBread》代表高端遊戲亦可透過 JavaScript 與 WebGL 呈現，同時仍能快速反應而提供絕佳的 3D 遊戲體驗。

若開發人員想測試此項技術，可透過最新版的《BananaBread》，隨時體驗點對點 (P2P)、多重使用者的 WebRTC 技術、提升過的 JavaScript 效能等特色。只要是支援 WebGL 的瀏覽器，均可運行《BananaBread》。

隨著 Web 的高效能遊戲已漸趨重視其基本效能的提升，Mozilla 亦跟著開始拓展行動裝置上 Web 遊戲。目前合作的領導遊戲廠商包含 ZeptoLab、迪士尼 (Disney)、美商藝電 (EA) 等，均透過相同的技術，要讓最熱門的遊戲達到最佳效能。

開發人員 (你) 手上有任何好玩的 App 與遊戲嗎？現在已經可以提交到 Firefox Marketplace 上了。而目前除了可於 Firefox for Android 上預覽 Firefox Marketplace 之外，今年內亦將於 Firefox OS 上發現 Firefox Marketplace 的蹤跡。
