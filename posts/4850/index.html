Mozilla Taiwan 首度前進校園徵才就業博覽會

3月92014

地圖不可用

活動時間

日期 2014/03/09 - 2014/03/22

地點

你想加入 Mozilla 一同為網路的開放自由努力嗎? 機會來囉～! Mozilla Taiwan 將於 2014 年參與春季校園徵才, 在杜鵑花開的繽紛季節, 直接與即將進入就業市場的台灣青年學子面對面, 徵選對網路技術、開放源碼、行動作業系統有興趣有熱忱的新血, 一起加入打造 Firefox OS 開放行動作業系統的行列。

Mozilla Taiwan 預計將有三場校園徵才活動, 分別是：

– 3/9 (日) 09:30 ~ 16:30 台北台灣大學

– 3/15 (六 ) 09:30 ~ 15:30 新竹交通大學

– 3/22 (六) 10:00 ~ 16:00 新竹清華大學

校園徵才現場將安排多位 Mozilla 資深工程師在場, 與青年學子懇談, 提供 Firefox OS 開發的經驗分享、 Open Source 參與建議與評估。在校園徵才活動中, 除了一般人認知的正職員工招募之外, Mozilla Taiwan 亦提供在學青年實習機會, 讓學生在求學期間就能培養 Firefox OS 實際專案經驗、提升技術能力。除此之外, Mozilla 也提供多項 Mozilla 專案志工機會, 有志青年可透過參與開放源碼的社群, 一窺國際專案的運作方式。而除了程式工程端的參與機會之外, Mozilla 的校園大使 (FSA) 專案則提供非技術背景的學生參與 Mozilla、認識開放源碼精神、並學習進行市場推廣等各項體驗。

Mozilla Taiwan 首次參與台灣校園徵才活動, 現場將以最誠摯的心, 和就業市場的青年面對面, 屆時並將邀請神秘嘉賓, 與徵才現場的同學們互動。歡迎自認對網路有熱忱有想法有能力的你, 把握加入 Mozilla Taiwan 與國際接軌、參與國際專案核心的好機會!

Mozilla 與你, 2014 校園徵才見!
