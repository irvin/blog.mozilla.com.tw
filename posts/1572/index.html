MSN 特別版 Firefox 瀏覽器 以 Bing 作為預設搜尋引擎，有效利用 MSN 服務

11月302012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－ Mozilla 基金會，2012年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。於美西時間 (27日) 針對 Bing 的使用者推出 MSN 特別版 Firefox 瀏覽器，除了以 Bing 作為其預設搜尋引擎，更加入可彙整社群熱門話題的 msnNOW、MSN.com 預設起始頁等獨家 MSN 風格。Firefox 目前共有近 20 款針對合作夥伴與使用者推出的客製化的版本，包括 AOL、Bing、Twitter、Yahoo 以及 Yandex 等，致力為使用者打造集客製、便利與立即於一身的優質瀏覽體驗。

Firefox 與微軟的合作初始於將 Bing 列為 Firefox 搜尋引擎選項之一，接著推出以 Bing 為預設搜尋的 Firefox 版本，如今更推出 MSN 特別版 Firefox 瀏覽器，不僅將 Bing 做為搜尋列和網址列（又稱 Awesome Bar，智慧位址列）的預設搜尋引擎，同時也於瀏覽器上方的 FirefoxApp Tab置入整合社群熱門話題的 msnNOW，並把 MSN.com 做為預設的起始頁。

透過 MSN 特別版 Firefox 瀏覽器，你可以在 Firefox 上體驗 MSN 風格，也能立即連上最熱門的 MSN 頻道，提供即時的新聞、娛樂、運動等你最關心的訊息。想搶先下載的你請點選這裡。若你已經是 Firefox 使用者，請在 Firefox 新增 MSN 附加元件 為你的瀏覽器添上客製化體驗。

MSN 特別版 Firefox 瀏覽器目前僅提供英文版。

原文參考：https://blog.mozilla.org/blog/2012/11/27/offering-a-firefox-with-msn-custom-experience/
