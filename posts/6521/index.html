Mozilla 宣佈與遊戲發售平台「Humble Bundle」合作

10月152014

Firefox 使用者直接體驗 8 款高人氣獨立製作遊戲

Mozilla 今日宣佈與創新的遊戲發售平台「Humble Bundle」合作，讓 Firefox 使用者能夠直接體驗 8 款高人氣的獨立製作遊戲，其中更包含獲獎的《超越光速 (Faster than Light，FTL)》。此促銷專案僅限期兩週，趕快到「這裡」加入遊戲饗宴吧！


Humble Mozilla Bundle

Mozilla 已經將率先開發之技術應用於 Web 商業遊戲，且玩家不須耗時下載大型檔案，即能夠享受高畫質、高效能的優質遊戲。只要一個連結就能直接進入遊戲的特性，讓玩家能在付款之前得以先行體驗遊戲，開發者亦能夠透過社交媒體分享自己的作品。

「Humble Mozilla Bundle」重點放在目前成長最快的獨立製作遊戲之上，亦包含之前僅限桌機版本或行動裝置下載的遊戲。除了新的《Democracy 3》遊戲之外，還有名稱短如《FTL》，甚至長如《AaaaaAAaaaAAAaaAAAAaAAAAA!!! for the Awesome》等的高人氣遊戲。這些遊戲都是透過 Mozilla 率先開發的 asm.js 及其他強大技術，成為 Web 上免外掛程式的遊戲。玩家只要輕點連結就能開始歡樂時光。

雖然新的網路瀏覽器都能執行這些遊戲，但唯有 Firefox 才能提供最順暢的體驗。Humble Bundle 另提供雲端的存檔同步功能，讓你能在任何電腦上繼續目前的遊戲進度。這些遊戲開發者更努力提供足讓消費者驚艷的效能，而 Humble Bundle 亦針對 Web 進行遊戲最佳化，讓我們相信 Web 當然不只是遊戲發展的絕佳平台，也能成為開發者獲利的理想平台。要想了解如何移植自己的遊戲到 Web，請參閱 MDN 上的遊戲開發頁面相關文章。

Humble Bundle 已透過 Web 建構全新的「玩家配遊戲」銷售模式，在特定短時間內讓消費者依自己意願支付金額，即可賞玩不同遊戲組合而成的遊戲包。消費者甚至可指定開發者與 Humble Bundle 的拆帳比例，剩下帳款均捐做公益慈善用途。此一銷售模式不論要在單一地區逐漸推廣，或是透過社交媒體迅速散播，Web 都絕對是絕佳的發佈管道。截至目前為止，Humble Bundle 已經募得 4,600 萬美元的慈善捐款。

因為這個銷售模式實在是太有意思了，Mozilla 決定要與「Humble Mozilla Bundle」內多款遊戲之一《Voxatron》的開發者合作，直接在 Firefox 首頁上提供互動式的酷炫遊戲，藉以確實展現 asm.js 遊戲的威力。正式開放後，只要在 Firefox 網址列鍵入「about:home」，就能獲得前所未有的樂趣，並選擇你願意在 Humble Bundle 上支付的價格。Mozilla 希望你接著發掘更多能在 Firefox 上體驗的遊戲。
