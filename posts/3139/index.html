Firefox OS 所代表的意義

7月182013

Firefox 近日宣佈 Firefox OS 的智慧型手機正式上市！(截至本篇文章發表為止，已有西班牙與波蘭的電信營運商開始販售。)

如果你所在地的市場還看不到 Firefox OS 手機，你看好 Firefox OS 會成功嗎？我們讓一位給予正面答案的專家來為大家解釋。

歡迎海爾曼 (Christian Heilmann)！

Q: 先談談你和 Firefox OS 的淵源？

A: 我本來就參與 Firefox OS 的相關工作，我協助開發者匯入他們的 Apps，也花很多時間寫了相關文章、設計 Demo、與新聞媒體相談，甚至只要某個人在我旁邊站上幾分鐘 (不然就是搭飛機時坐我隔壁)，我都會跟對方聊上幾句 Firefox OS。

Q: 為什麼你覺得 Firefox OS 很重要？

A: 有很多原因。

  • Firefox OS 徹底終止了「行動上網只限西方世界的有錢人才用得起」的刻板概念。 沒錯，住在美國或英國的人或許每半年就會換一支閃亮的新手機，但這不是所有人都負擔得起的。即使可以，在許多國家，沒有信用卡也不能購買任何手機上的 Apps。這現象不僅存在精英主義式的不公平，也與 Web 的理念背道而馳。Firefox OS 手機符合多數人的消費能力，更能透過預付費用或電信帳單代繳的方式購買 Apps。
  • Firefox OS 並 不要求一個十分快速、穩定、隨時處於連網狀態的網路環境 。 每次出國旅行時，我都會開始討厭自己的 Android 手機 (當然它還是有其它討人喜歡的特色)。因為我們不可能用漫遊費率，也不太想用速度緩慢又信號微弱的 WiFi 連線，隨時下載大型的更新檔。而 Firefox OS 沒有任何原生 Apps，且包含系統 Apps 在內的所有應用程式，都是以 CSS、HTML、JavaScript 撰寫而成。因此檔案容量不僅比較小，更新檔案也不大。只會更新部分的 App，而不會以整個 App 為單位進行更新。
  • Firefox OS 就是你的隨身 Web 。 Firefox OS 其實就是 Firefox，頂多再加上存取硬體的Linux 核心。當我開啟可完整支援手機瀏覽的網站時，不會再看到系統要求我下載原生 App。
  • Firefox OS 就是 HTML5 平台。 對開發者來說，我們的 HTML5 解決方案終於是這個環境中的高級公民 (The first class citizens) 了！Firefox OS 中的 HTML5 Apps 可以完整發揮硬體的效能，而不再侷限於執行緩慢的網頁。
  • Firefox OS 完全透過 Web 發佈 Apps 。 在 Marketplace 中，使用者能透過手動搜尋或 App 評分等級，找到我們所撰寫的 Apps。同樣方式亦可搜尋特定的歌曲、影片、足球隊伍，甚至一道菜的食譜。發掘 App 的過程充滿樂趣，而且就像使用 Web 或尋找網站一樣簡單。

Firefox OS 還有太多令人開心的特色，讓我興奮得像跳進糖果堆的 5 歲小孩一樣！(可到海爾曼的部落格觀看更多心得)
