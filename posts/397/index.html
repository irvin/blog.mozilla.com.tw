UPDATE: 投影片上線：

  • Gaia: A modern smartphone user interface in HTML5 and JavaScript
  • 淡淡定定用 JavaScript 寫 WebAPIs

上次 OSDC 的 Mozilla Taiwan 攤位與演講你錯過了嗎？還有機會！在 5/19 即將舉辦的 JSDC，Mozilla 也將有演講和攤位。Mozilla 這次大力參與 JSDC，與台灣致力推廣 JavaScript 技術的社群共同主辦 JSDC，這次更邀請到 Mozilla 研究總監 Andreas Gal 現身說明 Mozilla 對手機前端的想法，讓 Web 開發者都可以參與手機介面的設計與製作！

Mozilla Taiwan 這次用力參與 JSDC，你可以在會場找到：

  • Mozilla 研究總監 Andreas Gal 的 Keynote speech：「Gaia: A modern smartphone user interface in HTML5 and JavaScript 」
  • Mozilla Taiwan 軟體工程師 KanRu 的演講
  • Mozilla Taiwan 的攤位！

這次 Mozilla Taiwan 在攤位上又有新創意，也有十分豐富的紀念品，現場同樣提供 Boot to Gecko 手機搶先試玩，還有許多有趣的開發者資訊，快揪朋友來參加吧！
