Firefox 十週年活動 幸福的狐家元件

10月172014

Firefox 附加元件全球下載量超過 3,000,000,000！此數字大約是全球人口總數的一半。Firefox 附加元件讓你自訂功能、特性及外觀，給你最個人化的瀏覽體驗。

Firefox + 附加元件 = 幸福方程式！

立即參加「幸福的狐家元件」一窺狐家的幸福祕訣

Firefox 十週年系列活動「幸福的狐家元件」，教你成為人生勝利組的成功祕訣！Firefox 附加元件是自訂個人化瀏覽器的最佳選擇，全球超過 25,000 名 Firefox 附加元件開發者協力開發符合各種使用者需求的附加元件，無論你日常生活扮演的角色是上班族、家庭主婦、長青族、學生、科技宅、開發者、網路正妹…，都可以像狐家人一樣找到最適合你的幸福的附加元件！

Mozilla 於 2004 年開始 Firefox 附加元件中心 (簡稱 AMO)，並在 2009 年首開先例提供手機行動瀏覽器 (Firefox for Android) 的附加元件，讓行動與桌面瀏覽器使用這皆可享受自訂瀏覽器的樂趣，自訂諸如音樂、遊戲、休閒娛樂、運動、審美觀，以及在地化語言等最符合個人化與上網需求的功能。

據統計，全球超過 85% 的 Firefox 使用者至少會安裝一個附加元件，而慣用附加元件的使用者則平均每人安裝五個附加元件。最受歡迎的附加元件包括可阻擋網路廣告的 AdBlock Plus、開發者除錯得好用工具 Firebug、可聰明避開安全漏洞讓 Firefox 更加安全的 NoScript、更方便更換主題面板的 Persona Plus、輕鬆下載網路影片的Video DownloadHelper；最受歡迎的類別則包括隱私/安全、搜尋管理、書籤管理、和佈景/外觀。其中，由使用者自建的收藏集共超過 15 萬件，包括適合家庭使用的 Family Organizer、適合背包客的 Traveler’s Pack、給運動愛好者的 Sports Fanatic、防騷擾專用的 The Paranoid Kit、網站程式設計師的 Web Developer’s Toolbox 以及線上購物者最愛的 Online Shopping。

Firefox 十週年系列活動「幸福的狐家元件」，特別為台灣 Firefox 使用者勾勒出六種使用者形態，並推薦超過 30 種由網友票選最受歡迎、最實用的附加元件 (以台灣地區為主)。無論你對上網的需求是希望：網頁圖文自動變大一點、安全防護更嚴密一點、上網購物比價更方便點、逛 BBS 更容易一點、使用網路銀行更聰明一點、自訂滑鼠手勢更帥氣一點… Firefox 附加元件都可以滿足你的需求。如果你不曾知道瀏覽器可以如此強大，現在更是時候來瞭解 Firefox + 附加元件 = 幸福的方程式。

與此活動同步釋出的還有 Firefox 十週年特展「網路自由日」早鳥入場卷正式開放免費索取。早鳥報名時間 10/16 ～ 11/10，索取方式請詳見「幸福的狐家元件」活動內容。凡持有早鳥報名序號者，於活動現場報到後可獲得 2014 Firefox OS 獨家限量海報乙張，並享有「網路自由日」Firefox OS 手機抽獎機會。
