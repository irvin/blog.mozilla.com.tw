「10 月電子報有獎徵答」得獎名單公佈

11月192013

感謝大家熱情參與「10 月電子報有獎徵答」活動，共有 3 位幸運的得獎者，可以獲得 Firefox 悠遊卡一張！

本期電子報有獎徵答題目：

Mozilla 與 Ford 基金會共同推出，以圖像化的方式呈現所有人在 Web 上被追蹤的路徑，協助使用者更了解自身周遭 Web 環境的附加元件為何?

A. Firefox OS App Manager

B. Lightbeam for Firefox

C. MozTrap

正確答案為 B. Lightbeam for Firefox
