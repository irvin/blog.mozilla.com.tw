MozFest 2015：為 Open Web 團結社群並串連領袖

今年邁入第六屆的 MozFest ── 即 Mozilla 為 Open Web 所舉辦的盛大慶典。來自於全球的開發者、傳授導師、組織能手齊聚一堂，設法解決現今 Web 正面臨的問題。Mozilla 更同時站在技術、策略、社交圈的領導角度，希望能為所有人打造更好的 Web。

與會者包含如新聞界、教育界、科學界、硬體、物聯網 (Internet of Things，IoT) 等領域的人，獨立或共同主講相關課程，期能塑造 Web 的未來。來自於全球共 1,700 人參與，自主舉辦 300 場課程並訂立 50 種學習流程，讓整個週末充滿歡樂又啟發的思考活動，也提醒我們的確能夠讓更多人加入行動網路，更為 Open Web 注入更高的動能。

MozFest 是以科學市集的形式開幕，再以大家共同開發的設計物件舉辦展示派對而畫下完美句點。整個週末就如同是發明、建構、淬鍊所組成的狂野樂團盡情搖滾吶喊。可在這裡欣賞此週末的精采照片。

由與會者所建議的課程內容，讓我們得以深入了解 Open Web 社群最在乎的大小事。在 Digital Citizenship Space 中，與會者就針對線上隱私、監視、數位版權等的現狀進行討論，期能打造更安全的網路。Mozilla 資深政策經理 Raegan MacDonald 另概述了 Web 安全的威脅，例如英國的 Investigatory Powers Bill 就在蒐集、分析、儲存使用者資料。

但 MacDonald 也提及了光明面，像是自造者、思想家、工程師都竭盡心力研究解決方案：「在我的烏托邦裡，隱私與安全應該是如同預設設定的基本要求」。

也有年輕學生與導師在 MozFest 期間展現自己的專案。像是來自於英國的 12 歲少年 Kent 就設計了「The Gauntlet」互動賽馬場，即以 Raspberry Pi、Makey Makey、Python、藍牙、運動感測器所構成。而 George 則是對超過 100 名孩童傳授編碼與電子學的基本概念。隔壁更有 8 歲的 Otis 展現自己最新「Unity」引擎的創作：虛擬的飛行模擬器遊戲。

附近另有一群人在討論開放式硬體與 IoT 的主題。來自於 Wevolver 的團隊就呈現了開放源碼的機器人專案，可協助病童以遠端方式暢遊動物園。其他青少年則能戴上虛擬實境的頭部裝置，透過機器人的眼睛瀏覽動物園，並與動物或陳列品互動。

Wevolver 的 Bram Geenen 表示：「我們想讓大家知道開放式硬體正在興起，軟體發生的事也可轉移到硬體之中。」

會場的另一處，與會者正討論著〈IoT 設計宣言 (Interner of things Design Manifesto)〉，希望能引導工程師打造安全、隱私、好用的裝置。

接下來呢？我們該如何將 MozFest 所誘發的動力引介給高生產力、高創造力的開發者，讓他們繼承 Web 的開放思潮繼續創作？在 Mozilla，不論是過去幾天所做的，或是過去幾年所學習而得的，我們都會透過全球的社群與活動將之深化。我們也將提供更多機會培養領導者，並讓所有人都能再推動整個浪潮繼續前進。

你又是怎麼想的呢？在 MozFest 2015 所萌發的概念，也代表了我們團結的強大力量，而且我們只會更為茁壯。有關 IoT、隱私、監控、數位共享等問題的解決方案，其實都握在我們手中。讓更多人能聽見我們的聲音，也要團結更多力量為 Web 發聲。加入我們捍衛 Web 的行列！
