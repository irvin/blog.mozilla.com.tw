Mozilla 將 Web 推向遊戲平台新層次

3月162016

Mozilla 所率先開發的創新技術，已促使 Web 搖身成為重要的遊戲發佈與開發平台。而 Mozilla 更於目前正在舊金山進行的遊戲開發者大會 (Game Developer Conference；GDC) 上實際向大家展示 Web 作為遊戲平台的能力。強大的新功能除了將吸引更多玩家，對開發者造成更顯著影響力之外，也有助於 Open Web 遊戲軟體堆疊加速普及。

  • 由 Mozilla 率先開發的技術包含 WebGL 、 WebVR 、 asm.js 等，都已獲得重視並逐步起飛。
  • 新一代的 asm.js ，即目前屬於實驗階段的 WebAssembly 實作， 現已可透過 Firefox 每夜更新版 (Nightly) 進行測試 。
  • 於 GDC 當週上線的「 Open Web Games 」網站，可讓開發者與瀏覽器製造商前往展示最新的 Web 遊戲技術，並進行瀏覽器壓力測試實作。此外，亦能利用其上的多樣實際遊戲與展示，共同合作維護 Web 遊戲堆疊的穩定性以使其再度進化。
  • 現可透過 Firefox Nightly 體驗 WebGL 2 、 SIMD.js 、 Shared Array Buffer 等新一代的 Web 技術。
  • 為了持續推進 WebVR ， Mozilla 最近發佈 WebVR 提案 1.0 版 。
  • Mozilla 現正投入更多心血於「 Mozilla 開發者網路 (Mozilla Developer Network ； MDN) 」並直接提供技術支援，要協助開發者跳脫外掛程式而直接轉用 Web 技術。

在 2015 年，業界持續支援整個 Open Web 軟體堆疊：

  • 身為業界最大、評價最高的遊戲引擎之一，「 Unity 」 已從自家的 WebGL ( 即以 WebGL 與 asm.js 為其架構 ) 匯出功能中 移除了「預覽」標籤 ，證明 Web 軟體堆疊 已經準備好加入遊戲產業。
  • 3D 設計的領導廠商 AutoDesk 正為其「 Stingray 」遊戲引擎添增 Web 匯出功能。
  • 包含 EVERYDAYiPLAY 在內的行動遊戲獨立開發商，正針對 Web 打造如《模範英雄 (Heroes of Paragon) 》的遊戲，藉以拓展更多營收來源。另根據 公司報 告 指出，該遊戲 Web 版本的「各玩家日平均收益 (Daily Average Revenue per User ； DARPU) 」，現已高於 Android Play 版本，且幾乎快趕上 iOS 版的水準。


遊戲《Heroes of Paragon》不同版本的「各玩家日平均收益 (DARPU)」

  • 主要瀏覽器為因應新一代的 Web 遊戲需求，進而採用了更多關鍵 API 。業界的頂尖品牌更將擴充支援 Web ，讓開發者能在 Web 平台上成功打造豐富內容。


Open Web 遊戲的軟體堆疊已漸趨普及

更多資訊：

  • Mozilla 展區位於遊戲開發者大會 (Game Developer Conference；GDC) Moscone Center 南邊大廳低樓層 936 號。
  • 想進一步了解 Mozilla 今年在 GDC 做了哪些有趣的事，可至games.mozilla.org 看看開發者所撰寫的文章或實際參與。
