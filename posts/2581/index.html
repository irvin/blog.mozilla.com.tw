《戰爭機器》製作公司展現強大的 Web 遊戲平台

5月42013

在 2013 年的遊戲開發大會 (Game Developers’ Conference，GDC) 上，Mozilla 發表以 Firefox 執行 Unreal Engine 3 的成果，將逐步解放 Web 作為遊戲平台的強大效能。此程式是以 Emscripten 編譯 C++ 原始碼，不需外掛程式即可順暢執行高效能的作業。

由《戰爭機器 (Gears of War)》系列製作公司 Epic Games 與 Mozilla 合作，即於今天發表了「Epic Citadel」展示軟體，讓眾多玩家可親身體驗。為了達到最佳效果，我們建議使用較高版本的 Firefox Nightly (即 Firefox 23 或以上的版本)，以利用其最佳化的 asm.js，並可支援 Web Audio API。

由於此軟體就是採用了標準的 Web 技術為其核心技術，當然也可以在 Firefox 20 (目前正式發表的版本) 中執行；不過會稍微降低其執行效能，且不支援 Web Audio 的相關音效。

其他瀏覽器同樣也可執行此展示軟體，但受到 WebGL 建構、JavaScript 引擎、記憶體管理等品質所影響的程度較深。

為了確實展現相關的效能優勢，Mozilla 另外開發 asm.js 做為最佳化的 JavaScript 子集合，可讓 Emscripten 編譯過的 App 貼近其原生程式應有的效能。而這個基本的原生效能，就能讓開發人員在 Web 上兼顧絕佳的視覺效果，還有高速且順暢的遊戲體驗。透過此技術，Web 又可成為開發者的另一系統平台。

如果你自己就是遊戲開發人員而且想深入了解細節，可到新的 Mozilla Developer Network (MDN) 的遊戲首頁逛逛。我們會在接下來的幾個禮拜中持續豐富其中內容。如果想針對自己開發的遊戲或 App 看看其他可能性，也可參考 Emscripten 專案與 asm.js 的相關資訊。
