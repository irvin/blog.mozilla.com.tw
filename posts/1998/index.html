Firefox OS 無需下載即可使用 App；Firefox Marketplace 火力全開

2月252013

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織 ── Mozilla 基金會，已於 2012 年推出 HTML5 架構的行動作業系統 Firefox OS，並受到市場的熱切期待。Mozilla 不斷推動網路技術創新，同時致力保護網路環境中的使用者主權。在西班牙時間 2 月 24 日的下午 6 點 05 分，Mozilla 正式同步發表 Firefox OS 與 Firefox Marketplace，可同時支援跨平台並下載應用程式，為使用者提供個人化、功能強大、簡單易用的智慧型手機經驗，而使用者亦可輕鬆開發、發佈、查詢全球的 HTML5 應用程式。

Firefox OS

Firefox OS 是全球首創的開放式網路行動作業系統，以全球主要新興市場為首要切入點，透過合理的價格，達到使用者所期待的智慧型手機效能與個人化功能。Firefox OS 美觀、整潔、直覺、個性化、簡單易用，且其內建功能可協助使用者管理資料與儲存空間；另針對智慧型手機的使用過程中可最大化使用者所感受的價值、效率、整體舒適度。

Firefox OS 包含了使用者所能想到智慧型手機的既有功能，如照相、打電話、傳簡訊、收發電子郵件等。當然亦具備更高階的服務，如內建的預算小幫手、Facebook 和 Twitter 的社交功能、地理位置服務等。最受歡迎的功能即如 Firefox 瀏覽器，可查詢單次使用或可下載的應用程式，另具備 Firefox Marketplace 的服務。

Facebook 企業發展部副總 Vaughan Smith 表示：「具備社交功能的裝置一定更好用！能讓 Firefox OS 使用者輕鬆透過 Web 版本的 Facebook，即可享受到我們現有和未來即將實現的各種功能；我們真的很開心。」

透過 Firefox OS，使用者只要輸入搜尋詞彙，即可迅速建立單次使用或可下載的應用程式。此種「針對需求而建立/使用應用程式」的功能，將協助使用者完全掌控自己的應用程式，進而達到絕佳的手機使用經驗；且不論使用者在任何時候需要某個應用程式，亦可精確地找到其所需。

Firefox OS 提供強大的情境搜尋功能，讓使用者可精準取得自己需要的內容，而不只是相同分類之下的類似應用程式。Firefox OS 另具備更完整的深層搜尋功能，讓使用者可同時搜尋應用程式商店與網路。此功能；其他既有應用程式尚未具備此搜尋功能。舉例來說，使用者可透過 Firefox OS 輸入最喜愛的歌手姓名，即可從搜尋結果中直接購買喜歡的歌曲、演唱會門票、立即線上試聽喜歡的歌曲。一次即可滿足多樣需求。

若您熟知並喜愛 Firefox 瀏覽器的使用經驗，大可將 Firefox OS 視為其延伸。Firefox 早已擁有的個人化、安全性、隱私保護、用戶主導權等所有特性，使用者均可接著 Firefox OS 繼續享有。

由於 Firefox OS 是以純 Web 技術所打造，因此將為使用者、營運商、開發人員提供高度客製化的智慧型手機經驗，同時亦針對手機的所有核心功能提供離線支援。

Firefox Marketplace

Firefox Marketplace 內提供遊戲、商務、新聞/媒體、生產管理等應用程式。相關應用程式將緊密銜接使用者的線上身分，讓用戶可跨裝置/平台而持續使用。全球行動應用與 Web 的主要開發商，均將借助 Mozilla 所賦予 Web 的力量，在 Firefox Marketplace 中不斷發佈各式各樣的應用程式。

Web 本身就引領無限創新，而所有開發商亦將透過 Firefox Marketplace，輕鬆建立並發佈 HTML5 應用，讓使用者可隨時找到所需的應用；甚至不同地區的在地發燒議題亦然。Firefox Marketplace 提供多種普及的應用，例如 AccuWeather、Airbnb、Box、Cut the Rope、Disney Mobile Games、EA Games、Facebook、Nokia HERE、MTV Brasil、Pulse News、SoundCloud、SporTV、Terra、Time Out、Twitter 等。此外，Firefox Marketplace 亦具備個人化的客製功能，可輕鬆打造與當地生活高度相關的在地應用。

美商藝電 (EA) 的 New Platforms and OEM 副總裁 Glenn Roland 表示：「Firefox OS 讓我們有絕佳機會去接觸更多新興市場中的新客戶。我們很高興今年能在跨平台的 Firefox Marketplace 中，發表許多 HTML5 版本的頂尖遊戲與知名應用程式；其中也包含 Pogo 公司的《Poppit!》在內。」

迪士尼 (Disney) Mobile Game 資深副總裁 Bart Decrem 說：「像是《小鱷魚愛洗澡 (Where’s My Water?)》和《鴨嘴獸泰利 (Where’s My Perry?)》的熱門遊戲，都有助於拓展開發中國家的網路玩家。根據以往和 Mozilla 合作的經驗，我很期待 Firefox OS 所擁有的無限潛力。」

MTV Brasil 技術經理 Robson Gomes 亦表示：「透過 Firefox OS，只要是針對行動裝置所開發的新聞、影音、多媒體內容，甚至正在播放的節目，都可用 MTV Brasil 的新應用程式存取。使用者可更輕鬆取得自己喜歡的影音內容，亦可透過此途徑去體驗 MTV Brasil 充滿音樂、幽默感、生活品味的世界。」

目前大部分的行動應用程式，均是以 Web 技術為其核心，用專利技術將之封包後，再發佈到特定平台上。而 Mozilla 則透過 Firefox Marketplace，讓 Web 成為最大的行動開發平台，解除了行動應用程式的層層封包，讓消費者與開發人員掌握更多控制權。

Firefox Marketplace 可讓智慧型手機提供更優質的 Web 使用經驗。隨著 Mozilla 所開發的最新 Web API，Web 將有機會成為全球的最大市場。開放式 Web 平台與相關新 Web API，亦將讓開發人員不需考量任何門檻，即可直接發佈其應用程式，真正實現了Mozilla的使命───為 Web 創造更多選擇、創新、機會。

Mozilla 產品資深副總裁 Jay Sullivan 說：「Firefox OS 的應用程式，均是用像 HTML5 的 Web 技術所打造，可突破應用程式和 Web 之間的隔閡。隨著越來越多開發人員開始在 Web 上開發各種應用程式，我們期待看到更多為 Firefox OS 所開發的精彩應用。Firefox OS 提供了有趣、豐富、個人化的使用體驗，讓大家搜尋各種所需的相關應用，並立即使用之前從未安裝或使用過的程式。」

目前已可於 Firefox 行動瀏覽器開發版 (Firefox for Andriod Aurora) 上預覽 Firefox Marketplace。而今年全球首批搭載 Firefox OS 上市的手機，亦將正式提供Firefox Marketplace。
