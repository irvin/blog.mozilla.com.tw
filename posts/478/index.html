附加元件 SDK 再升級，1.7 版登場

5月172012

各位附加元件開發夥伴，讓我們歡迎 Add-on SDK 1.7 隆重登場！

Add-on SDK 是能幫助你利用 HTML、JavaScript、CSS 等 Web 技術開發 Firefox 附加元件的好東西，有了他、網頁開發者可以輕鬆進入附加元件開發的世界，混搭網路服務、創造更驚奇的體驗。您可以在開發者交流中心下載 Add-on SDK。

而 Add-on SDK 1.7 這個版本一如往常，一方面修正了些問題、二方面也增加了功能，以下描述一些重要的特色：

  • 支援複數詞語在地化：許多語言中複數及單數的用詞並不相同，這一版 SDK 中已經將此納入考量、正式支援複數詞語在地化。
  • 支援「Promise Abstractions」 – 對於不確定什麼時候會結束的非同步動作，Promise 提供一個代表其執行結果的物件、並提供與此物件互動的介面。在 Add-on SDK 中，我們依據 CommonJSPromise/A Implementation 的方式開發此功能。
  • 支援 contentStyle/contentStyleField，讓你可以動態地將 CSS 檔套用於網頁上

還有些其他東西不及備載，請見版本資訊。

歡迎各位開發夥伴玩玩看，並且提供建議與意見。你可用下列方式分享經驗：

  • 貼到討論群組
  • 到 irc.mozilla.org #jetpack 上找我們聊天
  • 回報程式錯誤，或者乾脆捲起袖子看原始碼，一起幫忙解決問題。

其他與 Add-on SDK 專案（又名 Jetpack）相關的訊息，歡迎到 Wiki 上查閱。
