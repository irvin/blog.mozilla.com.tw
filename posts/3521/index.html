快來下載 Firefox for Android Beta 版並測試新功能

Firefox for Android Beta 版

現已可下載並測試 Firefox for Android Beta 版。使用者可透過最新功能而迅速、輕鬆分享內容。

支援 WebRTC：Firefox for Android Beta 版現具備 WebRTC 元件，可讓開發者輕鬆跨 Web 整合即時通訊功能。諸如 DataChannels、PeerConnection、GetUserMedia 等元件，將展現 Web 的即時通訊功能，並於瀏覽器之間分享檔案並撥打視訊電話。開發者現可在 Firefox for Android 上體驗 WebRTC。

強化閱讀器 (Reader) 與閱讀清單 (Reading List)：

  • 閱讀器另重新設計了格式選單，可讓使用者切換 Serif 與 San-serif 字型。
  • 在閱讀器模式下，Firefox for Android Beta 版可根據房間內的光線強度，自動將閱讀器切換為「暗色調 (Dark mode，淺色背景搭配深色文字)」或「亮色調 (Light mode，深色背景搭配淺色文字)」。使用者亦可手動切換此 2 種模式。
  • 使用者只要長按閱讀器的圖示，即可迅速將文章新增至自己的閱讀清單中，而不需先切換至閱讀器再新增文章。並可離線存取所有的閱讀清單。
  • Firefox 現透過 URL 位址列的變化，更清楚呈現閱讀器的開關狀態。位址列將於閱讀器開啟時呈現橘色；反之則為灰色。

NFC Bump：只要有其他 Android 手機啟動了近場通訊 (Near-Field Communication，NFC) 功能，即可相互分享 Firefox 分頁。需先開啟裝置的 NFC 功能。

Quickshare：Quickshare 將根據裝置上可用的分享服務，於「分享 (Share)」選單中建立空間，進而分享如文字簡訊、電子郵件、社交網站等資訊。

Firefox for Android Beta 版新支援了瑞典文、英式英文、加泰羅尼亞的西班牙文，現已超過 24 種語言提供更好的 Web 經驗。

Windows、Mac、Linux 版 Firefox Beta 版

瀏覽器主控台 (Browser Console)：現為開發者更新了瀏覽器主控台。瀏覽器主控台可讓開發者檢測瀏覽器的記錄檔 (不限內容網頁的記錄檔)，對 Firefox 附加元件的開發者來說更有用。開發者現可根據記錄類型 (網路、CSS、安全性、JS 等記錄) 進行篩選或全文搜尋。此功能將取代現有的錯誤主控台 (Error Console)。
