Firefox 桌面版與行動版今天再添強化功能

我們最近幾個月來持續為 Firefox 愛用者強化使用經驗。今天很高興能為一般使用者首次釋出多程序 (Multi-process) 的 Firefox 桌面版 (Firefox for Desktop)，朝著更完美的 Firefox 又邁進一大步。使用者可發現 Firefox 現已大幅減少了畫面凍結的情況，也同時保有自己最愛的經驗與特色。

我們將透過 Firefox 48 逐步釋出多程序 Firefox (即所謂的Electrolysis 或 e10s) 予使用者體驗。開頭會先從 1% 的小眾使用者開始，另視情況再慢慢開放給將近一半的用戶。e10s 期能區隔 Web 內容與 Firefox 使用者介面 (UI) 的處理程序，藉以大幅提升瀏覽經驗。這意味著，即便某一網頁正消耗電腦大部分的處理資源時，你眼下的分頁、按鈕、選單也不至於遭鎖定而毫無回應。想知道你的 Firefox 是否啟用了 e10s 嗎？在位址列中輸入「about:support」之後，如果看到「多程序視窗 (Multiprocess Windows)」條列項目顯示為「1/1 (Enabled by default)」，就代表你的 Firefox 瀏覽器已啟用了 e10s。

除了改變 Firefox 桌面版未來的基本運作方式之外，另亦更新了瀏覽器的設計，讓你能更輕鬆搜尋多樣的內容。我們首先讓「智慧位址列 (Awesome bar)」更聰明了。現在只要你輸入新的查詢字串，就會看到更多、更豐富、顯示範圍更大的建議搜尋結果，讓使用者更易於閱讀。如果建議結果是某個網站，且該網站已經記錄於你的書籤或已開啟為某個分頁，旁邊也將透過圖示來提醒使用者。

我們另重新設計過 about:addons 的探索面板，協助你個人化自己的瀏覽經驗。新設計更針對精選附加元件簡化了安裝程序，只要點擊一次就可透過清爽的圖片與文字盡情體驗 Firefox。

我們又再提升了安全性，強化 Firefox 的下載防護功能。到這裡進一步了解。

最新 Firefox 行動版 (Firefox for Android) 也同樣添增了新功能。我們在今年稍早已經試驗了重新設計的書籤。 今天已正式將「閱讀清單」整合至「書籤」；「已同步的分頁」則加入「瀏覽紀錄」面板之中。你現在可跨裝置在書籤中找到自己的閱讀清單項目，不論何款裝置都能輕鬆存取自己喜愛的內容，可說是 Firefox 跨裝置的重大升級。

鑑於現在有越來越多人線上欣賞影片，我們也改善了 Firefox 行動版的音訊與視訊播放經驗。現在只要你用 Firefox 欣賞影片時正好有來電，裝置將自動暫停影片讓你專心回電。另新增的工具列可讓你在 App 中迅速取用音訊控制功能，且所有影片內容均已搭配更流暢的觸控功能，讓你使用起來更得心應手。

接下來數個月內，我們會持續為大家開放 Eelectrolysis 功能。就請你儘快下載 Firefox 桌面版與行動版的最新版本，隨時讓我們知道你的寶貴意見吧！
