松下電器 (Panasonic) 與 Mozilla 為新一代 Smart TV 協力開發新的 Firefox OS 開放平台

1月82014

日本大阪 / 美國山景城 ─ 松下電器公司 (Panasonic Corporation) 與 Mozilla 於今天共同宣佈合作關係，將攜手打造搭載 Firefox OS 的新一代智慧型電視 (Smart TV)，協力開發與推廣 Firefox OS 此一以 HTML5 與 Web 技術所打造的開放式平台。

全新 Firefox OS 平台將進一步延伸到 Smart TV，技術方面採用已在個人電腦、平板電腦、智慧型手機取得優勢的 HTML5 與 Web 技術。消費者將擁有更優化、更個人化的 Web 使用經驗，並透過網際網路輕鬆取得廣播內容與服務。

在新的 Firefox OS 開放平台問世之後，下一代 Smart TV 將可完全相容於目前的雲端服務，或未來多款連網裝置所使用的 Web 技術與 HTML5 標準。只要是來自於 Web 服務與裝置的資料，都將可透過單一 App 而輕鬆整合。開發者可交互運用網際網路與廣播的內容，靈活建構出新的App 與服務。另透過 Mozilla 率先提倡的 WebAPI (用於硬體控制)，新一代 Smart TV 亦可監控並操作居家內外的裝置，如現正普及的智慧型家電。

目前如電視選單與電子節目指南 (Electronic Program Guide，EPG) 等基本功能，均採用嵌入式程式撰寫。但在搭載 Firefox OS 的新一代 Smart TV 中，將以 HTML5 來撰寫相關基本功能，開發者可針對平板電腦與智慧型手機輕鬆設計 App，進而能遠端存取並操作電視。此外，透過 Web 服務，Smart TV 更將顯示個人化的介面，提供使用者最愛、最常用的功能，亦可添增新功能讓多位使用者共享同一畫面。

「Panasonic 透過自己的入口網站，不斷為自家電視擴充多樣的內容與服務。我們亦將持續與 Mozilla 合作，加速創新 Firefox OS 並激盪出更多新服務。」松下電腦 (Panasonic AVC Networks Company) 電視事業部總監 Yuki Kusumi 表示：「透過與 Mozilla 的夥伴關係，Panasonic 將進一步創新 Smart TV 的技術與功能，勢必讓消費者在居家內外都能獲得全新的互動與連網經驗。」

「Mozilla 很高興能和 Panasonic 建立合作關係，讓 Firefox OS 能透過更多平台觸及更多使用者。我們仍持續尋找更多願意支援 Firefox OS 與 Open Web 的夥伴。針對智慧型應用的解決方案，Firefox OS 更將強化 Open Web 的標準。」Mozilla 亞洲區運營總裁兼行動裝置資深副總裁宮力博士表示：「開發者和服務供應商可透過此新平台提供更多樣的 App 與服務，並達到全新的使用者經驗。我們發現 Mozilla 與 Panasonic 擁有共通的願景，而進一步整合雙方的專業技術之後，勢必將推出令人驚豔的絕妙產品。」

搭載 Firefox OS 的新一代 Smart TV 將由 Panasonic 發表，而 Mozilla 將與 Panasonic 一同推廣 Firefox OS 與開放的生態系統。

有關松下電器 (Panasonic)

松下電器公司 (Panasonic Corporation) 為全球電子技術與解決方案的領導開發商，專為消費者提供居家、戶外、行動、個人化等多項應用產品。自 1918 年創立以來，Panasonic 將業務擴展至全球，現已營運超過五百家附屬公司。而在 2013 年 3 月 31 日會計年度共達 7.3 兆日圓淨銷售額。Panasonic 持續跨多樣產品線而創造全新價值，並致力為消費者打造更優質的生活與更美好的世界。可至 http://panasonic.net/ 網站進一步了解。

有關 Mozilla

全球性非營利組織 Mozilla 致力打造 Web 的美好未來超過十五年。我們堅守自己的理念，致力以開放的標準進行創新，並推動 Web 成為眾人所共有的平台。目前全球共有五億人透過 Mozilla Firefox，在自己的電腦、平板電腦、智慧型手機上悠遊無垠的網海。可至 https://www.mozilla.org/ 進一步了解。
