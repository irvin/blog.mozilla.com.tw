募集強大的 Web App 開發工具

Web 最近幾年的發展變化劇烈。行動 Web 與新規格的智慧型手機崛起，也讓桌機規格之外的多樣 Web 解決方案應運而生。

現在早已有許多框架與工具可用，而且幾乎每個禮拜都有新工具誕生。因此 Web 開發者反而又要面對不同的問題：開發過程中的各個要點都有多樣工具可供選擇，且這些工具都沒有很明顯的優缺點可挑剔。讓人害怕的不只是這些工具的數量，其相似度也讓人難以抉擇。

Web 開發者每天都必須先克服這些問題，將這種多樣性轉為自己的優勢。

講到這裡，大家好像一直在重新發明輪子？

為 App 開發者推薦好用且完整的工具

Mozilla 現正集結核心工具，並要進一步推薦給 Web App 開發者，以協助大家能更順利的開發出作品。

我們會納入推薦清單的主要考量在於：

  • 現有完整的相關說明文件，且一般開發者都能輕鬆上手 (要接觸某項技術所必備的知識，我們也會清楚列出)
  • 儘量挑選低耦合 (Loosely coupled) 且模組化的工具 (如果你比較容易游移不定，就只要選用其中一項特性即可)
  • 已通過 Mozilla 產品的測試 (例如 UI 元件可於 Firefox OS 正確執行)；但將來都可能成為跨平台 App

即將集結而成的推薦工具，將囊括目前 Web App 核心所用的工具鏈，如 UI 框架、JavaScript 框架、排版工具/介面、自動化開發流程工具 (如 Grunt) 等。我們會儘量利用現有的解決方案，並寫出函式庫或公用程式以填補不足之處。

在目前進行的基礎上，我們會系統性的逐漸擴充，以跨不同的開發經驗，如離線處理或使用不同的 Web API。

這些成果都會集中發佈於 Mozilla 開發者社群網站 (MDN) 的《應用程式中心》。

要是我早就會用這些東西了呢？

Mozilla 誠摯邀請你和我們分享這些成功經驗，當然失敗的經驗同樣無比珍貴。我們隨時都需要Web 開發者提供反饋意見。如果你早就有自己用得順手的工具鏈或 JavaScript 框架等等，我們也不會要你轉用其他工具。Mozilla 只是想協助開發者能正確選用所需的工具。

加入我們！

根據 Mozilla 的傳統，這個專案將同樣交由社群主導。我們歡迎並感激你所貢獻的任何東西，也希望能因你的協助而完成此專案！

以下管道可供相關的討論活動：

  • 透過 dev-webapps 郵件群組。如果你就是 Web App 開發者，請儘快訂閱這個郵件群組
  • 當然也能到 irc.mozilla.org 上加入 #apps

針對剛入門的工具鏈推薦清單，我已經在郵件群組上開了討論串 (英文)，你可隨時加入。如果你覺得有任何值得大家討論的主題，也歡迎隨時開新的討論串。

接下來呢？

如果這篇文章引起你的興趣，那就太好了！對 Web App 開發者來說，2014 年絕對是充滿驚奇的一年。我們會隨時透過 Hacks 與 MDN上的《應用程式中心》發佈最新資訊。
