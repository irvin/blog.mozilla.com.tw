Mozilla@校園徵才博覽會 – 台大場 活動預告

地圖不可用

活動時間

日期 2014/03/09

10:00:00 - 16:00:00

地點

春季、三月微雨，Mozilla 即將在台大校園徵才博覽會與各位相見歡！為了招募更多的台灣年輕的精英參與 Firefox OS 開發，打開國內優秀軟體人才與國際接軌的管道，Mozilla 首度前進校園徵才博覽會，現場將由 Firefox OS 研發團隊核心工程師，與即將進入職場的學子直接面對面，除了提供多元化的外商工作經驗，更將大方分享國際開源軟體專案的工作實務經驗！

台大校園徵才現場除了專業的 Firefox OS 研發工程師之外，風靡全球擁有最多粉絲愛好者的超級吉祥物 – Firefox 火狐人偶本尊，也將現身 Mozilla 美商謀智現場攤位，喜愛 Firefox 的你，請千萬把握機會與火狐本尊來個愛的抱抱，而現場將照片上傳臉書並完成指定步驟者，還可獲得神祕限量禮物一份！

難得一次的現場見面活動，Mozilla 攤位熱鬧獎不完！其它相關現場活動：

  • 火狐愛抱抱 ：現場限時拍照上傳活動，即贈神秘禮物
  • 加碼送 ：現場手機分享 MyFirefox 文章，即贈火狐小徽章
  • 加碼抽 ：現場留下個人相關徵才資料，除獲贈小禮物外，還有機會抽中 Firefox OS 手機一只 (台灣未上市)

若你是尚未畢業的學生，秉持開放精神的 Mozilla 同樣提供你參與的機會，包括熱血的社群志工、全球性 Firefox 校園大使專案、Mozilla 實習生專案、即使不具備就業身份，依然有機會對網路世界作出貢獻、並於過程中體驗最新知識技能、於學生時期開始培養自身實力與經驗！

對網路技術、開源專案、全新型態智慧型手機有興趣的你，不需要是懂得 C++ 的工程師，只要你對網路抱有一份熱忱、想做點什麼來改變這個世界、相信分享與開放能成就最大的力量，請別錯過這次與 Mozilla 面對面的機會！

Mozilla Taiwan 美商謀智攤位號碼：椰林大道 32, 34
