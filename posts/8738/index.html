Firefox 繼續為 Ubuntu 預設的瀏覽器

4月222016

另將導入新的「snap」封裝格式

Mozilla 以透明、選擇、信任為基礎，除了要提供絕佳的使用經驗之外，也致力讓 Firefox 能跨多款平台、裝置、作業系統。而 Mozilla 與 Canonical 今天更新了雙方的合作關係，讓 Ubuntu 繼續以 Firefox 為其預設瀏覽器。我們很榮幸已和 Ubuntu 合作超過 10 年。Canonical 的背景與 Mozilla 類似，亦屬於開放源碼且有賴社群從旁協助的組織。


圖片來源：http://betanews.com

Ubuntu 16.04 版本另將導入「snap」封裝架構。透過此 snap 格式，我們即可持續針對 Ubuntu 而進行 Firefox 最佳化。就如同我們的版本釋出週期一般，即便已經過了作業系統的主要釋出日期，snap 格式也可讓 Linux 使用者隨時享受最新功能，特別是重要的安全性修正。

在此之前，Firefox 各個版本都是固定隨著新版本的作業系統一同釋出。但透過 snap 格式，舊版作業系統的使用者也可同步享受最新功能。今年稍晚，我們就會提供snap格式的 Firefox，以更輕鬆的將瀏覽器推送給使用者，而不需再依賴其他媒介才能讓使用者接收更新檔案。
