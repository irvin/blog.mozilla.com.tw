新版Firefox for Android Beta 正式展開測試

5月162012

全新Firefox for Android Beta 版本即日起可在此處下載及測試。此版本的特色在於嶄新的使用者介面設計，操作起來宛如Android 系統的原生程式，提升使用效能的同時更加入如 Flash 等好用的第三方外掛程式，以期提供使用者在行動裝置瀏覽的最佳體驗。

全新Firefox for Android Beta 版本具備所有您熟知並喜愛的的好用功能，例如Firefox Sync（同步功能）、Add-ons（附加元件）、Tabbed browsing (分頁瀏覽)，再加上全新 Awesome Screen（智慧畫面）、迅速開啓／回應時間的強化、圖像顯示和平移放大功能的優化。

Mozilla 針對 Firefox for Andoid 持續推出Beta 版本供開發和測試，而針對此版本，我們投注了更多的心力在提供更快更便利的行動瀏覽體驗。Mozilla 誠摯邀請喜愛Mozilla 的你，一起加入測試的行列，不論是從程式開啓／執行花費的時間，到圖像顯示功能的測試，Firefox for Android Beta 使用硬體加速來提升您使用 Web apps、遊戲及圖像的效能，在測試過程中，若您發現任何不順暢或問題發生，歡迎點此回報問題。

全新Firefox for Android Beta 版本支援多款Android 系統手機，從Android 4.0 ICS冰淇淋三明治作業系統到Android Froyo 2.2，此版本目前針對Android 系統提供英文版本，其他語系以及平板電腦版本將在近期內推出。
