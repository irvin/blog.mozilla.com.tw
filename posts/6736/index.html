菲律賓推出第一支 Firefox OS 智慧型手機

Mozilla 堅守己身使命，致力推動 Web 的機會、開放、創新。今天很高興宣佈 Cherry Mobile 即將於在菲律賓 (Philippines) 發表第一支 Firefox OS 智慧型手機。Cherry Mobile 推出親民價位且可自訂的 Firefox OS 手機 ─「ACE」，即透過開放源碼作業系統提供絕佳的 Web 體驗。另包含 Easy Taxi 與 OLX Philippines 在內的重要 App 廠商，亦將進一步豐富 Firefox Marketplace 並於菲律賓拓展 Firefox OS 生態系統。

Cherry Mobile 執行長 Maynard Ngu 說：「Cherry Mobile 在這幾年來恪守本身理念並不時留意當地的生活風格。我們瞭解客戶均有其個人偏好，也因此必須保持自己的活力並汲取多元資訊。透過和 Mozilla 的合作關係，我們將繼續為菲律賓民眾提供高品質產品並滿足大家的需求。Cherry Mobile 很高興能發表『ACE』手機，也是我們第一支執行 Firefox OS 的智慧型手機。」

Mozilla 總裁宮力博士表示：「很榮幸能看到 Cherry Mobile 在菲律賓發售『ACE』這款 Firefox OS 智慧型手機。有了另一位強大夥伴的支持，Firefox OS 當持續於亞洲茁壯。這些超低價位的 Firefox OS 裝置均將重新定義入門款智慧型手機，也能讓越來越多人以親民價格享受 Web 的力量。」

Easy Taxi 的行銷長 Paul Malicki 接著表示：「我們很高興能與 Firefox OS 攜手合作，進而拓展菲律賓市場並繼續參與未來的行銷企劃。HTML5 平台的靈活性與穩定度，確實幫助我們能在拉丁美洲順利推廣。而我認為本次發表活動，即因應了目前的合作關係而自然發生。菲律賓的年輕人口構成 Firefox OS 絕佳市場條件，必將於學生或商業流程委外 (BPO) 的科技員工之間產生吸睛效應；且此兩族群也是 Easy Taxi 的重要客戶群。特別期待加入我們的第一名 Firefox OS 乘客，說不定其中某些人可以得到我們提供的特別驚喜？」

OLX Philippines 管理總監 RJ David 也說：「很高興看到 Mozilla 的新款 Firefox OS 裝置，也期待能帶領菲律賓的行動市場大幅躍進。OLX 極重視行動用戶的體驗，而本次合作關係將為客戶打造更快、更簡單易用的銷售平台。Mozilla 勢必能為我們在菲律賓帶入另一批百萬名客戶並銷售其二手物品。」

Firefox OS 為首款全以 Open Web 標準所打造的行動平台，所有功能均開發為 HTML5 App。其靈活性、可調整度、強大的自訂特性，可讓使用者、開發者、產業夥伴建構自己所需的行動體驗。

Cherry Mobile 的「ACE」手機搭載 Firefox OS 與 1GHz 處理器，可擴充的記憶體並可達最高 16GB 容量。另可透過 Wi-Fi、藍牙、雙 SIM 卡輕鬆連線，且內建功能即可讓使用者監控自己的資料流量。該款手機亦支援 Firefox OS 的多個語言版本，且提供黑、黃、白、橘、亮藍、亮綠、亮紅等顏色樣式。

隨著 Firefox OS 邁入菲律賓市場之後，目前已於 27 個市場的當地夥伴共同合作，遍及歐洲、亞洲、澳洲、拉丁美洲。
