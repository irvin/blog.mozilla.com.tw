Mozilla 更新 Aurora 版本 Firefox Accounts 首度問世

2月112014

【2014年2月11日，台北訊】為了讓廣大 Firefox 使用者獲得最佳 Web 經驗，美商謀智 (Mozilla) 在最新發表的 Windows、Mac、Linux 版 Firefox 曙光版 (Aurora) (註1)中，首度推出 Firefox Accounts，使用者能輕鬆建立自己的帳戶，隨時透過 Firefox 登入而取得所須服務。此外，此次更新更大幅修改使用者介面 (UI)，外觀與自訂選項皆更加流暢靈活，同時Firefox Sync同步功能也更加簡單。

此次 Firefox Aurora 更新，Mozilla 一口氣推出 Windows、Mac、Linux 版提供使用者下載測試。此一測試版本最重要的特色，莫過於 Firefox Accounts 功能首度問世。以往 Mozilla 持續努力進行下載與更新作業的最佳化，也不斷嘗試將 Firefox Sync、WebRTC、社交媒體等功能整合入附加元件 (add-ons) 平台。現在透過 Firefox Accounts，使用者能輕鬆且安全的建立自己帳戶，並能隨時透過登入 Firefox 帳戶獲得內建於 Mozilla 產品中的各項服務。Mozilla 透過 Firefox Accounts，進一步為使用者整合相關 Web 服務。

除了全新 Firefox Accounts，此一版本也大幅更新 Firefox 使用者介面，新介面提供了更靈活的自訂選項、全新的流暢外觀，以及新版 Firefox Sync 同步功能。Firefox Aurora 現為重新設計過的瀏覽器介面，可協助使用者更迅速完成瀏覽作業。分頁具備嶄新流暢外觀，且閒置中的分頁將退回背景，使用者能更快速找到自己所須分頁。新的選單亦提供最常用的功能 (包含複製、貼上、縮放)；附加元件均加以整合為一，並以清楚圖示呈現。

此外，瀏覽器的自訂功能也更簡單、更易於尋找。選單中的新自訂功能，讓使用者能將選單、工具列、分頁控制列直接拖曳到所須位置，安排自己最愛用的順序；當然也能根據自己的喜好，輕鬆移除或搬動瀏覽器或附加元件的按鈕，使用者可以更快、更輕鬆取得最常用的任何功能。

而此次新版 Firefox Sync 同步功能，進一步簡化了服務與增添裝置的設定，並仍維持瀏覽器加密機制。其具備端點對端點加密 (End-to-end encryption) 機制，只要輸入電子郵件位址再選好密碼，即完成整個設定程序。Firefox 使用者能更輕鬆跨裝置攜帶自己的瀏覽資料 (如書籤、密碼、瀏覽記錄、已開啟的分頁)。

下載 Firefox Aurora版： http://mozilla.com.tw/firefox/channel/#aurora

更多功能細節，請參閱美商謀智部落格：http://blog.mozilla.com.tw/main

註 1：Firefox 釋出週期約為6周，共有 4 個釋出頻道。目前最新正式版本為 Firefox 27，適用於一般大眾；Firefox Beta版則是讓先期使用者能在相對穩定的環境下體驗最新功能；而Firefox Aurora版為 Firefox 的未來發行版本，使用者多具有技術背景，能在穩定度相對較低的環境下率先體驗更多創新瀏覽輕驗；另一 Firefox Nightly版則仍處於設計測試階段。
