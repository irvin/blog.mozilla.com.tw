Firefox Home 正式退役！功能可使用至年底

9月32012

早些時候 Mozilla 一直專注於要讓使用者在不同的平台與裝置上享有共同的美好體驗，為此我們為 iOS 的使用者釋出 Firefox Home，將 Firefox 部分功能帶進 iOS，主要是 Firefox Sync (同步功能)。此舉也為該平台帶來了相當程度的價值與創新體驗，而如今我們決定要將 Firefox Home 從 Apple App Store 下架，從而將資源分配到其他的專案。

若您想繼續使用或協助改善 Firefox Home 的 iOS 同步功能，Mozilla 已將該專案的程式碼放到 GitHub 供下載，並移除 Mozilla的商標讓其可獨立運作。如同 Mozilla 釋出的其他專案，我們希望開發者先熟知 Mozilla 商標政策。

Mozilla 依然矢志承諾，要繼續提供使用者絕佳的跨平台與跨裝置使用經驗，並將持續探索提供 iOS 使用者更優質經驗的可能。

Firefox Home 到今年年底前仍可繼續使用，這段期間 Mozilla 將試圖導引使用者參考其它可能的相關 Apps。
