以 Firefox 附加元件跨裝置與瀏覽器開發 Web

9月252014

開發 App 時的主要問題，就是必須跨多款裝置與瀏覽器。如果只要一種工具就能跨 iOS、Android、桌機進行 App 除錯作業，不就太方便了嗎？我們認為 Web 力量絕對可提供行動 Web 的開發解決方案，以滿足所有需要！

快來試試目前屬於實驗階段的 Firefox 附加元件「Firefox Tools Adaptor」，可銜接 Firefox 開發者工具與他款主要瀏覽器的引擎。我們專為 Firefox OS 與 Firefox for Android 所設計的絕妙除錯工具，可透過此附加元件而於 Android 的 Chrome，以及 iOS 的 Safari 這兩款主要行動瀏覽器上進行除錯。目前可用的 Firefox 開發者工具包含檢測器 (Inspector)、除錯器 (Debugger)、主控台 (Console)。

實機測試一直是最重要的階段。但裝置上的開發者工具往往不好用，或必須搭配特定廠商的裝置才能使用。在跨平台開發之前，開發者必須先學習使用不同瀏覽器的開發工具並來回切換。

而透過此附加元件，就能在自己桌機環境中開發多款小螢幕裝置的 App，且不會佔用寶貴的螢幕空間。不論該裝置搭載何種平台與瀏覽器引擎，只要使用該裝置就能在自己電腦上看到運作情形。

看看此附加元件如何運作

現在就體驗

本專案現仍屬早期階段。但針對許多好奇且想貢獻的開發者，我們提供了預覽版本。只要下載最新的 Firefox 每夜更新版 (Nightly) 與此附加元件即可。請依照 Firefox Tools Adapter 說明立刻動手開始。

此預覽版本可搭配 Android 版的 Chrome 37，目前亦可用於 iOS 上的 Chrome Beta 與 Safari。某些部分運作得還不錯，但仍有其他部分需要再加強。立刻加入測試並將你的意見提供給我們！

內部如何運作？

此附加元件是根據 Firefox 開發者工具的通訊協定實作而成，透過 Chrome 與 iOS 所開放的遠端除錯協定進行溝通，而不是直接介接其內容。此實作建構於 Firefox 程序之內，交由 Firefox 開發者工具於內部使用。

何時提供正式版本？

我們目前提供早期預覽版本，並將根據開發者的意見而於未來數個月內積極開發。只要準備釋出正式版本時，隨即會透過部落格告訴大家！
