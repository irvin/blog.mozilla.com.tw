Firefox for Android 正式支援 WebRTC

9月182013

Mozilla 很開心的宣佈，新版 Firefox for Android 現已支援全新 HTML5 API ─ WebRTC 即時通訊功能！透過瀏覽器搭載的 WebRTC 功能，即可透過瀏覽器直接撥打視訊電話。開發者如今也能為行動平台建構即時通訊的 Web 體驗，並可跨平台同時在桌面與行動瀏覽器上進行溝通。

Firefox for Android 現已支援 WebRTC 全部 3 大元件：

  • getUserMedia (gUM) ─ 可存取使用者的麥克風與網路攝影機
  • PeerConnection ─ 可於瀏覽器之間進行通訊
  • DataChannels ─可於瀏覽器之間進行點對點資料傳輸

自 6 月起 Firefox 桌面瀏覽器已全面支援 WebRTC，包括 Windows、Mac、Linux 版的 Firefox。Mozilla 今天更進一步讓 Firefox for Android 也支援 WebRTC。若想進一步了解這個新加入 Firefox for Android 的絕妙 HTML5 功能，可參閱本篇文章以獲得更多資訊。

請各位努力發掘 WebRTC 的潛力！我們滿心期待著 WebRTC 的酷炫新體驗。

– WebRTC 資深工程經理 Maire Reavy
