史上第一次，Firefox 與 Chrome 跨瀏覽器新利器 – WebRTC

2月52013

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－ Mozilla 基金會，2012年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。於美西時間 (4日) 釋出重大里程碑，WebRTC技術讓 Firefox 和 Chrome 可互通，可傳送清晰音訊、高話質影像(HD)和優良通話品質。而此技術乃由開放社群 W3C、IETF、Firefox 以及 Chrome 工程師共同開發打造。這個技術是未來 Web 朝向即時互通、真實、開放美好前景的第一步。

純 Web 打造的 WebRTC 技術，可幫助開發者在 Web 開發過程中，擁有具便利性、整合性以及安全的視訊體驗，由於這項功能被內置在瀏覽器中，使用者不需安裝外掛程式也能進行使用，開發者也可利用 Firefox 的 WebRTC 技術直接向 Chrome 的WebRTC 技術發出需求，Chrome 和 Firefox 瀏覽器使用者現在可以用 Opus 和 VP8編解碼器進行影音視訊；DTLS-SRTP 來加密；ICE 做網路傳輸。

WebRTC 展示影片。

左為 Google 產品管理總監 Hugh Finnan、右為 Mozilla 首席創新官 Todd Simpson。

欲嘗試的使用者可用桌機版的 Firefox Nightly 以及Chrome 25 Beta 版，Firefox 使用者請至 about:config 並設定 media.peerconnection.enabled pref 到 “true.” 接著前往 WebRTC demo site 並開始體驗。
