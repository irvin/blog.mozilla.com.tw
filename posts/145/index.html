Mozilla 踏足移動市場 – 讓網路成為最大平台 於世界行動通訊大會(MWC)宣佈首款網路手機合作夥伴

2月272012

(2012/02/27 巴賽隆納訊) 致力於網路開放自由的 Mozilla，其 Firefox 瀏覽器在全球擁有數億名愛好者，今年擴大參與世界行動通訊大會 (Mobile World Congress / MWC)，並於 MWC 開展的首日舉辦記者會，宣佈與西班牙電話公司 (Telefonica) 合作打造首款 Boot to Gecko (B2G) 網路手機。其他業界領導廠商如：Adobe 和高通 (Qualcomm) 也對B2G專案公開表示支持，並共同承諾將以HTML5 做為移動市場主要的平台選擇。

於今日舉行的聯合記者會中，Telefonica 宣佈將與 Mozilla 合作於 2012 年發表全球第一款 HTML5 網路手機。該手機完全採用網路環境打造，技術根植於 HTML5 和 Web APIs，預期該款手機在速度和價格上將極具優勢。記者會中並展示由 HTML5 開放技術所打造的網路手機。Telefonica 表示，Mozilla 的 B2G 專案將現今行動裝置上網路的限制完全鬆綁，透過新的功能與 APIs，完全釋放 HTML5 的技術潛力。之後的行動裝置不再需要任何作業系統與中介軟體，透過 HTML5 執行的應用程式將更快速，並更具價格優勢。

Telefonica 是一家國際電訊公司，其服務包括固定通訊線路，行動電話，網路，有線電視…等，是西班牙語和葡萄牙語國家影響重大的電信公司。Telefonica 是西班牙最大的跨國公司，也是世界上最大的電信公司之一，市場包括歐洲及拉丁美洲，全球擁有超過三億客戶。

除了與 Telefonica 的合作之外，Mozilla 對開放網路平台的計劃亦獲得德國電信公司 (Deutsche Telekom) 的支持。Deutsche Telekom 已承諾其 Innovation Labs 將投注固定資源，參與 Mozilla 的 Boot to Gecko 專案，並透過雙向交流評估進一步的可能性。

在 MWC 展間，Mozilla 也展示了各式 Web apps 和 Mozilla Marketplace，全力推廣由 HTML5、CSS 和 JavaScript 撰寫的跨平台 Web Apps。此外，Mozilla 也展示全球第一個名為 Persona 的網路身分識別服務，強調保護個人在登入網站時的隱私權。Mozilla 展示一系列最新專案與工具，讓使用者和開發者在各式行動裝置上，得以掌控自己的網路經驗。

Mozilla 一直以來都以推動網路的完全開放作為首要使命。Mozilla 已將目前推動的所有創新專案技術向 W3C 申請標準化，期望可帶領網路的創新與進步，真正打造一個以人為本的網路環境，讓網路本身成為最大的平台，也是 Mozilla 對全球社群的承諾。

Mozilla 攤位資訊：Mobile World Congress, Stand 7B96, Hall 7
