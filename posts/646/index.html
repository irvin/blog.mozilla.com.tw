快速、聰明、安全的Firefox for Android 版本全新登場！

6月272012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，旗下瀏覽器Firefox for Android 版本於美西時間26日釋出。Mozilla 將桌機市場成功推動的開放精神延伸至行動市場，今宣佈Firefox for Android 版本可以在Google Play store 進行下載。新版本使用嶄新的使用者介面設計提供更快、更便利、最安全的行動上網體驗。

超快效能

我們投注了很多的心力，設計出速度快得驚人的Firefox for Android新版本。大幅增進的啟動、頁面載入、移動及縮放頁面、Web app執行效能速度，新版本都能讓使用者感受到前所未有的絕佳體驗，您甚至可以邀朋友一起在手機上玩 Mozilla 的 BrowserQuest 多人線上遊戲。根據業界諸如Sunspider的指標性效能評比之測試結果，Firefox for Android 版本效能相較於內建的瀏覽器快上了許多。此外，根據Mozilla 開發名為 Eideticker 的測試工具，測量使用者的網路瀏覽狀態，Firefox for Android 在此測試中得到超越內建瀏覽器兩倍的優異成績。

行動瀏覽再進化

擁有全新的使用者介面及個人化的啓始畫面，不單節省時間和便利性，加入Firefox Sync（同步功能）的Awesome Screen（智慧畫面），更把使用者在電腦中的瀏覽紀錄、書籤、密碼，以及表單資料同步至 Android 手機。同時，針對您最愛的功能，例如分頁瀏覽、同步、以及附加元件做了最佳化，提供您更好的行動瀏覽體驗。

新版本加入支援 Flash功能，使用者可以在瀏覽器中觀賞您喜愛的影片、玩遊戲，或是使用其他互動式網路體驗。此外，新加入的Do Not Track 功能，提供使用者最私密的安全防護。Firefox 尊重您的個人隱私，透過諸如Do Not Track、主控密碼、HTTP 強制安全傳輸技術（HSTS, HTTP Strict Transport Security）等各項安全功能來保護您的隱私。

全面邁向網路平台

新版本支援更多 HTML5 功能，提供開發者使用 HTML5、JavaScript、CSS，及其他開放網路標準為基礎製作網頁、開發網頁應用程式。作為網頁技術的提倡者，Mozilla 設計了全新的 Web API 並提交至標準化組織，致力於讓網頁全面進化成威力無窮的網路平台。Mozilla 已協助制定了攝影（Camera API）、震動（Vibration API）、行動連線（Mobile Connection API）、電力狀態（Battery Status API）、螢幕方位（Screen Orientation API）及地理資訊（Geolocation API）等標準 API。

您可以在電腦與行動裝置上試玩 Mozilla 的 HTML5 BrowserQuest 多人線上角色扮演遊戲，體驗網頁的速度可以多快、多好玩，威力多強大。
