Firefox Beta 新增內建 PDF 文件閱讀器，純 HTML5 和 JavaScript 技術打造

1月112013

可供 Windows、Mac 和 Linux 使用者下載和測試的全新 Firefox Beta版本現已開放使用。新功能加入原生 PDF 文件閱讀器，使用者不需安裝額外的外掛程式，就能獲得快速、安全的瀏覽體驗。

同時，Firefox Beta 版本支援開發者，在桌機環境下測試 Firefox for Android 和 Firefox OS 中的行動 Web 應用。開啟遠端除錯器 (remote debugging)，請到 devtools.debugger.remote-enabled 為 true 即可。
