Mozilla 與合作夥伴即將發佈首款 Firefox OS 智慧型手機！

7月12013

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球性非營利組織 Mozilla，為了讓所有人都能享有 Web 的力量，於今日 (1日) 宣佈展開 Firefox OS 智慧型手機的區域性銷售。Deutsche Telekom 與 Telefonica 將發表其首款 Firefox OS 裝置 ─ ALCATEL ONE TOUCH Fire 與 ZTE Open。其他合作夥伴也即將在各個市場宣布相關推出細節。以下就 Firefox OS 特色與合作夥伴證言做說明，並歡迎參考Firefox OS 展示影片: http://youtu.be/Iu8q-oISbas。

Firefox OS 的特色

Firefox OS 智慧型手機，是第一款完全由 Web 技術打造而成的裝置，除了具備智慧型手機的功能之外，亦提供全新具特色，且帶給消費者其他手機無法比擬的個人體驗。

Firefox OS 具備智慧型手機的基本功能 ─ 接撥電話、收送訊息、電子郵件與相機功能… 等等之外，亦同時滿足消費者對智慧型手機的期待。Firefox OS 除了內建 Facebook 與 Twitter 的社交功能，也提供地理位置服務的 Nokia HERE 地圖，可查詢在地交通資訊並提供離線使用功能。此外，Firefox OS 手機也內建廣受喜愛的 Firefox 瀏覽器、提供最新搜尋功能讓使用者找到可一次性使用或永久下載的 Apps、以及 Firefox Marketplace… 等多項功能。

Firefox OS 對智慧型手機作出全新詮釋 — 「App 適應性搜尋功能」，讓手機不需透過任何下載即可滿足消費者的各項需求。只要在 Firefox OS 主畫面上向右滑動，輸入你感興趣的關鍵字，就可依據你的需求自訂最符合個人需求的全新體驗。

舉例來說，你可搜尋自己最喜愛的音樂歌手，即可得到歌曲或音樂會門票的購買資訊，亦能立刻聆聽喜歡的單曲。你也可以針對這些內容建立一次性使用或可下載的 Apps，完全客製化自己的使用經驗，讓你隨時隨地都能取得所需要的服務內容。

「Firefox OS 的發表將激勵整個行動產業。Firefox OS 實現了第一款完全以 Web 技術打造的智慧型手機，它將刺激並帶動新一波 Web 技術的創新風潮。」Mozilla 首席營運長 Jay Sullivan 表示：「我們很榮幸能夠為使用者帶來無與倫比的經驗，而且將 Web 的力量交付到眾人手上。」

Firefox OS 手機另將整合 Firefox Marketplace，以期在行動裝置上完全發揮 HTML5 Apps 的效能。Firefox Marketplace 擁有全球廣受歡迎的 Apps，也提供區域性專屬的 Apps。Firefox Marketplace 上的 Apps 分類包含：遊戲、新聞與媒體、音樂與音訊、商業與生產力等類型，並更進一步提供可因地制宜或依特定需求量身打造的 Apps ─ 隨時滿足使用者的需要。使用者在 Firefox Marketplace 上將找到許多廣受歡迎的 Apps，如AccuWeather、Airbnb、美商藝電 (EA) 的《Poppit》益智遊戲、Facebook、Nokia HERE、MTV Brasil、SoundCloud、SporTV、Terra、Time Out、Twitter[EJ3]  等等，以及依區域特性而打造的本地化 Apps。

Firefox OS 是 Firefox 桌面與行動瀏覽器在行動裝置上的延伸。Firefox 在全球擁有數億名愛好者，使用者將可在 Firefox OS 上獲得與 Firefox 瀏覽器相同的體驗：安全性、隱私性、自訂性，讓使用者擁有掌控權，如同 Firefox 一直以來對使用者的承諾。

在全球獲得超過 20 家硬體廠商與電信營運商的參與，Firefox OS 將逐步完備這個自由、高適應性的行動平台，真正為使用者提供具備創新與自訂性的行動 Web 體驗。全世界都期待著更親民的智慧型手機。根據 Kleiner Perkins Caufield & Byers 風險投資公司的報告指出，全球僅 21% 的行動用戶擁有智慧型手機。

「OEM 廠商、開發人員、電信營運商都逐漸看重真正具替代性的行動平台，以期能跳脫目前主流而進一步拓展新的商機。」IDC 研發副總 John Jackson 接著表示：「這種平台能夠解決系統完整性、開放性、規模性等條件下的各項挑戰。Mozilla 和合作夥伴正引領第一波的 Firefox OS 風潮。很明顯的，不論是具影響力的 OEM 廠商與電信營運商，甚至是全球性與高度地方性的開發者社群，都看到 Firefox OS 成為下個主要平台的潛力。」

Telenor 將在今年率先在中歐和東歐推出搭載 Firefox OS 的手機，Firefox OS 手機將於下半年在更多市場發表。

廠商證言：

Alcatel

「ALCATEL ONE TOUCH 只提供最好的產品，同時在全球不斷擴展合作規模，以滿足消費者的需要。我們提供一系列的智慧型手機，並達到高品質、高效益等特色。另預計在 2013 年的第三與第四季發表更多產品。」ALCATEL ONE TOUCH 的首席行銷長 Dan Dery 指出。

Deutsche Telekom

「我們很高興能夠迅速在波蘭境內推出 Firefox OS 行動裝置。」Deutsche Telekom 的歐洲董事會成員 Claudia Nemat 說：「能在歐洲發表第一款 Firefox OS 智慧型手機，也代表我們在不同生態系統之間，跨出了創新且高競爭力的重要一步。Deutsche Telekom 將透過開放的平台，為客戶提供更多元、更自由的選擇。」

Telefonica

「我們相信智慧型手機市場需要更加的開放，而以 Web 為平台的概念可以使開放成真。消費者不應被單一平台制約，而是應該自主的選擇所需的內容，並在更換裝置時可以將內容無縫接軌。今天起是行動市場頗具轉折性的一週，在 Firefox OS 為消費者打造的機型正式在西班牙亮相的同時，也表示未來我們將在市場上推出越來越多的Firefox OS 手機。」Telefónica 首席營運長 José María Álvarez-Pallete 指出。

中興通訊 (ZTE)

「ZTE 很榮幸能夠加入 Telefonica 與 Mozilla 的合作行列，發表我們第一款 Firefox OS手機 ─ ZTE Open。這支手機具備強悍的功能，可帶給使用者全新的體驗。身為全球前五大智慧型手機製造商之一，ZTE 將為所有消費者提供更多選擇，讓大家都能享受高階科技所打造的美妙生活。」ZTE 副總裁 Mr.Dai Wenhong 表示。

有關美商謀智

Mozilla 身為 Web 的開創先驅已超過10年。我們建立並提倡開放的標準，矢志讓 Web 成為大家所共有的創新平台。目前全球已有數億人在桌機、平板電腦、行動電話上，透過 Firefox 探索、體驗、接觸 Web 的浩瀚無邊。若需進一步了解，可前往http://mozilla.com.tw/。
