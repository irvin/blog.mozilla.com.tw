現已釋出 Windows 適用的 Firefox 64 位元版本

12月172015

Mozilla 現已釋出 Windows 所適用的 Firefox 64 位元版本，要讓 64 位元系統的使用者享受絕佳效能。我們很高興能讓 Windows 7 或更高版本的使用者獲得更優質的遊戲與應用程式經驗。

針對 Windows 所設計的 Firefox 64 位元版本，尚未能完整支援外掛程式。且使用者會發現特定網站所需的外掛程式，即使已能於 Firefox 32 位元版本中運作，卻還無法用於 64 位元版本。如我們之前所提，Mozilla 將於 2016 年底停止支援 Firefox 中的大多數 NPAPI 外掛程式。

若要下載並安裝 Firefox 64 位元版本，請至 Firefox 全系統頁面。

歡迎你提供反饋意見，協助我們持續改善 Firefox 64 位元的支援。
