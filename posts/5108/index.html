強力徵求：將 HTML5 App 移植到 Firefox OS 與 Open Web

3月62014

背景概述

2013 年，Mozilla 與合作夥伴已於 14 個市場中發表了 Firefox OS，其中包含三款 Firefox OS 智慧型手機，還有 Geeksphone 提供的一款開發者預覽手機。相關開發團隊也於全球舉辦了八場僅受邀者參加的 App 開發者工作坊：位於美國山景城 (Mountain View) 的 Mozilla 辦公室、英國的倫敦、西班牙的馬德里、波蘭的華沙、巴西的阿雷格里港、墨西哥的瓜達拉哈拉、匈牙利的布達佩斯等地。今年，Firefox OS 將登上更多款裝置，並現身於全球更多國家之中。Firefox OS 裝置希望能執行更多的 HTML5 App，並讓 Firefox Marketplace 更加豐富、茁壯。

透過之前的「Phones for Apps program」企劃，Mozilla 送出數百支 Geeksphones 給全球的 App 開發者，讓大家將現有 HTML5 App 移植到 Firefox OS，還能自己保留開發者預覽手機。現已有數百位開發者率先將 App 送入 Firefox Marketplace。非常感謝大家！

新企劃：Cordova/PhoneGap 移植作業的手機

Mozilla 今年想找出所有的最佳 App ─ 能為 Firefox OS 使用者提供優質經驗與高度本地化特色的 App。對 HTML5 App 的開發者來說，跨平台工具能輕鬆建構原生平台上的 App，也能更輕鬆存取原生裝置的 API。我們才剛介紹過 Firefox OS 與 Cordova 3.4 之整合，並能在 Firefox OS 平台上發佈 Cordova App。此專案目前仍在運作中。開發者可透過 Apache Cordova 3.4 版本了解重要功能。而上述「Cordova 3.4 整合文」也說明新功能的使用方式，可協助你移植現有的 App。

如果你已經在用 Cordova/PhoneGap 開發過 App，現在正是能快速、輕鬆移植 App 的絕佳機會。Mozilla 已經聽說有開發者才花一個週末，就成功把 PhoneGap App 移植為 Firefox OS App 了。你沒看錯！不用幾個禮拜或幾個月；只要幾個小時。我們也早知道現已有很棒的 PhoneGap App。也因為如此，廣受大家歡迎的「Phones for Apps」方案第三階段，就是專為 App 的「移植者」所設計，期待有更多開發者移植目前高人氣、高評價的 Cordova/PhoneGap App。

注意：可透過此框架將 HTML5 App 封裝為可安裝的格式。Cordova 則是奧多比 (Adobe) 產品 PhoneGap 中的底層軟體。

誰來申請

如果你就是 Cordova/PhoneGap 開發者，並已於任何平台上發佈過 HTML5 App，就請參加 Mozilla 的「Phones for Cordova/PhoneGap Apps」計畫。只要向我們展示你的高評價 App 且確定適合提交至 Firefox Marketplace 之中，Mozilla 就會寄出開發者手機，協助你將 App 提交至 Firefox Marketplace。加入此計畫的方式如下：

  • 先到這裡申請： Phones for Cordova/PhoneGap Ports application
  • 我們在收件之後隨即開始審查。通過審查的申請人均將收到通知。
  • 只要你確認即將移植 App，我們就會寄發裝置給你。

即將展開的 Firefox OS 工作坊與開發活動

除了線上的「Phones for Cordova/PhoneGap Ports」計畫之外，Mozilla 也打算在其他國家舉辦名額有限、僅能受邀參加的 Firefox OS App 工作坊。很高興能搶先告知大家：Mozilla 將於捷克共和國的美麗城市布拉格 (Prague) 舉辦一場工作坊，並預計在下一季發佈確切日期。現已開放工作坊登記表，接受符合資格的 HTML5 開發者前往申請。

必要條件：開發者必須正在開發 Firefox OS App，或有現成的 HTML5 App 正要移植為 Firefox OS App。請提供現有 App 的連結，或開發中 App 的程式碼以資佐證。如果你無法提供撰寫中 App 程式碼的相關連結，恕我們將拒發工作坊參加資格。

立刻申請布拉格的 App 工作坊

如果你住得離布拉格很遠呢？別擔心！Mozilla 仍計畫在今年舉辦多場 App 工作坊。敬請隨時注意本網站的相關訊息。

不論你才剛入門，或是已經為 Firefox OS 開發了許多 App，都能參加 Mozillian 在全世界主辦/協辦的 App Days、駭客松，以及許多活動。了解這些由 Mozilla Reps 主辦，並交由當地社群主持的 Firefox 活動，以及 Mozilla 已參與或即將參與的多項講座與活動。
