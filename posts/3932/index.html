Firefox OS 1.1 新增功能、提升效能、支援更多語言

Mozilla 在發表首款 Firefox OS 智慧型手機之後沒有多久，就提供了更新版本。新版 Firefox OS 新增多項功能、支援新的語言，並大幅提升了整體效能。Firefox OS 新添的功能何其多，我們這裡先概略說明其中數項：

  • 支援多媒體簡訊： 現可透過 MMS (Multimedia Messaging Service) 傳送/接收圖片、音訊、視訊等檔案。
  • Push Notification API： 開發者可利用推撥功能，將即時性的通知推撥至 Apps 中，並有助降低耗電量。
  • 自訂性 App 搜尋功能 現置於主畫面前方的中央區塊，讓使用者能更快找到所需的內容。
  • 強化聯絡人資料管理： 匯入 Gmail 與 Hotmail 聯絡人資料：現在能更輕鬆將聯絡人資料匯入 Firefox OS 手機之中。除了目前的 Facebook 或 SIM 卡匯入選項之外，現更新增 Gmail 或 Hotmail 聯絡人資料的匯入功能。
  • 新增為聯絡資料：現在只要是列於電子郵件、電話撥號記錄、SMS 文字簡訊上的聯絡人，都可輕鬆將其電話號碼或郵件位址加入聯絡人資訊。只要長按該筆資訊，就可直接建立新的聯絡人，或加入現有的聯絡人資訊中。
  • 撥號建議：在撥號畫面中鍵入電話號碼或聯絡人名稱，即可迅速找出特定聯絡人。
  • 提升效能： 加快 Apps 載入速度，且達到更流暢的捲動效果。
  • Firefox 瀏覽器即可下載： 可透過瀏覽器輕鬆儲存圖片、音訊、視訊。
  • 改善鍵盤功能： 可自動修正使用者偶然拼錯的字。
  • 強化電子郵件功能： 草稿模式：可自動儲存撰寫中的電子郵件，以利稍後完稿並寄發電子郵件。
  • 下載音訊與視訊附加檔案：儲存圖片、音訊、視訊等的電子郵件附加檔案，且圖片將儲存到自己的圖庫中。
  • 寄發圖片格式的附加檔案：電子郵件 App 可直接附加圖庫中的圖片並寄發出去。
  • 音樂搜尋： 新的音樂搜尋功能可輕鬆找到自己最愛的音樂。只要在「Music」App 之內向下滑動即可開啟搜尋列，接著能以歌名、專輯名稱，或歌手姓名找到所愛的音樂歌曲。
  • 強化行事曆功能： 直接建立事件：長按所需的時間區段，即可建立新的行事曆事件。
  • 行事曆事件提醒：可設定提醒功能，通知使用者即將發生的行事曆事件。
  • 現已支援 14 種語言。

本次 Firefox OS 更新，將很快提供給目前的使用者，亦將提供給開發者手機使用。若要進一步相關資訊，可參閱 MDN 上的開發者裝置相關說明。
