Mozilla 全球佈局 ，Firefox OS 大步邁入行動市場！

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012 年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。今於 MWC (Mobile World Congress ) 大會正式開始前，Mozilla 搶先預覽第一款以搭載 Firefox OS 的行動裝置，並宣佈與合作夥伴的推出計劃。第一波 Firefox OS 行動裝置將在巴西、哥倫比亞、匈牙利、墨西哥、蒙特內哥羅共和國、波蘭、塞爾維亞、西班牙和委內瑞拉登場，其他地區的推出計劃也將在之後陸續釋出。

目前全球有 17 家電信商承諾將推出Firefox OS 行動裝置，其中包括墨西哥美洲電信公司（América Móvil）、中國聯通（China Unicom）、德國電信（Deutsche Telekom）、Etisalat 阿聯酋電訊、Hutchison Three Group、KDDI、韓國最大電信業者（KT Corp）、俄羅斯電信商（MegaFon）、卡達電信（Qtel）、新加坡電信（SingTel）、Smart、美國電訊商Sprint、義大利電信集團（Telecom Italia Group）、西班牙電信（Telefónica）、挪威電信Telenor、葡萄牙最大的行動網路業者TMN以及俄羅斯第二大行動營運商（Vimpelcom）。Telstra（澳大利亞電訊）也對於 Firefox OS 的到來十分歡迎，並視這項創新可為顧客帶來更多行動 Web 體驗。廣受市場支持的 Firefox OS 顯示出 Mozilla 當初倡導的完全開放、百分百融合的行動平台概念。

Firefox OS 智慧型手機以開放網路標準為主，每一個手機應用功能皆為純 HTML5 技術打造。Web Apps 跨越既有概念中 HTML5 的弱點，並嘗試裝置中的任一個可能，並達到令人驚豔的優秀表現。Firefox OS 平台的高靈活性，可方便開發商量身訂做專屬界面，並加進符合當地的使用者習慣。

Mozilla 目前正與LG、TCL (Alcatel) 以及ZTE 合作生產，第一支採用Firefox OS 系統的手機，華為 Huawei 在下半年也預計有 Firefox OS 系統手機產出，這些機型將採用 Qualcomm Snapdragon™ 的行動處理器。此外，Mozilla 已經擁有許多重要的策略性夥伴，可以共同就內容、服務進行對外傳遞，確保用戶在使用 Firefox OS 中很好的使用者體。使用者將得到前所未有，豐富且開放的智慧型手機體驗，任何使用者最愛的網上行為，包括 Facebook 和 Twitter 的整合，提供簡單使用、界面快速以及經濟實惠優點於一身的絕佳使用體驗。

“Firefox OS 為行動市場的使用者帶來高度自由與無窮創新”，Mozilla CEO Gary Kovacs 指出，”透過活躍的社群支持和合作夥伴們的共同努力，Mozilla 致力打造公平的競爭環境並迎接內容與服務大量充斥的需求，為未來的二十億上網人口的多樣性服務。

作為首發，“第一波”發佈計畫將由墨西哥美洲電信公司（América Móvil）、德國電信（Deutsche Telekom）、西班牙電信（Telefónica）和挪威電信（Telenor）宣佈，目前計畫是在今年年中的時候推出。各合作夥伴的發佈計畫如下：

América Móvil

“América Móvil 將推出 Mozilla 開發的行動作業系統，其絕佳的自訂選擇性對新興市場的消費者來說更是一大利基，加上開放平台以網路標準為準，提供一個開發者合適的電信生態系統來開發各式各樣的應用程式”，América Móvil CMO Marco Quatorze 表示，” América Móvil承諾將於 2013 年在墨西哥和其他可能市場釋出 Firefox OS 手機。”

Deutsche Telekom

“合作是通往創新的重要途徑，我們倚賴開放平台，是為了打造自由選擇，提昇顧客使用權益” Chairman of the Board of Deutsche Telekom 於MWC 大會指出，”因此我們支持 Mozilla 具創新、開放標準的行動開發成果，首款於歐洲推出Firefox 行動作業系統搭載的智慧型手機更是重要的一步，表示與各家電信生態系統正面競爭。

Etisalat

Etisala 的CEO，Ahmad Abdulkarim Julfar表示 “很高興Etisala 參與這項創新專案，並支援 Mozilla 推出搭載全新行動平台的行動裝置，Firefox OS 智慧型手機的釋出不只讓Etisala公司本身多了一項重要的產品，更讓我們的顧客擁有更豐富以及客製化的使用體驗，恭喜 Mozilla並大力歡迎 Firefox OS的到來。”

Hutchison Three Group

“Hutchison Three Group認同 Mozilla 對開發商的開放倡議，同時提供使用者可負擔的入門級智慧型手機”， Deputy Chairman of Hutchison Whampoa Europe的Christian Salbaing表示，“我們相信 Firefox 加入會提升整體行動市場的競爭，增加消費者選擇性，並期盼降低硬體配備的價格，把智慧型手機和更優化的服務帶給更多的顧客。”

KDDI

“我們很高興宣佈 Firefox OS 智慧型手機即將在這一、兩年內進入日本市場”， KDDI 的CEO Takashi Tanaka 表示，“Firefox OS 智慧型手機的推出，相信可以提供更好的使用者體驗和更多樣化的選擇性，這樣的開放標準網路平台將為市場注入創新力，並帶來公平公正的市場競爭。如此的理念與KDDI策略相符，KDDI目前與 Mozilla 密切合作，在 Firefox OS 開發上努力，其中特別針對互聯網和安全功能上做更多著墨。”

KT Corporation

KT Corporation的CEO Dr. Suk-Chae Lee表示，針對目前現有 All-IP 網路的變化，包括語音、數據資料、媒介等所有的資訊和服務都透過 All-IP 網路傳遞，Firefox OS 由 HTML5 打造，讓 All-IP 為基礎的服務變簡單和容易，同時跨裝置和平台。KT Corporation全力支持此創新，不只增進市場成長，更對全球經濟有益。”

MegaFon

“我們看到對整體市場十分正面的力量，特別是這樣一個以開放、跨平台發展，以及具市場支持度的創新概念”， MegaLabs 的COO Vitaly Starodubov指出。

SingTel

Paul O’Sullivan，CEO Group Consumer，SingTel， 提到：“SingTel堅信需要為客戶提供更多選擇，並鼓勵創新，因此我們非常興奮的看到通過HTML5打造的開放行動生態系統所帶來的各種可能。”

TMN

“TMN 一直以來都在找尋市場上最好的可能性，持續專注在未來發展，以提供新概念、高顧客滿意度和無縫整合服務，TTMN 與 Mozilla 的合作和共同推出 Firefox OS， 對葡萄牙的電信市場而言十分重要，特別是提供全新平台、使用者界面和電信生態系統的改變。” Portugal Telecom Group的 CMO Luís Avelar 指出。

Smart

Smart 的Chief Wireless Advisor Orlando B. Vea 說明，“我們十分期待看到 Firefox OS 加入行動市場戰局，讓各裝置和平台連接起來，這樣一個以 HTML5 技術為準的 Firefox OS 將用可負擔的行動裝置把開放電信生態系統帶進來，這與我們要把行動裝置帶給每一位顧客的理念相符。”

Sprint

“Spring 支持開放電信生態系統，提供自主性與創造力給開發者”， 身任senior vice president 、product development 與operations 的Fared Adib　指出，“Firefox OS 提供一個具高自由度、讓開發者開發創意apps、讓顧客十分喜歡的 apps。”

Telefónica

“2013年對Telefónica 來說是一個有全新展望的一年，一個以開放標準為主的一年”， Telefónica的Chairman & CEO　 César Alierta 指出，“我很高興我們的客戶在巴西、哥倫比亞、西班牙和委內瑞拉會是全球第一批體驗 Firefox 的用戶，提供更多的選擇性、自主性與彈性來使用智慧型手機。過去Firefox 在 Web 上激發出無比創新，我相信在行動市場上可以有同樣的影響，期待在2014年底前讓全球的Telefónica 客戶，都能使用到搭載 Firefox OS 的行動裝置。

Telenor

“ Telenor 很高興可以和 Mozilla 以及其他主要營運商成為夥伴，共同開發 Firefox 行動裝置，在這個主張下，我們目標讓更多的客戶擁有更多的選擇性和更好的行動上網體驗，以及連接下一個上億的 Web 用戶”， Telenor CEO Jon Fredrik Baksaas 指出。

VimpelCom

“我們大部分的客戶都位在新興市場，而他們對行動上網的需求，以及OTT應用正在急速成長中”， Jo Lunder, CEO of Vimpelcom指出，“在Vimpelcom 我們致力在 OTT 和核心行動服務間的無縫整合，我們支持開發的開放標準，因此我們一定對 Firefox OS 絕對認同，Firefox OS 是經濟實惠的智慧型手機系統中的另一選擇。

Qualcomm

“Qualcomm 一直與 Mozilla 緊密的合作，以開發和優化 Firefox OS與Snapdragon行動處理器的效能，Cristiano Amon, Executive Vice President, Qualcomm Technologies, Inc. and Co-President, Qualcomm Mobile and Computing 指出，“在軟體與硬體間如此密切的合作，讓我們可以增強並豐富 HTML5 的 Web 體驗，並進而達到從而未有的境界。”

Research Analyst

” Firefox OS 目前已得到令人非常驚豔和優秀的營運陣容，同時獲得高度信任和經銷承諾，在在指出 Firefox OS 的無窮潛力，同樣的，2013 年在行動市場上會是具突破性的一年。John Jackson, Research Vice President for IDC 指出。具有不同的營運商支援，從新興市場到已開發市場，對平台的信任而言是很強力的背書，同時也能吸引第三方前來開發新創意。目前，就營運商的承諾和手機製造商以及內容提供商的進展來看，Mozilla 指出其能解決兩個關鍵市場挑戰，一個是成功經銷，一個提供第三方營收方式，就兩者能吸引到開發者和得到他們的支持。

全新 Firefox OS 品牌

Mozilla 在 MWC 上同時展示全新Firefox OS 品牌視覺。Firefox OS 品牌定義以 “解放 Firefox” 做為出發點，將 Firefox 的品牌價值視覺化：Firefox OS 趣味與友善；支持並保護；快速且強大。
