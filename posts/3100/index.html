Firefox OS 裝置正式推出！

7月162013

大約 2 年前，我們才在 Mozilla Hacks 部落格上宣佈啟動Boot to Gecko (B2G) 專案的消息，而當時還在談論 B2G 專案的目標與未來規劃。今天，所有相關工作都已開花結果，Firefox OS 裝置也在商店中正式推出了！

上個禮拜，由 Telefónica 在西班牙的首都馬德里開始銷售第一支 Firefox OS 手機。這些手機儼然成為 Open Web 技術所建構低成本手機的最佳範例。

如果你還沒開始使用 HTML5 與適應性設計 (Responsive Design) 建構自己的 Apps 或網頁內容，請把握現在這個能讓你在此全新市場坐擁一席之地的最佳時機。

將在更多國家現身

西班牙 Telefonica 營運商只是 Firefox OS 的第一站，接著波蘭、哥倫比亞、委內瑞拉等國也將開始銷售，且後續會在更多國家現身。

Open Web Apps

Firefox OS 可直接執行 Web 內容，且本身也內建瀏覽器功能。但如果你想的是要提供類似 App 操作的使用經驗、利用更多的 APIs、硬體、特色效能，那打造一個 Open Web App 就是你的最佳選擇。

再次拿出你現有的 HTML5 技巧與網站內容，將它們封裝起來吧。下列幾篇文章曾提及 Open Web Apps 與其諸多可能性：

  • Getting started with Open Web Apps – why and how
  • Introducing WebAPI
  • Introducing Web Activities

Firefox OS Simulator

工欲善其事，必先利其器。Firefox OS Simulator 絕對是快速測試 Apps 的最佳首選。

我們已準備發表第四版 Firefox OS Simulator，且很快就會提供更多細節。先列出幾項要點：

  • 新的連線按鈕 ─ 可讓開發者工具連上你指定的 App(s)
  • 遠端樣式編輯器 (Remote Style Editor) ─ 可於遠端編輯樣式
  • 隱藏版功能：Shift + Ctrl/Cmd-R ─ 可重新整理 App 並清除永久性儲存資料 (AppCache、localStorage、sessionStorage、IndexedDB)
  • 更新 Dashboard 的設計 ─ 更簡潔的介面提高了實用性
  • 觸控事件 (Touch Events) ─ 整合 Gaia 的觸控事件模擬功能。現在只要用滑鼠與 Simulator 互動，同樣會產生觸控事件

導入付款機制

Firefox Marketplace 同樣屬於 Firefox OS 的一環，可供開發者上傳並分享自己的 Apps。Mozilla 現在開心的宣布，如果你想透過自己的 App 獲得營收，現在就加入Marketplace 吧！

針對 App 開發者，我們在開發者交流中心 (Developer Hub) 的 Payments 頁面中提供 App 收費說明，另曾透過《在 Firefox OS 上建立付費 App》一文提供相關資訊。

目前 Firefox Marketplace 共有 2 種付費情形：

  • 購買 App 本身
  • In-app payments (應用程式內部付費機制)

Firefox OS 堂堂登場

總而言之，現在 Firefox OS 裝置已正式上市。我們希望這些功能可為 Open Web 開發者帶來更優質的內容與經驗，同時能獲得正面回應。Mozilla 會努力做到最好，為大家提供更多樣的選擇與可能性。
