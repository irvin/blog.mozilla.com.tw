Firefox OS 的亞洲開發者現可自 eBay 訂購 ZTE Open

10月152013

【美商謀智訊息稿】Mozilla 是全球性的非營利組織，以推動網路的平等、自由、開放為宗旨，旗下的 Firefox 瀏覽器、Firefox for Android 行動瀏覽器、Firefox OS 開放行動作業系統廣受全球市場喜愛。Firefox OS 手機 ZTE Open 自今年 8 月中開始在 eBay 上販售，立即受到高度注目，現貨於一周內全數銷售一空。為服務電信營運商未能觸及到的地區之開發者與使用者，ZTE 將擴大 Firefox OS 手機販售的運送範圍至更多國家，包括日本、南韓、印尼、台灣、新加坡、澳門、馬來西亞、菲律賓、泰國、越南、寮國、汶萊、澳洲和紐西蘭。

上述運送範圍國家的開發者將可在 eBay 上購買到英文版本的 Firefox OS 手機，網站售價 79.99 USD (約台幣 NTD 2,364，不含國際運費與關稅)。ZTE Open 支援三頻 3G 網路：HSDPA 850/1900/2100 (美國與加拿大: 850/1900、亞洲與澳洲: 850/1900/2100、歐洲: 900/2100)。開發者應於 eBay 購買時，依所在國家選擇適合的頻段 。

這次於亞洲地區販售的 Firefox OS 手機新增 fastboot (快速開機) 功能，可讓開發者依據自身需求重建手上的 Firefox OS 手機。現有 ZTE Open 使用者也可透過 ZTE 提供的 SD 卡升級包來啓動 fastboot 功能。如需更多相關資料，請上 MDN (Mozilla Developer Network) 參考相關文章：

https://developer.mozilla.org/en-US/docs/Mozilla/Firefox_OS/Developer_phone_guide/ZTE_OPEN
