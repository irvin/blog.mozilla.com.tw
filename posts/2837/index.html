軟硬結合 再下一城 鴻海攜手 Mozilla 合作Firefox OS開放平台

6月32013

(2013/06/03 台北訊) 鴻海集團積極進行硬體與系統整合，今日宣佈與 Mozilla 簽署合作備忘錄，雙方將進行策略聯盟，攜手合作開發搭載以 HTML5 與開放網路技術為核心的 Firefox OS 裝置。鴻海自去年開始便以提供軟硬體結合的裝置為首要目標，如今與 Mozilla 攜手合作 Firefox OS 裝置，再度佈局「軟硬實力結合」，觸角延伸開放式作業平臺，積極實現硬體＋軟體＋內容＋服務的整合計畫。

“鴻海目前全力朝『 8屏1網1雲 』的產品佈局發展，除原本鴻海就擅長的硬體製造技術優勢之外，力求開發更多提供『 硬體＋軟體＋內容＋服務 』的整合型全新載體”，鴻海創新數位系統事業群總經理劉揚偉表示：”很高興此次鴻海與 Mozilla 因願景相同而進行合作。Mozilla 所開發的 Firefox OS 以 HTML5 與開放網路技術為主要架構，與鴻海目前推動的軟硬結合策略不謀而合，相信藉由彼此合作所開發的軟硬體整合之全新載體，將可讓更多客戶與終端消費者受益。”

“非常高興能與鴻海合作，也期待有更多合作夥伴加入支持 Firefox OS 與開放網路的行列。” Mozilla 的全球行動裝置資深副總裁，兼任亞洲區營運總裁及台灣分公司執行長宮力博士表示：” 在 Mozilla 我們認為 Web 才是行動裝置最大最好的平台，打造 Firefox OS 的目的也在於強化行動市場的網路標準，讓系統整合更開放，不同裝置彼此之間可以自由溝通。而這次與鴻海的合作，也展現 Firefox OS 做為開放網路行動作業系統，在支援各式行動裝置上具備無限的潛力。”

Firefox OS

Mozilla 的 Open Web 行動作業系統 Firefox OS，以 HTML5 等網路技術打造。Firefox OS 使用 Linux 核心，開機之後進入以 Gecko 為基礎的執行引擎，而使用者所執行的應用程式 (App) 亦完全透過 HTML5、JavaScript 及其他開放 WebAPIs (Open Web Application APIs) 所打造。Firefox OS 搭配 Firefox Marketplace 形成一套完整的行動 App 生態體系，植基於開放網路標準，讓行動市場的手機業者、開發者和使用者，無需再受制於封閉且互不相容的行動作業系統。Firefox OS 目前在全球共獲得 18 家手機廠商、電信業者公開表示支持。全球首支 Firefox OS 手機預計於 2013 年夏天問世。市場高度期待的 Firefox OS 基於 Firefox 瀏覽器，提供與 Firefox 完全相同的體驗，其安全、隱私、自訂性的特性深受喜愛並獲得全球肯定。

關於 Foxconn (鴻海科技集團)

鴻海科技集團成立於1974年，在董事長郭台銘先生的領導下，提供全球最具競爭力的生產製造技術，以前瞻性的眼光自創具備機光電垂直整合、一次購足整體解決方案優勢的3C代工服務『eCMMS』商業模式。鴻海科技集團近年逐漸轉型以科技服務為導向，提供客戶最完整的解決方案。鴻海科技集團為客戶創造最大利益之餘，並致力於提升環保概念於製造過程中，以作為客戶值得信賴的夥伴及全球企業的楷模為目標。

關於Mozilla (美商謀智)

Mozilla 長期致力於打造更美好的網路環境，十多年來推動並參與制定許多網路標準，其願景是打造一個更開放自由的網路未來，提升人們的網路經驗。Mozilla 主要產品 Firefox、Firefox for Android, Firefox OS。其中 Firefox 瀏覽器為全球市場前三大瀏覽器，在世界各地擁有廣大支持者，提供80多種語言免費下載。Firefox for Android 在 Google Play 上榮獲使用者 4.5 顆星的評價。Firefox OS 為全新概念的開放網路行動作業系統，廣受市場高度期待。Mozilla Taiwan 美商謀智是 Mozilla 在全球的第九個據點，主力研發 Firefox OS、並進行 Firefox OS 與 Firefox 瀏覽器的相關合作推廣。
