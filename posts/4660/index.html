給開發者的 Firefox OS ─ 共同加入完全的HTML 5 平台

1月22014

Mozilla 此一系列短片將說明 Firefox OS 對開發人員所代表的意義，另外也讓大家了解應如何加入這次世界性的革命。

Mozilla 一直在許多會談中接收到類似的訊息：開發人員將 Firefox OS 視為繼 iOS 與 Android 之後，第三個加入行動領域的平台。但這完全不是 Mozilla 所預設的目標。Firefox OS 自己即擁有獨特的本質，而且也不打算再加入已飽和的高階市場。

反之，Firefox OS 將瞄準高階智慧型手機以外的市場，或未使用封閉式 App 機制的市場，進而取代一般的功能性手機。此外，Mozilla 也將打造第一個完全開放、以 Web 標準為架構的生態系統與行動作業系統。對 Web 而言，Firefox OS 並沒有任何封閉或不相容的要素。

因為如此，Mozilla 錄製一系列的影片，期待以簡單易懂的方式，引領各位成為 Firefox OS 的開發者，亦將解釋應如何擺脫其他平台的限制，進一步撰寫 HTML 5 而建構 Web 標準的 App。

本篇短片是由 Mozilla 的 Chris Heilmann (@codepo8)，與 Téléfonica Digital/ W3C 的 Daniel Appelquist (@torgo) 所錄製，其中將說明 Firefox OS 的目標。亦可到此觀看影片 (中文字幕)。

影片中所提及的資源如下：

  • 成功打造 App 的關鍵
  • Firefox OS 開發者交流中心，內含 App 規劃、基礎要件、App Demo 的相關資訊，可供你建構自己的 App。另有將 App 提交至 Marketplace 的說明文件
  • Firefox OS 模擬器 (Firefox OS Simulator)

Mozilla 希望你會喜歡這些影片，並讓影片解答你對 Firefox OS 的某些疑問：

  • 搜尋 App： 是如何能輕鬆發掘，或推廣自己的 Firefox OS App
  • 著手開發 HTML5 App： 該如何開發自己的 App
  • Firefox Marketplace 與其他選擇： 該如何在 Firefox Marketplace 中陳列並推銷自己的 App
  • Web-API： Web-API 的必要性
  • Web Activities： Web Activities 的相關概念
  • Web Components 與 Mozilla Brick 函式庫： 正夯的 Web Components，還有 Mozilla 為開發者準備的 Brick 函式庫
