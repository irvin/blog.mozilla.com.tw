Firefox Beta 全新強化隱私瀏覽模式，可獨立開啓隱私瀏覽視窗！

2月232013

可供 Windows、Mac 和 Linux 使用者下載和測試的全新 Firefox Beta 版本現已開放測試。新版 Firefox Beta 強化原本的隱私瀏覽模式，使私密瀏覽更方便。同時，新版本更提供嶄新的下載體驗，以及多樣開發者最愛的開發者工具。

  • 可獨立開啓新視窗的隱私瀏覽模式：Firefox 現在讓使用者可同時在兩個 Firefox 視窗下，使用隱私瀏覽模式與正常模式。使用者只需將滑鼠移到任何連結，右鍵點選並選擇開啓連結即可輕鬆使用。Firefox for Android Beta 版也提供隱私瀏覽模式，請點選此處。
  • 更新 Firefox 下載管理：Firefox 讓您的檔案下載體驗再進化！使用者可在工具列直接掌握下載進度，不需再倚靠下載視窗。

  • 外掛程式提醒：當外掛程式開啓時間超過 45 秒，Firefox 將讓您選擇是否要重新開啓外掛程式，而非重新啓動瀏覽器本身，讓您的瀏覽體驗不受干擾並可持續使用。
  • getUserMedia：全新 HTML5 DOM API 讓瀏覽器可直接捕捉到照相機以及/或麥克風裝置，此工具讓 JavaScript 開發者方便撰寫可與照相機和麥克風裝置連結的程式碼。gUM 是 WebRTC 功能下的第一個 API，是一個提供零時差、互動性、兩方語音和影音電話互通，以及檔案同時分享的全新 API。
  • CSS Flexbox：新版 Firefox Beta 讓開發者針對不同尺寸的螢幕和行動裝置，可以自動調整網頁的內容顯示。這個功能幫助開發者調整螢幕顯示方式，將重要的網頁內容置頂，或減少網頁上的內容顯示列，以因應使用者在較小尺寸的行動裝置或平板瀏覽時，一樣擁有良好的閱覽體驗。
  • 開發者專屬工具列：全新Firefox Beta 加進開發者專屬工具列，提供立即且最實用的工具，如將 Web Console、Debugger、Inspector以及 Style Editor 放置於同一視窗中。
  • 圖層重疊模式（Canvas Blend Modes）: Canvas Blend Modes 讓開發者可將不同圖素加在既有圖素上，進而達到圖層重疊的效果，比如透過像素的調整，創造出不同的視覺效果。
