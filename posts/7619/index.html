Trainspotting 系列文章：Firefox 39

7月12015

附註：由於發佈前最後一刻發現穩定性的問題，我們因此推延 Firefox 39 直至本週稍晚釋出，另將透過本篇部落格隨時更新釋出消息。

〈Trainspotting〉系列文章將重點提示 Firefox 最新版本的已上線功能。Firefox 現固定每六個星期釋出新版本，而我們稱此種模式為「Release trains」。

新版本 Firefox 已經準備完畢，且針對 Web 平台與開發者工具再度有所提升。本文將提示其中幾個亮點。

若要完整了解相關變化與元件，可參閱《Firefox 39 版本說明》。

對開發者工具的愛

我們透過 UserVoice 聆聽開發者的意見，並根據大家的反饋意見打造出更強大、更簡單易用的 Firefox 開發者工具。其中一項功能就是要重新排序檢測器 (Inspector) 中的元素：

CSS Animations 的編輯與調整作業都更簡單了。Firefox 39 讓開發者不需切換 App 亦可暫停、放慢、預覽、重新啟動新的動畫時間軸。

  • 進一步了解 Firefox 39 開發者工具

CSS Scroll Snap Points

CSS Scroll Snap Points 可讓 Web 開發者在瀏覽器中建構捲動軸的定點元素，並以更精簡的程式碼建立更順暢、更能輕鬆互動的介面。

  • 觀看即時展示 (Firefox 39 與 IE 10 或更高版本均可運作)
  • 進一步了解 CSS Scroll Snap Points

提升 Mac OS X 上的 Firefox

Firefox 39 現針對 Mac 另提供多項改善與更新：

  • 已啟用「 絲綢 」專案 – 透過硬體的 VSync 達到更緊密的繪圖功能，以強化捲動與動畫效果。了解「絲綢」專案。
  • Unicode 8.0 skin tone emoji – 已針對 emoji 修正卡通人物膚色修改器的繪圖錯誤。
  • 虛線效果 – 大幅提升虛線 (Dotted-) 與點虛線 (Dashed line) 的繪圖效果。進一步了解修正的錯誤。

Service Workers 的進度

Firefox 仍持續實作 Service Workers API – 現已為 Workers 啟用 fetch，並擴及一般 Web 內容均可使用。且現在完成設定之後即可使用 Cache 與 CacheStorage。

Firefox 39 還有更多變化與提升之處，可參閱開發者版本說明以了解開發者導向所進行的改變，或可參閱此版本完整的錯誤修正清單。
