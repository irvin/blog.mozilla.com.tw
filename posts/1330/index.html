燒～Firefox OS 將在本週六 (10/27) 首度前往熱情的南台灣 – 高雄，與大家相見歡！第一站將前往的地點是高雄知名的百人級資訊研討會 – MOPCON (Mobile Open Platform Conference) ，由開發成員之一的工程師 Cervantes  與大家現場分享與交流。歡迎大家透過線上報名或現場報名參加盛會，詳情請參考: http://registrano.com/events/mopcon2012 與講程: http://mopcon.org/session.php

演講時間：10/27 (六) 上午 11:00 – 11:50

演講地點：高雄軟體園區 A 棟，高雄市前鎮區復興四路12 號頂樓海景會議廳

主題：以 HTML 5 打造的新世代移動作業系統 – Firefox OS

議題大綱：

Firefox OS 是 Mozilla 基金會針對移動平台所打造的作業系統，其特色是基於自由、開放、標準的技術，並以 HTML 5 作為 App 開發的平台。本 Talk 將介紹 Firefox OS 中 Web App 以及所使用到的 Web API 是怎麼實作出來的，內容包括：

  • Firefox OS 簡介
  • Firefox OS 架構 – Gaia，Gecko，Gonk 層介紹
  • Device APIs
  • Web API 實做範例：Telephony/RIL
  • 如何參與開發、測試、搶先試用
