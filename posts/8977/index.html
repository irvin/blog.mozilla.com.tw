Mozilla 釋出新版 Firefox for iOS 再度提升使用者經驗

7月272016

我們在去年底發表 Firefox 的 iOS 版本  (Firefox for iOS) 時，就獲得熱烈回應並達到數百萬下載次數。許多 Firefox 使用者特別高興能在 iPhone 或 iPad 上又見到自己最愛的瀏覽器。今天我們要再釋出 Firefox for iOS 多項重大改善功能，讓使用者能享受更快速、更靈活、更多選擇等特色；這同時也是 Mozilla 關心的重點。

更快速的 Firefox

現正值暑假工讀的時間，而我們的 Firefox for iOS 實習工程師泰勒．拉克羅伊 (Tyler Lacroix) 在知道了他要負責讓 Firefox 的速度更快之後，隨即延後了自己在這個月內的所有行程而全心投入。最新版 Firefox 經測試之後，他成功降低最多 40% 的 CPU 資源以及最多 30% 的記憶體用量。這樣的測試結果，意味著使用者能更快瀏覽頁面並省下更多電量。當然，所有裝置與其使用情況不一，最後結果亦將有所差異。不論如何，Mozilla 很高興能向大家宣佈 Firefox for iOS 的新優勢！

現在可輕鬆新增任何網頁專屬的搜尋引擎

Mozilla 現已將多款高人氣的搜尋引擎加入 Firefox for iOS 中，但使用者也可能想直接從網址列搜尋其他網站吧？倘若消費者想為自己的單車換一對酷炫手把，就可以添加如 eBay 的拍賣網站。要成為冷知識萬事通？輕點一下就可進入維基百科。只要網站本身具備搜尋框，用戶再輕點 Firefox 本身的「放大鏡」圖示，就能將該搜尋功能加入自己的搜尋引擎列表中。

####

新選單更簡化瀏覽作業

許多使用者都想將自己的桌機＼筆電瀏覽經驗無縫接軌到 iOS 版瀏覽器之上，這其實有點困難。但 Firefox for iOS 現於工具列上提供全新選單，能達到更輕鬆的瀏覽並更快啟動常用功能；從新增書籤到尋找頁面中的特定文字皆然。

####

恢復已關閉的分頁並迅速滑看現正開啟的分頁

行動版瀏覽器的分頁一般較難用。不易閱讀、不易整理、不易瀏覽，而且可能關閉就再也找不回來了。但新升級的 Firefox for iOS 現可讓使用者輕鬆恢復所有已關閉的分頁，也可滑看正開啟中的分頁。

輕點一下就能回到首頁

幾乎所有人都有某個每天搶先瀏覽並常常回來查看新消息的「首頁」。在 Mozilla 擴充可自訂 Firefox for iOS 經驗的新功能之後，使用者已可將最愛網站設定為自己的首頁。只要點擊「設為首頁」圖示，即可開啟此一特定網站，並能在數秒鐘之內就前往自己最愛的網站。

Mozilla 重視並隨時聆聽使用者的意見，因此持續為 Firefox for iOS 添增新功能，同時期待能聽到更多對於新版本的建議。若想體驗本次全新功能，請到 App Store 下載 Firefox for iOS 並不吝提出你的想法。

不論是 iOS、Android、Windows，甚或是 Linux，我們都為更健全、更開放的 Web 而打造更好的 Firefox。但若沒有跨平台的數億名愛用者，以及活躍的社群成員，也就沒有 Mozilla 今日的成就。謝謝大家！
