Firefox OS 讓你盡情享受每一刻！

地圖不可用

活動時間

日期 2013/10/01 - 2013/11/07

地點

Firefox OS 為你的智慧型手機帶來絕妙瀏覽經驗！

Firefox 將使用者擺在第一位，提供全新的選擇與創新，進而從此改變了瀏覽器的發展。如今 Firefox OS 將再次掀起浪潮，為你的智慧型手機帶來同樣絕妙瀏覽經驗。讓你能完全掌握自己的線上生活，隨時隨地都能體現最佳的 Web 感受。

活動介紹：

不論友情、創意、享樂、旅遊、交流、探索……各種時刻，Firefox OS 專為你的生活所打造，讓你更融入周遭世界並盡情享受每一刻。

活動期間只要上傳自己生活中美好一刻的照片至活動網站與 Firefox OS 相框結合，在投票期間獲得最高人氣的照片，上傳者就有機會獲得 Firefox OS 行動裝置等各項大獎！

活動時程：

本活動 10/1 開跑，除了上傳照片者有機會得到好禮，投票者一樣可以參與抽獎！趕快拍下你生活中美好一刻的照片，這將是你大展身手的時刻！
