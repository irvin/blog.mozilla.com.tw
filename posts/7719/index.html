給微軟執行長的一封公開信：請別在選擇與控制權上走回頭路

8月32015

致微軟執行長 Satya，

我因為 Windows 10 讓人不快的一個地方寫信給您：其更新作業就是要讓貴公司的使用者摒棄他們所慣用的網路經驗，並還要以微軟設定好的經驗取而代之。

Windows 10 的升級作業，卻是針對使用者的網路瀏覽器及其他 App，主動覆寫其現有的偏好設定。在我們首度發現此情形之後，立刻就聯絡了貴公司的相關團隊討論了此一問題。很遺憾的，即便我們發信提出問題也未能看到任何後續的實質改變。

還好，我們知道單就技術層面仍可保留使用者先前的設定與預設值，但是整個升級經驗以及預設設定均已變更過，因此保留既有設定頗有難度。滑鼠點擊次數就超過之前的一倍有餘，而使用者還必須捲動看過整個設定內容。此外，某些技術上的謬誤還逼得使用者必須重新設定自己早在前一版 Windows 完成的選項。這真的很讓人困惑、難以瀏覽，還容易迷失在整個流程中。

Mozilla 志在為所有人提供選擇權、控制權，以及諸多機會。基於此一理由，我們打造了 Firefox 與其他產品、設立 Mozilla 非營利組織，並努力讓大家所享受的網路經驗能遠遠超出產品本身所代表的價值。

我們有時能看到消費性產品尊重個人選擇的權利；這代表 Mozilla 念茲在茲的理念受到肯定。但隨著 Windows 10 的上市，我們相當失望的發現微軟又大大向後退了一步。

因為是我們打造出 Firefox 的組織，所以無法妥協於這種讓人坐立難安的變化。這個變化之所以讓人坐立難安，在於數百萬名 Windows 愛用者的選擇遭到忽視。只要有人做出微軟不喜愛的其他選擇，就必須忍受這刻意提高的複雜程序。

Mozilla 強烈建議您應重新思考業務策略，透過更簡化、更清晰、更直覺的流程，讓使用者能在升級作業中保留既有的設定，代表您再次尊重使用者線上經驗的選擇權與控制權。使用者應能在預設設定 API 與使用者介面中輕鬆找到新的偏好選擇，而並非只看到微軟的其他產品。

請讓您的使用者能在 Windows 10 獲得應有的選擇權與控制權。

Mozilla 執行長 Chris Beard
