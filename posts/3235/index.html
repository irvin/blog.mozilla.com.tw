Mozilla 近期動態與每季度發表的 Firefox OS 版本

Mozilla 與全球社群合作不斷推陳出新。如今我們已在不同的平台發表瀏覽器，包含：

  • Firefox ─ 在正式桌面版釋出之前，Firefox 另分為每夜更新版 (Nightly)、曙光版 (Aurora)、測試版 (Beta) 共 3 個釋出頻道，並再分為Mac OS X、Linux、Windows 系統相容版本 (近期內即將推出新改版)
  • Firefox for Android ─ 相容於數千款行動電話與平板電腦，同樣分為 4 個釋出頻道 (每夜更新版、曙光版、測試版、正式版)
  • Firefox 延長支援版 (ESR) ─ 此版本更新頻率將視情況而有所不同，主要適用於企業客戶

上述產品均共用 Gecko 開發引擎，而且幾乎每 6 個星期就會統一發佈給所有使用者。Mozilla 的技術與組織就如同練唱有素的合唱團，隨時根據版本發佈週期而調整為相同旋律。一致的律動讓我們能跨越整個 Web 貫徹相同理念、定期將新功能 (亦有數次跨平台的功能) 提供給使用者，且任何產品均即時擁有最新的安全功能。

你應該也早已耳聞我們的最新產品 ─ Firefox OS。坦白說，這個專案是 Mozilla 對網路的一大承諾，獲得社群與夥伴的大力支持，Mozilla 終於順利推出 v1.0 版本。

不同於 Firefox 的桌面版與行動版，Firefox OS 除了同樣讓使用者擁有絕佳體驗之外，我們還必須兼顧其他方面，如調整產品時程以滿足所有合作 OEM 廠商、電信營運商、晶片製造商的需求。這些新變數，讓 Mozilla 釋出 Firefox OS 的前 2 個版本均以 Gecko 18 為主。雖然沒有按照版本更新的常規，但我們堅信自己是朝正確的方向前進。

在 v1.0 誕生之後，Mozilla 仍不斷尋找更多合作夥伴，並盡力要讓 Firefox OS 回歸版本更新的正軌。針對前 2 個主要版本，我們同樣會每 3 個月就發表新功能，而且每 6 個星期就會釋出安全性更新。就我所知，Firefox OS 擁有目前行動裝置 OS 中最高的更新頻率 (往後仍會做出些微調整)。

除了多款瀏覽器產品在釋出週期上同步之外，現在 Firefox OS 也加入此更新週期；這一切都出乎我們的意料。同樣身為 Mozilla 一份子的你，和我們一起加油吧！
