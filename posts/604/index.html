2015 Mozilla 校園徵才博覽會 等你來相會！

以捍衛 Web 自由與開放為使命的 Mozilla，即將於三月份展開一系列的徵才招募活動！

2014 年 Mozilla 發表了眾多 Firefox OS 生態系統的合作夥伴與新款裝置，今年延續著此番銳不可擋的趨勢，我們極力邀請更多台灣的年輕菁英加入 Firefox OS 的研發陣容，與我們攜手合作開發搭載以 HTML5 與開放網路技術為核心的 Firefox OS 裝置！
