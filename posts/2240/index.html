Mozilla 基金會執行長 Mitchell Baker：慶祝 15 年的美好 Web

Mozilla 在 2013 年 3 月 31 日滿 15 歲了。不得不說這幾年的變化甚鉅：Web 已經融入數十億人的日常生活，而且也讓我們活得更充實、更開心。而且，Mozilla 扮演了相當重要的角色。

回頭再看，Mozilla 的計畫也大幅影響了 Web 本身：透過開放的源碼與社群而建立卓越的軟體，同時設立網際網路關鍵技術的開放性。這些都是商業性機構不會做，也做不到的事情。此時就需要像 Mozilla 這種非營利性、社群導向的組織，站出來為大眾解決相關難題。

Mozilla 的第一階段，就是在桌面上為 Web 帶入抉擇、競爭、自主授權。我們透過 Firefox，讓數百萬人體驗更優質的網路。在此同時，我們也透過 Firefox 而橫跨完整的 Web 生態，提供更完整且開放的機會 ─ 開放源碼、開放式標準、開放的開放程序；可讓所有人都能參與 Firefox 架構、可隨時微調以因應當地的社會環境、可隨時客製擴充以滿足個人需求。全部都隨開發人員自己的喜好，而不需再徵求 Mozilla 或其他任何人的同意。

不容諱言的，網際網路的商業巨人微軟 (Microsoft)，建構了封閉且環環相扣的軟體、硬體、相關服務。為了達到上述目標，我們必須勇於打破此一窠臼。

除了這 10 多年來為 Web 解放的創造力、創新思緒、競爭樂趣，Mozilla 更受到大家的協助，將 Web 重心反轉到更開放的地方，讓更多人能按照「自己的」條件，悠遊於無邊無際的 Web 之中。這種開放的思潮連帶影響了其他活動，從開放的解密資訊、開放的政府政策，直到更開放的科學領域。但更重要的是，數十億人可每天擁有開放的 Web，同時根據自己的目標與夢想，再自行建立、串連、發現許多資源；而且完全不需取得少數商業組織的許可。

與 15 年前相較，Web 接下來的機會與威脅只能說是有增無減。隨著資訊爆炸成長且裝置的功能日新月異，網際網路必將成為人們生活的重心。針對網際網路的運作方式與使用經驗而言，最基本的就是個人控制需求。Mozilla 可以，也必須再次扮演關鍵角色。我們擁有必要的宏大視野、成熟產品、高階技術。我們也知道應如何讓大家加入特定活動，提出自己的見解與想法，進而參與這個不斷豐富且壯大的 Web。

目前是 Mozilla 與 Web 的關鍵時刻。近幾年內又會有 20 億人口加入網際網路這個超大社群。當然，這些人都要能享受 Web 最完整的開放度與授權性。而瀏覽器軟體就是其中的必要元素；當然，我們要做的還不僅如此。

「不僅如此」的另一個部分就是 Firefox OS ─ 全新的行動裝置系統已具備自由度與開放性。同樣可讓個人按照自己的需求而建立 Web 環境、跨系統生態而促成新的良性競爭、建構出具當地精神的內容或想法。

我們現在也正在架構 Mozilla Webmaker 專案。Webmaker 將提供相關工具與技巧，協助大家從單純的消費者，轉而能共同創造美好的線上經驗。另針對身懷熱忱的人們，Webmaker 亦可作為教學平台，協助更多人接觸 Web 世界的力量。最後，我們仍繼續支援所有地區的社群，協助他們成長茁壯。

請大家一同分享 Mozilla 這些日子以來的成就與喜悅，同時也讓我們一起發想未來還能完成哪些創舉。Mozilla 願與大家攜手打造未來，建構一個理想的 Web 世界藍圖。

註：Mitchell Baker 身為 Mozilla 發展的核心人物，同時也是基金會的蜥蜴鬥士 (Chief Lizard Wrangler)，致力不懈於推動網路開放源碼的發展與進步。
