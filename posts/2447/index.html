Geeksphone 從 4 月 23 日起開始販售 Firefox OS 開發者預覽手機！

4月232013

Mozilla、Geeksphone、Telefonica 一同設計了開發者預覽手機。我們也很開心的跟著宣布 4 月 23 日起手機開賣囉！開發人員當然該儘早接觸硬體，即可於實際環境中搭配行動網路，進而測試 Firefox OS 的功能，並確切了解硬體的特性參數 (如相機與加速規等功能)；這些都是 Firefox OS 模擬器所無法比擬的。此外，新硬體更好玩！

Firefox OS 開發者預覽手機 (Developer Preview) 處於未鎖定狀態，可隨時隨地因應開發人員的測試需求，且亦可定期更新為 Firefox OS 的最新版本，讓開發人員能隨時探索 Web 的潛能，為全球數十億行動使用者提供 Web 的絕佳體驗。

Geeksphone 目前販售 Keon 與 Peak 共 2 款 Firefox OS 開發者預覽手機，並同時提供線上販售與全球出貨服務。與電信商將於今年鋪貨給消費者的手機規格相較，Keon 擁有幾乎相同的硬體規格。而 Peak 則屬於較高端的裝置，適合開發人員用以測試 App，以準備未來於 Marketplace 上販售。

Keon 擁有如 Firefox 一樣的橘色機身，搭載 3.5 吋螢幕、Qualcomm 1 Ghz 處理器、512 MB RAM。含稅售價為 91 歐元，並已包含物流費用

Peak 則搭載較大的 4.3 吋螢幕與雙核心Qualcomm 1.2 Ghz 處理器。含稅售價為 149 歐元，並已包含物流費用。

你可直接至 Geeksphone 的 http://shop.geeksphone.com/es/ 網頁下單。而 Geeksphone 另將直接提供支援與更新服務。當然，你也可以瀏覽 Mozilla Developer Network 進一步了解 Firefox OS 的相關資訊；另有 Mozilla Developer Hub 提供 App 開發資訊，亦可供開發人員提交自己 Web App。

由開發者所建立並發佈的有趣 Web App，已經放上了開發者預覽手機。你現在可透過手機搶先體驗。我們希望此 2 款開發者預覽手機能幫大家發想更多點子。相關範例請至 Marketplace [https://marketplace.firefox.com] 看看獨立開發人員為解決問題所開發的 App，進而汲取更多靈感：

  • 了解非洲肯亞的選舉消息
  • 掃描並建立 QR 碼
  • 監控空氣污染狀態

或者可以玩玩這些獨立遊戲，或挑戰自己來寫個遊戲：

  • 沙漠救援行動
  • 剪不斷理還亂

只要稍微修改自己的現有網頁，你就可以建立 Firefox OS 的 App。就像這位薩拉德加思 (Alexander Saladrigas) 上禮拜說的：「很開心能看到 Firefox OS 真正出現在硬體上了，而且還很貼近最終會鋪貨的消費硬體。之前還有許多開發人員需額外透過小小的 JSON manifest，才能推銷自己針對瀏覽器或其他平台所撰寫的 Web App。……而現在第一次使用這些 App 就幾乎達到完美。我還在想：『哇喔，這麼簡單？就像網頁一樣？』」

用實際硬體測試自己 App 的感覺當然很讚！但如果你目前不方便購買任一款 Geeksphone，也有許多方法可測試 Firefox OS 與 App：

  • 在自己的 Android 手機上安裝 Firefox Aurora for Android，然後就可以使用內建的 Marketplace 安裝 App。你的朋友或家人也可以一樣輕鬆安裝 App，而且還能一起測試你的 App。
  • 使用瀏覽器版本的 Firefox OS 模擬器，就可以用桌上型電腦測試自己的行動版 App。
  • 在自己的硬體上安裝 Firefox OS。

為什麼要開發 Firefox OS 用的 App？

  • 讓大家永續擁有開放式的網路。請與我們一起支持網路開放的概念，確保任何人都能享有網路的便利與功能 ─ 當然行動裝置也應有相同的理念。
  • 簡單易用。用單一技術堆疊 (HTML5、CSS、JavaScript、任何新的 WebAPI) 進行開發作業，並能跨網路與裝置使用。
  • 自由無拘束。任何人都不應受單一廠商所控制的系統所綁定。只要透過自己的網頁、Firefox Marketplace，甚至任何 Mozilla 開放式 App 技術所架構的商城，你都可以散佈或販售自己的 App。

如果有其他任何問題，可參閱 Firefox OS FAQ。下方另有 Geeksphone 的相關 FAQ。

我們期待能有更多傑出的開發人員，加入 Mozilla 行動裝置的行列。祝開發愉快！

Geeksphone FAQ:

Q. 今天所發表的重點為何？

A. 我們發表由 Geeksphone 公司所製造的 2 款 Firefox OS 開發者預覽手機，並可至 www.geeksphone.com 直接下單購買。此 2 款行動裝置可於實際環境中測試 Firefox OS 的功能，另可協助了解裝置的實際效能，還有行動網路的互動體驗等特性參數。

Q. 這 2 款裝置的規格與售價？

A. Keon 搭載 Cortex-A5 1Ghz 處理器、3.5 吋多點觸控螢幕、3 頻 UMTS/HSPA、4GB ROM、512 MB RAM、長效性 1580 mAh 電池。稅前且不含運費的售價為 91 歐元。

Peak 則搭載 1.2 GHz 雙核心 Qualcomm 8225 晶片組、4.3 吋 qHD IPS 螢幕、機背為 8 百萬畫素相機、前視鏡頭為 2 百萬畫素、3 頻 UMTS/HSPA、1800 mAh 電池。稅前且不含運費的售價為 149 歐元。

Q. 為何用這 2 款手機？

A. Keon 的規格，即已貼近電信業者所設計的實際手機硬體，並將於今年下半年正式出貨給消費者。Peak 則為較高端的裝置，可讓開發人員測試 App，以利未來正式的商業販售行為。

Q. Geeksphone 願意出貨到任何地方嗎？

A. 是的，可出貨至全球任何國家。

Q. Geeksphone 是怎樣的公司？

A. Geeksphone 為西班牙公司，專門設計並製造新一代的智慧型手機。該公司創立於 2009 年，並為歐洲第一家開發 Android OS 智慧型手機的公司。

Q. Telefónica 與 Mozilla 在其中所扮演的角色？

A. Telefónica 與 Mozilla 密切合作已久，並共同協助 Geeksphone 製造相關的開發測試單元。

Q. 你為何需要這些裝置？

A. 在為大眾市場發表手持裝置時，必須先提供開發者預覽手機，進而確保豐富的完整系統組合。Geeksphone 手機可讓開發人員在實際環境中測試 Firefox 的功能與特色，而不會侷限於現有的模擬器。另可測試數樣特性參數，如實際效能與行動網路的互動過程。

Q. 這 2 款是首發的 Firefox OS 裝置嗎？

A. Firefox OS 開發專案已交由不同的 OEM 廠搭載於不同的裝置之中，以利後續測試與展示。而這是第一次實際提供開發者預覽手機，讓專案之外的工程師能加入並測試 App。今天所發表的產品，並非 Firefox OS 正式投入消費市場。我們預估 2013 下半年將有搭載 Firefox OS 的裝置正式販售。

Q. 一般消費者也可以取得 Geeksphone 手機嗎？

A. 此 2 款裝置並不是為消費者所設計，且其搭載了 Firefox OS 開發者預覽版本。當然，我們知道許多人很期待能取得 Firefox OS 的裝置，但也只能請大家耐心等待，等到手機開始真正舖貨至消費市場，就能暢快享受截然不同的體驗。

Q: 那這算是哪一版的 Firefox OS？

A: 1.0.1

Q: 都多久會更新？

A: 幾個禮拜都會更新 1 次為新版本。

Q: 軟體功能完整嗎？正式發表之前會不會再加新的東西？

A: 此開發工程機搭載了開發者預覽版本。當然會定期持續提升其功能、效能、穩定度。若與以後送到消費者手上的版本相比，這裡所使用的軟體，肯定不會 100% 相同。

Q: 那我要怎樣才能測試以後的版本？

A: 新版本會自動透過 OTA (over the air) 提供。只要開發人員手上有 Geeksphone，均可更新到最新版本，以測試自己的 App。
