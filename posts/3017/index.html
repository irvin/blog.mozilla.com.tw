全新 Firefox 識別標誌 ─ 迎向 Firefox 的新世代

7月42013

《可點我看大圖》

這些年來關於 Firefox 是什麼的概念不斷與時共進。不久前，Firefox 還只是單純的桌機瀏覽器，但因應 Web 的多元發展，Firefox 也隨著擴展到 Android、Mac、Windows、Linux 等系統之上，同時進一步蛻變為 Apps 商城 (Firefox Marketplace) 與全新的行動作業系統。在如此短的時間內 Firefox 的蛻變著實驚人！

基於這一切轉變，我們決定現在是一個好的時間點在視覺上重新呈現 Firefox 標誌與產品圖示，以反應 Firefox 在這段時間的變化。如同你在上圖所看到的，Firefox 的原始概念仍維持不變，但我們嘗試以更前衛的設計理念，詮釋出 Firefox 在如今的 Web 以及 2013 年開始以後所將扮演的角色。

與先前的版本相較，新標誌特別納入「行動」的設計概念。其實原來的 Firefox 呈現在任何螢幕尺寸上都已經很漂亮了，但我們仍對其進一步微調，使其在小螢幕與較低解析度的裝置上看起來更清晰且乾淨。當然，新標誌亦適用於 Retina 螢幕 上。不同於以往，現在更能夠以 SVG 格式精確重現新標誌。這些特色都讓新標誌更貼近「著眼於 Web/來自於 Web」的概念。

目前 2013 版本的標誌是由 Sean Martell 所設計，而他本人已經在部落格上提供了許多設計細節。Mozilla 也首次在 Firefox 23 Beta 中使用了新標誌。接下來的幾個星期裡，新標誌將擴及所有的 Firefox 版本、網站、宣傳素材之中。

如果你是 Firefox 的粉絲，並持續關注 Firefox 標誌的演變過程，那你應該也會喜歡下面幾篇文章：

  • Sean Martell 的文章說明最新版標誌的發想過程
  • Alex Faaborg 圖解上次重新設計的細節與說明 (2009 年)
  • Jon Hicks 摘要說明了原始標誌的設計過程 (2004 年)
