Focus by Firefox – 為 Open Web 打造的內容攔截功能

我們在今天發佈了免費的「Focus by Firefox」內容攔截 App，即針對 iOS 9 的 Safari 所設計。此 App 可依追蹤器類型 (如用於廣告、分析、社交媒體的追蹤器) 進行攔截，讓使用者控制自己的資料流量，並可攔截 Web 字型而提升行動裝置的效能。

Mozilla 的理想網路，應該要能尊重使用者、讓使用者擁有更多選擇，更必須建立大家對網路的信任感並謹慎維持。已有太多使用者對網路存疑，也無法有意義的控制己身的數位生活。對網路的不信任已經影響到整個生態系統，有時更造成負面觀感。內容攔截功能將強化使用者的控制權，藉以重建大家的信任感。我們希望透過這些工具打造健康、開放，並能支援商業活動的生態系統，而不該只是鎖定 Web 或污名化特定的產業或內容。這也是我們之所以訂立三大內容攔截原則的理由。

Mozilla 現已開始實踐此三大原則。我們認為內容攔截清單的建立與維護方式，應該要揭露予內容發佈商或其他供應商知道；而非單純的讓特定內容永不見天日。我們希望透過此一功能鼓勵使用者與內容供應商展開對話，不要只是拿使用者的信任換取利潤，更不應該讓 Web 生態系統所代表的價值蕩然無存。大家在免費使用 Signal by Firefox 的同時，Mozilla 亦不會以其他方式從中牟利。

許多內容攔截 App 的攔截標準並不清楚，且其選擇方式亦不透明。這類 App 也不曾提供相關方式，讓受攔截的內容供應商得以改善內容進而解除攔截。某些受到攔截的供應商甚至直接付錢了事，以求自己能移出攔截名單之外。

但 Focus by Firefox 則採用不同的方式。我們加入產品之中的攔截名單，是由反追蹤應用程式開發商「Disconnect」根據通用公共授權 (General public license) 所建立。說到開放資料的價值，我們認為 Disconnect 的公開名單是絕佳入門點。該公司是以「追蹤」的公開定義為基礎來建立名單，並隨時更新名單的變化，讓使用者與發佈商能清楚得知其所套用的標準。這些公開標準就是發佈商 (此案例多為追蹤功能的使用者) 得以著手改善其內容的方式。在發佈商真正有所改善之後，Disconnect 另已提供一套清楚流程讓發佈商能解除攔截，藉此形成重要的良性循環。

內容攔截仍是 Mozilla 剛涉足的領域，相關解決方案也未臻完備。我們需要時間建立更完善的反饋機制，促使內容發佈商讓使用者真正掌握自己的線上經驗。而我們也應進一步了解使用者的需求：有人在乎隱私，常用行動裝置的人則較注重效能。透過 Focus by Firefox 讓大家對追蹤作業擁有更多選擇的同時，Mozilla 更規劃要能讓使用者控制裝置的效能與資料流量。

透過此創新產品，Mozilla 也將大方公開我們的攔截原則，藉以打造更完善的反饋機制。這也是我們之所以相信攔截工具在給予使用者控制權並贏回信任的同時，反而更能強化網路商業活動的原因。
