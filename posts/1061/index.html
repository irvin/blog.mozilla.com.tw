Persona 網路身分識別服務 – Beta 版首度釋出！

9月282012

在過去一年中，Mozilla 持續進行一個實驗性質的網路身分識別服務 Persona，此登錄系統保障使用者個資，消除各網站需使用不同密碼登入的問題，在兼顧便利性和網路個資安全性的前提下，Mozilla 很高興宣佈即將脫離「實驗」標籤，正式朝 Persona Beta 版邁進。

Persona 已經可以在主流的智慧型手機、平板、桌機瀏覽器上使用，使用者體驗也經過多次的測試和謹慎地檢視，此外，在核心 API 的基礎建設上，具備高度使用性和穩定度。

「Persona 絕對比 Open ID 或 OAuth 更易於使用，基本上可以全部分客戶端藉由 JavaScript 完成」－ 引用自開發者 David Somers。

在過去幾個月，我們致力於 Persona 的優化和完善，和起始相較起，我們改善了：

  • 全新品牌名：由專案名 Browser ID 更名至 Persona。
  • 打造全新、更好用的 API
  • 流暢的第一次使用者體驗
  • 在登錄視窗支援用戶網站名和 LOGO 顯示
  • 登錄視窗內容增強，網站連結的服務條款和隱私設定

這些優化都得到很好的評價，我們看到 Persona 獲得 Mozilla 以外的關注。如果你是一位開發者，現在是時候來試試 Persona 了。 Persona 是一個開放專案，歡迎大家通過寄件群組或是IRC 管道 (#identity on irc.mozilla.org) 共同投入開發和合作。

此版本是眾多 Beta 版中的第一版，我們計畫在未來放進更多的驚喜。你還在等什麼？只需一點時間整合且 Persona 與現有的登錄系統絕對可以並存。此外，Persona 網路身份認證服務將保有網站與用戶間最直接的關係，Persona 只透過 Email 認證，您可點選此處了解加入詳情。今天就把 Persona 加到你的網站吧！
