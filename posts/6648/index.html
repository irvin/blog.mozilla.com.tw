首款開發者專屬瀏覽器即將登場

Mozilla 知道開發者就是 Web 的基石，也因此積極推動標準並持續開發強大工具，要讓大家能輕鬆建構絕妙的 Web 內容與 App。

每當開發者針對 Web 撰寫內容時，所使用的多款工具又往往無法妥善配合。這也代表你必須不斷切換不同的工具、平台、瀏覽器，也連帶拖慢自己的速度並影響產能。

因此 Mozilla 決定讓開發工具團隊專注於瀏覽器之上，要找出更簡單的開發方式。

現在就先讓你稍微看看第一款專屬於開發者的瀏覽器：

我們以全新的觀點重新設計了瀏覽器，將開發者感興趣的東西擺在第一位。即由開發者專為開發者所打造的瀏覽器，可讓你對整個 Web 進行除錯，並更輕鬆打造絕妙的 Web 經驗。另亦整合如 WebIDE 與 Firefox Tools Adapter 的強大新工具。

Mozilla 很快就會打包更多資訊給大家，讓你成為獨立 Web 的建構者。

另請隨時散播 #Fx10 的相關資訊，或可訂閱「Hacks」上的電子報，可即時得知瀏覽器發佈的訊息。

我們期待著 11 月 10 日跟你分享。
