Firefox for Android 支援搭載 Intel x86 處理器的裝置

Mozilla 致力於為各地的使用者帶來最佳 Web 體驗。過去一年中，我們努力讓 Firefox for Android 能支援更多智慧型手機，讓你不論是使用哪一款裝置，都能隨時、隨地擁有絕妙的 Web 經驗。

秉持此一信念，Firefox for Android 如今更進一步支援 Intel x86 處理器的平板電腦與智慧型手機。全世界數百萬組裝置都能透過 Firefox for Android 享受直覺、強大、可自訂的使用經驗。

現在透過 Google Play 商店，使用者已可直接下載支援 Intel x86 處理器裝置的 Firefox for Android。搭載此款處理器的智慧型手機與平板電腦，包括有 Motorola RAZR i、ASUS Fonepad ME371、Acer Liquid C1、Lenovo K900 IdeaPHone、Samsung GALAXY Tab 3 (10.1) … 等數款裝置。

Firefox for Android 在 Intel x86 裝置上擁有高速度高效能的表現，更能加快執行具備豐富圖像的網站與 App (如遊戲)。使用 Motorola RAZR i 做為 Intel x86 手機的測試裝置，Firefox for Android 在 Octane、Kraken、Sunspider 等 JavaScript 效能標準測試上獲得極佳的評測分數。

Firefox 一直是功能強大、使用直覺、可個人化的瀏覽器，要將 Web 的力量交給所有人共享。Firefox for Android 同樣能簡化行動瀏覽作業、具備直覺性的設計、擁有的自訂特色為所有行動瀏覽器之最，並提供多項領先業界的隱私與安全功能，如不要追蹤我 (Do Not Track，DNT)、隱私瀏覽分頁 (Private Browsing)、訪客瀏覽階段 (Guest Browsing)。Firefox 可跨多款裝置而同步使用者的密碼、瀏覽記錄、已開啟的分頁，讓你確實掌握自己的 Web 經驗。
