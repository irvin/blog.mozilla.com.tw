Mozilla 行動市場總動員：走出自己的路！

2月262013

在過去十五年裡，Mozilla一直是網路的先驅和宣導者。我們創造和推動開放標準，致力於提升網路使用體驗，為網路的創新和開創而努力。

過去 Firefox 瀏覽器在桌機市場為使用者帶來創新和選擇性，今日，我們決心用一樣的熱忱翻轉行動市場，致力於消除行動裝置和 Apps 應用程式間的高牆，讓開發者、營運商和使用者都可以走自己的一條路。

於西班牙時間 24 日晚上，我們在媒體記者會上宣佈， Firefox OS 在世界各地的合作夥伴、2013 年的推出計劃以及第一只採用 Firefox OS 的手機和內容預告。

現在，快跟著西班牙巴塞隆納 MWC 的腳步，和我們搶先體驗 Firefox OS 帶給世界的創新與改變，就讓 “Firefox 的解放” 中的小狐狸為你我揭開序幕。

Firefox OS 中的小狐狸，趣味和友善；支持和保護；快速和強大。解放後的小狐狸不受傳統束縛，隨時出現在任何你需要牠的地方，為你的行動體驗努力，突破你的所有想像，捍衛並保護你的個人資料，並為你的網路生活增添最豐富多樣的色彩。活力、快速又輕巧，小狐狸讓你體驗自由掌握在手中的快感。

我們邀你一起和我們期待 “Firefox 的解放” 為行動市場所帶來的改變，請密切留意，Firefox OS 下一波動態！
