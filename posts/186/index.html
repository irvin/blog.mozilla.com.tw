Firefox 新增開發者工具和附加元件同步功能

3月142012

Ｍozilla 今日釋出最新版 Firefox，在瀏覽器內建可將網頁元件視覺化的開發者工具。Firefox 也強化Firefox Sync（同步）功能讓使用者可以將附加元件輕鬆同步到不同平台的 Firefox 瀏覽器。

Firefox 新增頁面檢測器的「3D 檢視」模式，可以 3D 方式呈現整個網站的結構，是很強大的開發者工具，此外也可以輕鬆編輯 CSS 的程式。這是一個全新的 WebGL 網站視覺化工具，可將每個網頁的結構以3D方式呈現，較平面顯示更為清楚，任何人都可以立刻了解網頁外觀與程式之間的關係。以往像檢視原始碼之類的開發者工具對於如何開發網頁很有幫助，而3D更可以呈現清楚的呈現網站的結構。在頁面檢測器中選取3D之後，將您的滑鼠移動到元件上即可得到更多資訊。

Firefox新增最新樣式編輯器，讓開發者編輯 CSS 樣式表就像編輯一般文字一樣，可以立即透過瀏覽器看到所作的修改。對於設計網站相當方便。一旦修改完成後，Firefox 樣式編輯器並提供簡單儲存檔案的方式。

新版 Firefox 擴充 Sync 的功能，可簡單完成附加元件的同步。使用者透過附加元件同步功能，可以將二台電腦 Firefox 的附加元件作同步，讓家中和公司的電腦具備相同功能。使用者可在「選項」（偏好設定）視窗開啟並同步。
