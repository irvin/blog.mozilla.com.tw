Firefox Marketplace 首度釋出 Firefox Aurora 測試版，歡迎開發者提供意見

10月222012

Firefox Marketplace 首度釋出 Aurora 測試版 (編按：Firefox 的產品在正式推出前會經歷過 Nightly, Aurora, Beta 三個進程，各自擁有六週的測試開發期，Aurora 主要提供給有興趣參與測試的開發者)，現在開放給使用 Firefox for Android Aurora 的初期使用者和測試者搶先體驗！新版本的釋出，目的讓開發者，針對其開發的應用程式得到回饋，同時以利整體 Marketplace 的開發和成型。Mozilla 專案開發都是在開放的環境中進行，預計這次的早期發佈將會對未來 Marketplace Aurora、Beta ，以及Firefox OS 在 2013 年的釋出有所幫助。

本次發佈精選了好幾種展示用的 Apps，包含遊戲，產品訊息和新聞平台等應用程式，供用戶瀏覽、安裝和應用。首批開放測試的 Apps 包括：Distant Orbit，Jauntly，Soundcloud 和Twitter。

目前的 Firefox  Marketplace  Aurora 尚處於初期階段，爾後我們將會增加金流、排名、審核等更多豐富的功能，敬請期待。

從去年開始，我們便致力將 Web 打造成一個開放的 Apps 開發平台。我們創造了 Firefox Marketplace，讓開發者能夠使用 Web  技術，例如 HTML、JavaScript 和 CSS 來開發豐富的、酷炫的 Apps，並透過 Marketplace 散佈並獲得收入。Mozilla 在開發跨平台 Web APIs 方面已經有令人驚喜的進展，我們希望釋放 Web 在行動市場的潛能，重現我們在桌機瀏覽器市場的成就。

如何開始使用 Marketplace 並提供意見?

  • 在你的 Android 手機/平板下載Firefox Aurora 行動版
  • 打開 Firefox Aurora 後可以直接在頁面上看到 “Get apps from the Marketplace” 的點選項目，若沒有看到，也可以到 “工具”的 Apps 找到
  • 進入 Marketplace 後可以使用和安裝
  • 在”設定”點選 Feedback （回饋）告訴我們您的意見

初期的 Firefox Marketplace 將僅可在 Firefox for Android Aurora 中使用。在 Firefox OS 和其他平台 Firefox 上的支援將陸續跟進。想瞭解更多關於在 Firefox Marketplace 上開發 Web apps 的訊息，請點選此處。
