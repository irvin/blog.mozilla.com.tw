Mozilla 持續向前

4月152014

Mozilla 目前正經歷著意料之外的領導階層過渡時期。我今天很高興能和 Reid Hoffman 以及 Katharina Borchert 這二位董事會成員，一同宣佈這次過渡期已進入下個階段：由 Chris Beard 加入 Mozilla 公司董事會並兼任代理執行長。

Mozilla 數個月前就開始評估 Chris 加入董事會的相關事宜。Chris 身為 Mozillian 已有很長的一段時間，在 Firefox 1.0 發佈之前，就與 Mozilla 有著密切的關係，在公司期間亦主導過 Mozilla 的許多創新專案。Chris 有著和其他人相同的 Mozilla 理念與使命感。我本人信賴他的判斷與建言將近 10 年。現在正是 Chris 對全球展現 Mozilla 願景的最佳時刻。

我們另外任命 Chris 身兼代理執行長。在目前過渡期間，他就是領導 Mozilla 的最佳人選。熟知 Mozilla 使命並將之轉化為實際方案與產品理念，Chris 絕對是我看過最了然於胸的人之一。在 Mozilla 草創之初，Chris 就負責帶領 Mozilla 的產品、行銷、創新等團隊，而近年身為行銷長時，也負責主導社群、使用者、開發者的全球性交流活動。不論是 Firefox 首次登上 Android 平台，或是 Firefox OS 在世界行動通訊大會 (MWC) 上初試啼聲，他均無役不與。Chris 現為帶領 Mozilla 度過這段時間的不二人選，也是未來執行長的強力候選人。

Mozilla 需要果斷、快速的行動力。包含執行長在內，Mozilla 的所有領導階層都必須眼明手快。而 Chris 的經驗與視野都能契合 Mozilla 的目標。在 2014 年，我們將持續讓 Firefox OS 投入更多市場並搭配更多規格的裝置，藉以提供 Open Web 專屬的使用者經驗。Mozilla 將為 Firefox 使用者帶來更豐富、更靈活的 Web 體驗，並會進一步整合更多服務。

目前 Open Web 與 Mozilla 所面臨的挑戰，正類似 Mozilla 草創時所遭遇的難題 ─ 集中化的技術與商業模式，不斷對互通性 (Interoperability) 與新加入的創業者設下屏障。我們今天豐富無比的線上生活，卻高度集中在少數幾個龐大組織的手上。這個環境深切需要活化的開放式替代方案，進而展現 Open Web 所能造成的影響。

Mozilla 正為全世界建立上述特性的替代方案，這也是我們之所以存在的理由。我們因為有著共同的理念與目標而相聚在一起，要將最近發生的許多事情化為動力，讓 Mozilla 能繼續主導 Web 的開發與創新。Chris 接任代理執行長只是整個過程的第一步，接下來就是執行長一職與新任董事的長期規劃。相關人選必須能協助 Mozilla 成功並繼續主動支持每一位 Mozillian，讓所有人都能完整發揮本身潛在的領導特質。

─ Mozilla 基金會主席 Mitchell Baker
