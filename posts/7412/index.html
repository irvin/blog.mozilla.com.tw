全球 Mozillian 群像系列 ─ 美國的班傑明

5月52015

有了遍佈全球的 Mozillian 熱情支持，我們才能走到今日。就讓我們簡單介紹具代表性的幾位 Mozillian 吧！

這些人將分享自己對 Mozilla 與 Web 的想法，包含他們的回憶、身為 Mozillian 的日常生活，以及對下個十年甚至未來的期待。

為大家介紹班傑明 (Benjamin Kerensa)

「Mozilla 帶動 Open Web 前進、保護 Web 的自由與透明、開放給所有人取用並打造出自己喜愛的樣子。Mozilla 就是開放的代名詞。」

先向大家簡單自我介紹，想三個可以形容你的關鍵字？

我是住在波特蘭的班傑明 (Benjamin Kerensa)，今年三十歲，工作是 IT 顧問。我一直很熱血的主動建立開放源碼專案社群，希望能推動 Open Web 持續向前。我閒暇時喜歡拍照或做手工皂。三個關鍵字就是：開放、創新、社群。

你何時、如何接觸到 Web？

我在 90 年代還是學生時，偶然機會用到 Netscape Navigator 的前身 ─「Mosaic」探索 Web，且這也是我接下來幾年主要在用的瀏覽器。我瀏覽網頁並透過早期的搜尋引擎找到所需的資訊。

你如何簡單形容網際網路及其潛力？

我只能說網路的可能性及其未來所深入的程度，都將超乎我們的想像。所以保護這個資源並讓所有人都能取用，就顯得格外重要。

簡單一個字或一句話描述 Mozilla？

對我來說，Mozilla 帶動 Open Web 前進、保護 Web 的自由與透明、開放給所有人取用並打造出自己喜愛的樣子。Mozilla 就是開放的代名詞。

你最喜歡貢獻給 Mozilla 的哪一點？

我喜歡那種「以正面力量影響全世界數百萬人」的感覺。從在校學生直到世界級的領袖，都在使用我從旁協助打造出來的軟體，而且這款軟體又幫助他們進行自己熱愛的事物。

對你所屬社群成員最美妙的記憶是？

許多 Mozillian 群聚在研討會熱烈討論，了解自己現正努力的事物，而且設法成就超越自己本身許多的大我。

對 Mozilla 的未來願景？

我深切看好 Mozilla 的未來，而且全球幾百萬人都將督促 Mozilla 打造更好的 Open Web，並朝著開放源碼軟體，以及嶄新平台與工具繼續邁進。
