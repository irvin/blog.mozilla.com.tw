Firefox 歡慶 10 歲生日 Firefox 周年紀念版與開發者版上線

「 Firefox 十周年特展－網路自由日 」

11 月 22 、 23 日華山 1914 文創園區熱鬧登場

美商謀智 (Mozilla) 今日歡慶 Firefox 十歲生日！Mozilla 在十年前打造 Firefox，要將網際網路留在所有人的手上，也就是為使用者提供更多選擇，成功改寫瀏覽器歷史。為彰顯今天的 Firefox 十週年慶，Mozilla 除了推出目前速度最快的 Firefox 週年紀念版，更發表首款專為開發者打造的瀏覽器 － Firefox 開發者 (Developer) 版本，內建獨立工具可提升產能並建構更好的 Web。此外，Mozilla 也於 11 月 22 日與 23 日兩日在華山 1914 文創產業園區首度舉辦大型特展－「Firefox 十週年特展 – 網路自由日」，台灣使用者屆時將可完整了解 Firefox 的過去、現在與未來。

生日就是回顧過去成就的時候。意義非凡的 Mozilla 生日，同時也是你的生日：我們一同拯救了網際網路。所謂「拯救」不是維持現狀，亦不是因任何公司的利益而控制我們的線上生活。微軟 (Microsoft) 在當時可說是支配了整個 Web，讓網路無法依他人的創意自由發展，卻停滯於單一公司的眼界之下。所幸有 Firefox 改變了此一現象。

Mozilla 不遺餘力的推廣線上選擇與機會，並專注建構開放標準而非專利＼私有的技術。我們讓使用者得以控制自己的線上經驗，也努力保有 Web 的開放性。我們並未建構更好的瀏覽器；卻是打造與眾不同的瀏覽器。事實證明，我們確實展現了更靈活的可能性、更好的方法、更好的未來。我們觸發了能量、競爭力、創造性，也藉此塑造出大家今天所熱愛的 Web：開放、獨立、無所不在。

Mozilla 與眾不同，並非傳統軟體公司。Mozilla 作為全球性的非營利組織，持續推廣線上的開放、創新、機會等核心理念。不論你是才剛開始接觸 Mozilla，最近才加入專案貢獻的行列，甚或是上禮拜才動手使用 Firefox，我們都希望你能了解並一起宣揚此一廣大社群所成就的事、對世界造成的影響，以及這一路上結交的許多朋友。

到今天為止，Firefox 已橫跨桌機與行動平台提供超過九十種語言的版本，讓全球數億人享有獨立的機會與選擇。為了慶祝此一里程碑，我們也在今天發表新產品與新規劃，要繼續打造更好的網路：

  • Firefox 週年紀念版不僅是目前最快的版本，也將透過新功能讓大家再進一步掌握 Web 的力量。我們將「DuckDuckGo」搜尋引擎內建於桌面版與行動版瀏覽器之中，也導入新的隱私權控制功能。了解更多 Firefox 週年紀念版的細節。
  • 今日我們也揭露一個全新策略功能 ─「Polaris」。它整合最好、最明確的方法，來強化線上的隱私權控制功能。Polaris 目的在匯集我們的全球社群，並公開招募民主與科技中心 (Center for Democracy and Technology，CDT)、Tor Project，以及其他組織的專家。我們初期會建構實驗性的反追蹤工具並與 Tor 密切合作，藉以測試且強化其網路結構。了解更多 Polaris 細節。
  • 首次展示的首頁磚 (Tile)，以及「興趣計分板 (Interest Dashboard)」附加元件，將取得廣告與建議內容之間的平衡；當然尊重使用者仍為Mozilla的首要考量。了解新分頁的首頁磚規劃，還有以使用者為中心的廣告趨勢。
  • Firefox 開發者 (Developer) 版本是首款專為開發者打造的瀏覽器，我們以全新的觀點重新設計了瀏覽器，將開發者感興趣的東西擺在第一位。此一版本是由開發者專為開發者所打造的瀏覽器，可讓開發者對整個 Web 進行除錯，並更輕鬆打造絕妙的 Web 經驗。另亦整合如 WebIDE 與 Firefox Tools Adapter 的強大新工具。內建的獨立工具可提升產能並建構更好的 Web。我們要讓開發者撰寫整體的 Web，而並非單純提供單一公司所須內容。
  • 強化的技術與搜尋功能，確保 Mozilla 產品仍具備領先業界的優勢，也是賴以推動 Web 平台持續向前的動力。Javascript 效能現居領導地位，Firefox 每夜更新版 (Nightly) 持續更新 E10S，並另已發表 Firefox 的 64 位元 Windows 版本，且在 3D 遊戲、通訊、安全性等方面亦各有斬獲。

接下來數週到數個月內，Mozilla 將陸續分享更多資訊。再次宣揚的理念：我們不但尚未結束，而且才剛要開始。與之前相較，Mozilla 認為自己目前在全世界所扮演的角色從未如此重要過。數位權、隱私權、網路中立性、線上安全性，均在在對我們的日常生活產生實質影響。線上生活的步伐與複雜度只會越來越快、越高。我們現在做的決定，會對未來線上生活產生本質上的影響。

自 2004 年 Firefox 1.0 版釋出，Firefox 瀏覽器始終堅持將使用者放在第一位，公開 Firefox 瀏覽器的每一行程式原始碼，供全球社群檢視。Mozilla 開放透明不藏私的特性，不僅有別於一般商業公司，也讓 Firefox 瀏覽器贏得全球超過五億名使用者的信任。而 Mozilla 本身去年更在美國被評選為最受信賴的網路公司。適逢 Firefox 歡慶十週年，Mozilla 更首度於 11 月 22 日與 23 日兩日在華山 1914 文創產業園區舉辦大型特展－「Firefox 十週年特展 – 網路自由日」，現場將完整呈現 Firefox 的過去、現在、未來，台灣的使用者將能瞭解 Mozilla 做為一個全球化的開放社群，如何以 Firefox 貫徹其信念，並以 Firefox OS 作為展望行動網路未來的利器。現場更安排多個闖關活動，讓民眾透過趣味性的參與，更加瞭解 Firefox 與網路世界，現場完成闖關活動即可獲得當日限量贈品。除動靜態展示外，現場民眾亦可參與兩天社群議程，聆聽來自 Mozilla 台灣社群的無私分享。

讓我們一起祝 Firefox 十週年快樂！
