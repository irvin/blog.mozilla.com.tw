全新 Firefox Logo 與 Firefox Beta 版亮眼登場！

6月282013

呼叫開發者，你想把自己的網站或服務整合至 Firefox 的 Social API 上嗎？現在，快透過 Social API，將自己的 App 整合至 Firefox Beta 中。

Firefox Beta 版新功能：

  • 分享 (Share)：Firefox Beta 版現在為 Social API 新增了「分享」按鈕與面板。Facebook 使用者只需輕點一下滑鼠，就可以透過新功能與親朋好友即時分享資訊（包括新聞、照片、影片、網站連結等）。
  • 封鎖混合內容 (Mixed Content Blocker)：只要網頁內同時包含安全性 (HTTPS) 與非安全性 (HTTP) 的內容，且經由 SSL 傳送該內容至瀏覽器，就會產生混合內容。將保護網站非安全性內容不被破解，而避免攻擊者惡意讀取或修改非安全性內容。
  • 網路監測器 (Network Monitor)：開發者可透過「網路監測器」功能細分出各個元件的載入時間，進而協助自己找出問題所在。
  • 針對 Mac OSX 10.7 進行 Firefox 最佳化：Firefox Beta 支援 Mac OSX 10.7 的反彈效果，當瀏覽時滾動網頁到達頂端與末端時，Firefox Beta 提供反彈效果，並支援 Mac OSX 10.7 最新的滾動條設計。

同時，Mozilla 很高興能在 Firefox Beta 版推出之際介紹最新的 Firefox 企業識別標誌。這個前衛的新標誌特別納入「行動」的概念，在「小螢幕和較低解析度裝置」上看起來「更為清晰和乾淨」。在接下來的幾個星期會有更多相關訊息更新。若要進一步了解 Firefox 的新標誌與其演變過程，可參閱 Mozilla 創意團隊的這篇文章。
