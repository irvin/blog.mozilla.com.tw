解放你的遊戲魂，加入我們的假日競賽

12月62013

得獎者可獲得總值 $45,000 美元的獎項

你有辦法設計出自己的互動遊戲展示，而且比我們的《Jetpack Santa》還更吸引人嗎？我們相信你絕對辦得到，也想幫你實際建構出遊戲。現在快呼朋引伴來參加 Mozilla 與 Goo 合辦的遊戲開發競賽 ─ Game Creator Challenge。

Open Web 一直是大家共有的資產，也因此 Mozilla 想看看一旦熱愛遊戲的玩家開始創造遊戲，而遊戲開發者為數十億人動手打造跨平台、跨瀏覽器 Web 遊戲的同時，此二者到底能激盪出怎樣的火花。

Mozilla 除了創新相關技術之外，早將 Web 定位為遊戲的開發、散佈平台，並能進一步主導遊戲競賽。其中如 WebGL、Emscripten、asm.js、Web Audio API 等的多項技術，就足以讓 Web 成為最完整、最好玩的遊戲平台。

快來選出最能發揮自己創造力的三項競賽之一！並參閱下列相關細節。

你必須使用 Goo 平台建構出自己的遊戲。Goo 平台所搭載的 Goo Engine，就是完全以 WebGL/HTML5 打造的 3D 遊戲引擎。另需搭配 Goo Create ─ 同樣是以 Goo Engine 執行的視覺效果編輯工具。可觀賞此影片中的 Goo Create 互動場景，激發出自己的靈感吧！

 你可從下列三項競賽中挑出最符合自己創造力的項目：

  • 最佳業餘互動式遊戲場景
  • 最佳桌面版遊戲
  • 最佳行動版遊戲

以下將接著說明這三個競賽項目與獎項。

我們的專業評審團成員分別來自於 Goo Technologies (首席技術官 Rikard Herlitz)、Mozilla (JavaScript 首席技術官 Brendan Eich 與 WebGL 工程總監 Vlad Vukicevic)，以及開發《憤怒鳥 (Angry Birds)》遊戲的 Rovio 公司，將在 1 月 17 日選出贏家。

你還在等什麼？快利用這次假期弄出有趣的玩意吧！就從這裡開始解放自己的創造天份！Mozilla 等不及要看到大家的作品！

競賽分類與獎項：

1. 最佳業餘互動式遊戲場景 ─ 給擁有熱情創造力的技術、美術人員

需透過 Goo Create 內建的物件函式庫 (即所謂的基本構成元件*)，建構出絕妙視覺效果的互動式遊戲場景與人物。可先參考全以 Goo Create 與其內建基本構成元件，所建構出的 Jetpack Santa 範例場景。作者可於 Goo Create 內複製此場景並作為自己場景的基礎。本獎項的參賽者必須完全沒有，或略有涉獵遊戲/場景的建構經驗)。評分重點包含音效、光線配置、景色架構、人物原創性進行評分。

積分前五名的參賽者可獲得：

– 現金 $1,000 美元

– 免費使用 Goo Create Pro 五年 (價值 $2,900 美元)

*基本構成元件 (Primitives) 即如方塊、球體等的簡單 3D 模型，並可進一步操作並整合之，藉以設計出人物或場景中的其他物體。

2. 最佳桌面遊戲 ─ 給絕妙創造力的工程師

需使用 Goo Create 並整合 Goo Engine JavaScript API，針對桌面版瀏覽器建構出玩家能身歷其境的遊戲。評分重點包含音效、視覺設計、互動機制的原創性、是否展現出作者本身對 Open Web 技術的掌握度，以及該遊戲是否能吸引玩家停不下手。

獎項：

  • 預定 2014 年 3 月 17 ~ 21 日於美國舊金山舉辦的遊戲開發者大會 (2014 GDC Conference)；或預定 8 月 13 ~ 17 日於德國科隆舉辦的歐洲最大電玩展 ─ 科隆遊戲展 (GamesCom Cologne)。雙人行程價值最高可達 $10,000 美元。得獎者亦可選擇領取現金 $5,000 美元。
  • 免費使用 Goo Create Pro 五年 (價值 $2,900 美元)
  • 於 Mozilla Apps 部落格中發表文章

3. 最佳行動裝置遊戲 ─ 給絕妙創造力的工程師

需使用 Goo Create 並整合 Goo Engine JavaScript API，為行動 Web 建構出玩家能身歷其境的遊戲。若跨平台的遊戲亦能夠於桌面環境執行，將可獲得額外積分。

評分重點包含音效、視覺設計、互動機制的原創性、是否展現出作者本身對 Open Web 技術的掌握度，以及該遊戲是否能吸引玩家停不下手。審查作業環境將使用 Firefox for Android 26，以近似 Nexus 4 的硬體中執行。同樣的，若能於 Firefox OS 中執行遊戲，將可獲得額外積分。

獎項：

  • 預定 2014 年 3 月 17 ~ 21 日於美國舊金山舉辦的遊戲開發者大會 (2014 GDC Conference)；或預定 8 月 13 ~ 17 日於德國科隆舉辦的歐洲最大電玩展 ─ 科隆遊戲展 (GamesCom Cologne)。雙人行程價值最高可達 $10,000 美元。得獎者亦可選擇領取現金 $5,000 美元。
  • 免費使用 Goo Create Pro 五年 (價值 $2,900 美金)
  • 於 Mozilla Apps 部落格中發表文章

###

在你開始動手之前：

先到競賽頁面報名之後，我們將隨後寄出相關說明，協助你開始建構自己的第一個互動式遊戲場景和人物。快啟動你的創造基因，馬上報名！

作品提交期限：

提交期限是太平洋時間 (PT) 的 1 月 14 日禮拜二午夜 12 點為止。可將自己完成的場景網址或遊戲寄送到 mozgoocreate@mozilla.com，即完成作品提交。
