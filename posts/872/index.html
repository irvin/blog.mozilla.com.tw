Firefox 大突破，超低記憶體使用率，提供高速瀏覽體驗

8月292012

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla 基金會，2012年推出以HTML5 為基礎的行動作業系統Firefox OS並獲得市場廣大期待。今於美西時間28日釋出全新Windows、Mac和Linux 版本適用的Firefox 瀏覽器。此版本大幅降低記憶體的使用率，即使在開啓附加元件或長時間瀏覽時，使用者仍可感受到快速、流暢與立即的瀏覽體驗，事不宜遲，請您立即下載馬上體驗。

Firefox 新增功能讓網站設計師更便利的打造優質上網體驗。全新 JavaScript 除錯器是一個快速的內建功能，能讓網站設計師更容易檢查程式錯誤的好用工具。程式設計師更可透過連上當地網絡，使用同一個JavaScript 除錯器，遠端測試您在Firefox for Android 上的 Apps 應用程式，大幅簡化行動程式開發流程。

Firefox 新版本的 WebGL 即日起支援壓縮材質 (compressed textures)，提供遊戲開發者在影音動畫等重度耗資源的開發時，能不喪失其性能和流暢度，Firefox 利用優化的 JavaScript 引擎和功能更強大的WebGL 打造更順暢的動畫效果。此外，新版本也提供遊戲開發商和互動內容提供者高達千分之一秒的準確度，請看 Firefox 針對遊戲體驗的示範影片。
