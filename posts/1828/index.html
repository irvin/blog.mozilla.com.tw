Mozilla Firefox Flicks 全球微影片召集令開跑！

1月302013

【美商謀智訊息】以推動網路平等、自由、開放為宗旨的全球非營利組織－ Mozilla 基金會，2012 年推出以 HTML5 為基礎的行動作業系統 Firefox OS 並獲得市場廣大期待。於美金時間 (30日) 盛大推出全球性影片募集活動 Firefox Flicks (微影片競賽)，強力號召全球對影片拍攝有熱忱，或有獨到想法的您，藉由 2013 年主題 「行動市場總動員」為今年 Firefox OS 的推出醞釀無比爆發力，不單讓 Web 與所有人的生活息息相關，更要每一位參賽者透過影片創作說出您對 Firefox 的熱愛與對 Web 的所有想像！

今天我們很榮幸宣佈 Firefox Flicks 又來了！這個由 Mozilla 基金會發起，乃一全球性的短片競賽，邀請所有 Firefox 愛好者、電影從業人員運用您的獨特創意，說出專屬於 Firefox 的故事，打造您獨一無二的 Firefox 短片並參與競賽。去年 Firefox Flicks 活動成果非凡，總共募集超過 400 支短片參與競賽，並號召超過千名電影從業人員從世界各個角落一同參賽。

2013 年 Firefox Flicks 的主題是「行動市場總動員」。我們深信透過 Web 的強大威力，可以讓 Web 更貼近所有人的生活。讓 Web 在你我生活中，用最簡單便利的方式，傳遞訊息、分享歡樂、翻轉每個人的所有想像！同時，Firefox Flicks 更為 Firefox OS 在 2013 年的推出醞釀無比爆發力，而一個前所未見，由 Web 基礎打造的 OS 將實現這個承諾。您可在下列網址找到詳細的影片製作需知。

Firefox Flicks 大力召集 Mozilla 社群、Firefox 愛用者、創意無窮者、電影工作者以及動畫製作師一同參與本活動。Firefox Flicks 影片上傳時間自 2月13日起，開放上傳 60 秒或少於 60 秒的影片，參賽作品需同意在線上以及在本活動播放使用。

優勝獎項包括：

  • 早鳥獎：提早上傳的人有機會贏得美金 1,000 元，而早鳥作品中的前三名影片將可進入決選作品內。
  • 區域獎：將從北美區、拉丁美洲區、歐洲區和亞洲區中選出各區冠軍，將得到美金 5,000 元的現金獎勵；而四大區分別再選出分區亞軍，將得到美金 2,500 元的現金獎勵。
  • 最佳人氣獎：在 8 月1 日至 8 月 31 日期間開放全球人氣票選，您可藉著投票選出心目中的最佳人氣獎，前三名將獲得 Firefox Flicks 獨家背包。

我們將在接下來的幾個月陸續釋出更多 Firefox Flicks 細節－包括好萊塢的評審名單、其他獎項、活動等等。您可在活動網址：www.firefoxflicks.org  找到更多活動內容，現在，讓我們看看去年的優勝作品如下。
