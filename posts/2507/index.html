「Firefox 祕技」你在用哪個版本的 Firefox 呢？

5月22013

在我們的部落格文章裡面，常常會看到「在最新版的 Firefox 中，我們提供了某些好用的新功能」這類的字眼。

但又要怎麼知道自己是不是在用最新版的 Firefox？在之前的文章裡面，我們解釋了 Firefox 自動更新的方法，但每隔一段時間才會有更新，且不會一次就釋出完整更新。

現在就告訴你怎麼看到目前 Firefox 的版本，超簡單的！

若你使用的是 Mac 系統，在視窗左上角你可以找到 Mozilla Firefox 的「Firefox」選單，點選「關於 Firefox」。接著就可以看到 Firefox 下方的版本編號。

若你使用的是 Windows 系統，請到 Firefox 視窗上方靠右的「說明」選單，同樣點選「關於 Firefox」。接著就可以看到 Firefox 下方的版本編號。

這裡有開啟 Mozilla Firefox 視窗的相關說明，而根據所使用的電腦可能有所不同。
