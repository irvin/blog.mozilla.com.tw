Firefox OS 安全性系列報導之二

12月32013

使用者經驗與安全性更新

每次我們只要展示 Firefox OS，幾乎都會有人對「安全性」抱持懷疑態度。用 Web 技術所建構的作業系統安全嗎？Mozilla 又內建了哪些機制，可避免使用者在不知情的狀況下接受某些下載作業 (Drive-by downloads) 或惡意軟體呢？本系列的第二支影片是由 Mozilla 技術傳教士 Christian Heilmann (@codepo8) 與 @OWASP Board 主席 Michael Coates (@_mwc) 所錄製，將討論 App 安裝與使用的使用者經驗，以及 Firefox OS 的安全性更新模型。

你也可到 YouTube 觀看此影片。

Firefox OS 是以強化 Web 的頂尖技術所打造。而 Mozilla 基於對 Firefox 超過十年的安全性實務與知識，另以多層次系統建構出了 Firefox OS。在提供行動 Web 強大功能的同時，亦將保護使用者的資訊安全。相關機制可讓使用者確實掌握自己的資料，也讓開發者擁有對應的 API 與技術，進而解放 Web 的力量。
