Firefox OS 安全性系列報導之一：Web 安全模型

11月212013

每次我們只要展示 Firefox OS，幾乎都會有人對「安全性」抱持懷疑態度。用 Web 技術所建構的作業系統安全嗎？Mozilla 又內建了哪些機制，可避免使用者在不知情的狀況下接受某些下載作業 (Drive-by downloads) 或惡意軟體呢？本系列共二支影片是由 Mozilla 技術傳教士 Christian Heilmann (@codepo8) 與 @OWASP Board 主席 Michael Coates (@_mwc) 所錄製，將討論 Firefox OS 的所有安全問題。

Firefox OS 是以強化 Web 的頂尖技術所打造。而 Mozilla 基於對 Firefox 超過十年的安全性實務與知識，另以多層次系統建構出了 Firefox OS。在提供行動 Web 強大功能的同時，亦將保護使用者的資訊安全。相關機制可讓使用者確實掌握自己的資料，也讓開發者擁有對應的 API 與技術，進而解放 Web 的力量。

系列影片將提及 Firefox OS 中的安全性設計與控制項。前二段影片說明 Firefox OS 的安全性，包含項目有：權限模型、更新模型、多層次架構、執行階段決策、使用者資料保護等等。可至 YouTube 上觀看本影片。

其他更多資訊：

  • Firefox OS 安全性架構
  • Firefox OS App 權限模型
  • App 安全性
  • 資料隔離 (Sandboxing)
  • Firefox OS 版本更新排程
