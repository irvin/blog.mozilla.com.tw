Firefox OS App Days 得獎專訪 – 第一名 When Arrival

2月62013

Q: App 構想來由

騎車去接女朋友時，因為路況多變，總是抓不準到達的時刻。在半路上使用手機傳簡訊通知對方最新估計的到達時刻是相當危險的行為。為了解決這樣的需求，以便有效率的運用雙方的等待時間，於是決定要開發一個 App 能夠在路途中，全自動根據目前的位置，推算即將到達的時間，並且通知對方。

Q: App 使用到的技術

GeoLocation API

Device Storage API

jQuery Mobile

Q: 參加 Firefox OS App Days 的原因與心得

我大學同學 Gina 在 Mozilla 公司上班，她在她的臉書分享了這項活動的資訊。我們公司的一些同事也互相分享了這個消息。我以前在 JSDC 的 Mozilla 攤位也見識了 Firefox OS 平台，覺得很有興趣想深入了解，這項活動也有提供教學課程，所以便決定參加了。

參加了這個活動，發現參加的開發者都很熱血又很有創意，聽到了很多超棒的 idea。在教學課程中，Mozilla 的工程師們不吝於傳授我們許多重要的知識，同時也學到了很多開發技巧。在下午的時候， Mentor 們非常認真的為我解答問題，讓我順利的完成想實作的功能。很感謝 Mozilla 公司舉辦這樣的活動讓有志一同的開發者們有機會齊聚一堂，也謝謝 Mozilla 工程師們的教學和指導，更要謝謝各組開發團隊分享精彩的成果展示，這真是一場很棒的知識饗宴。
