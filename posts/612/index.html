6月20日前進上海，Boot to Gecko軟實力發威！

6月132012

【美商謀智新聞】以推動網路平等、自由、開放為宗旨的全球非營利組織－Mozilla基金會，其瀏覽器Firefox全球擁有數億名愛好者，2012年起，將其在桌機市場成功推動的開放精神延伸至行動市場。繼 2 月於西班牙巴賽隆那的Mobile World Congress (MWC) 首次發表「The Web is the Platform」概念並展示一系列行動市場產品以來，受到全球各界熱烈關注。Mozilla Taiwan 身為開放行動作業系統 Boot to Gecko (B2G) 重點研發中心，瞄準亞洲行動市場軟硬體的蓬勃發展，於今 (13) 日宣布，將前進上海「GSMA亞洲移動通信博覽會」，於當地時間6月20 日至 6 月 22 日，展出最新具備智慧型手機功能的 HTML 5 行動作業系統 B2G，同時，展示新一代行動瀏覽器 Firefox for Android，以及可跨平台使用的 Web Apps 交易平台 Mozilla Marketplace，展現在行動市場上的強烈企圖心，打破行動市場現有之封閉現象，建立新世代行動網路標準。

首款 B2G 行動裝置利用 HTML5 技術，打電話、傳簡訊、Webs App 下載技術獨家大公開

即將在「GSMA 亞洲移動通信博覽會」展出的最新 B2G 作業系統，提供一個以 HTML5、JavaScript、CSS 等傳統 Web 技術打造的平台。B2G 作業系統藉由提交 WebAPIs 實現利用 Web 打電話、傳簡訊、App下載… 等一般時下智慧型手機的功能，開創未來 Web 技術在行動裝置上應用的無限可能！同時，Mozilla 正與合作的電信營運商 Telefonica 合作，提供客製化的使用者介面，期待提供未來智慧型手機使用者不同以往的操作經驗。

B2G 行動作業系統將可搭載於任何可支援 HTML5 技術的行動裝置，且預期未來使用者可在 B2G 上執行以 HTML 5 撰寫的 Web Apps。為完善此一架構，Mozilla 將同時展出Marketplace 商城，該交易平台目前已有數百個 Web Apps，預計下半年起提供使用者下載。Mozilla 期待透過完整的 Open Web 系統，以行動裝置、瀏覽器和交易平台的多方整合概念，提供開發商和使用者最便利的使用體驗，落實「The Web is the Platform」網路即是最大平台的全新概念。

前進上海「GSMA亞洲移動通信博覽會」，進軍行動通訊市場之絕對企圖心！

以「The Web is the Platform」為主題進軍上海「GSMA亞洲移動通信博覽會」的Mozilla Taiwan，將在現場展示三大行動產品線，包括「Open Web Devices」、「Firefox for Mobile」和「Mozilla Marketplace」，其中 Firefox for Mobile 展出新一代 Firefox for Android 行動瀏覽器，全新使用者經驗設計與極快速的連網體驗，讓行動瀏覽器與桌面瀏覽器一樣智慧優質。GSMA 亞洲移動通信博覽會為二月西班牙 MWC 之亞洲版盛會，首次移師上海舉辦。今年展覽吸引全球重量級電信營運商及手機製造業者參與，包括中國移動、中國聯通、諾基亞、騰訊等大廠。主辦單位全球移動通信協會（GSMA）預估，今年參展人數將超過2萬人。Mozilla Taiwan 期待透過本次活動，讓全球知名企業、大陸內需市場及上海當地消費者再次看到台灣堅強的軟實力！
